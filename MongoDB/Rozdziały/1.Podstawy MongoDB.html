Podstawy MongoDB

MongoDB to nierelacyjna baza danych NoSQL, która przechowuje dane w formacie dokumentów BSON (Binary JSON). Jest znana ze swojej elastyczności i skalowalności, co czyni ją popularnym wyborem dla wielu aplikacji internetowych.

Instalacja MongoDB

Aby zainstalować MongoDB, postępuj zgodnie z poniższymi krokami:

  1. Odwiedź stronę pobierania MongoDB i pobierz odpowiednią wersję dla swojego systemu operacyjnego.
  2. Zainstaluj pobrany pakiet, postępując zgodnie z instrukcjami instalatora.

Uruchamianie serwera MongoDB

Aby uruchomić serwer MongoDB, wykonaj następujące kroki:


# W systemie Windows
"C:\Program Files\MongoDB\Server\5.0\bin\mongod.exe" --dbpath "C:\data\db"

# W systemie macOS/Linux (po zainstalowaniu z Homebrew)
mongod --dbpath /usr/local/var/mongodb

--dbpath wskazuje lokalizację, w której MongoDB przechowuje dane. Upewnij się, że ten folder istnieje przed uruchomieniem serwera.

Użycie Mongo Shell

Mongo Shell to interaktywne środowisko wiersza poleceń umożliwiające pracę z bazą danych MongoDB.


# Uruchamianie Mongo Shell
mongo

Po uruchomieniu polecenia mongo, możesz wykonywać zapytania i operacje na bazie danych. Oto kilka podstawowych komend:


// Sprawdzenie dostępnych baz danych
db.adminCommand('listDatabases');

// Przełączenie na określoną bazę danych
use myDatabase;

// Dodanie dokumentu do kolekcji
db.myCollection.insert({ name: "Jan", age: 30 });

// Wyświetlenie wszystkich dokumentów z kolekcji
db.myCollection.find();

Komendy w Mongo Shell przypominają JavaScript, co czyni je intuicyjnymi dla programistów front-end i back-end.

Podsumowanie

  • MongoDB to baza danych NoSQL, przechowująca dane w formacie dokumentów BSON.
  • Aby uruchomić serwer MongoDB, użyj polecenia mongod i podaj --dbpath.
  • Mongo Shell pozwala na interaktywne zarządzanie bazą danych i wykonywanie operacji CRUD.
