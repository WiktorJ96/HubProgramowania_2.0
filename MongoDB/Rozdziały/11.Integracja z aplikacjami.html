Integracja z Aplikacjami w MongoDB

MongoDB można zintegrować z różnymi językami programowania, korzystając z dedykowanych bibliotek i sterowników. Poniżej omówiono, jak używać MongoDB z Node.js, Java, Python i PHP.

Node.js - Mongoose

Mongoose to popularna biblioteka ODM (Object Data Modeling) dla Node.js, która upraszcza pracę z MongoDB. Umożliwia tworzenie schematów, walidację danych i wykonywanie operacji CRUD.


// Instalacja Mongoose
npm install mongoose

// Przykład użycia Mongoose
const mongoose = require('mongoose');
mongoose.connect('mongodb://localhost:27017/mojabaza', { useNewUrlParser: true, useUnifiedTopology: true });

const UzytkownikSchema = new mongoose.Schema({
    imie: String,
    wiek: Number
});

const Uzytkownik = mongoose.model('Uzytkownik', UzytkownikSchema);

Uzytkownik.create({ imie: 'Anna', wiek: 28 })
    .then(doc => console.log('Utworzono użytkownika:', doc))
    .catch(err => console.error('Błąd:', err));

Java - MongoDB Driver

Dla języka Java MongoDB udostępnia oficjalny sterownik, który umożliwia bezpośrednią pracę z bazą danych.


// Dodanie zależności do projektu (Maven)

    org.mongodb
    mongodb-driver-sync
    4.3.1


// Przykład połączenia z MongoDB
import com.mongodb.client.MongoClients;
import com.mongodb.client.MongoClient;
import com.mongodb.client.MongoDatabase;

public class MongoDBExample {
    public static void main(String[] args) {
        try (MongoClient mongoClient = MongoClients.create("mongodb://localhost:27017")) {
            MongoDatabase database = mongoClient.getDatabase("mojabaza");
            System.out.println("Połączono z bazą danych: " + database.getName());
        }
    }
}

Python - PyMongo

PyMongo to oficjalna biblioteka umożliwiająca integrację MongoDB z aplikacjami Python.


# Instalacja PyMongo
pip install pymongo

# Przykład połączenia z MongoDB
from pymongo import MongoClient

client = MongoClient('mongodb://localhost:27017/')
db = client.mojabaza
kolekcja = db.uzytkownicy

wynik = kolekcja.insert_one({'imie': 'Jan', 'wiek': 35})
print('Dodano dokument o ID:', wynik.inserted_id)
