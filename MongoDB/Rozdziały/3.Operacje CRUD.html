Operacje CRUD w MongoDB

Operacje CRUD (ang. Create, Read, Update, Delete) to podstawowe operacje wykonywane na dokumentach w bazie danych MongoDB.

Tworzenie Dokumentu (Create)

Aby dodać dokument do kolekcji, używa się metody insertOne lub insertMany:


// Dodanie pojedynczego dokumentu
db.kolekcja.insertOne({ imie: \"Jan\", wiek: 30 });

// Dodanie wielu dokumentów
db.kolekcja.insertMany([
    { imie: \"Anna\", wiek: 25 },
    { imie: \"Piotr\", wiek: 35 }
]);

Odczytywanie Dokumentów (Read)

Do odczytu dokumentów służy metoda find. Można użyć filtrów, aby zwrócić określone dokumenty:


// Odczytanie wszystkich dokumentów
db.kolekcja.find();

// Odczytanie dokumentów z filtrem
db.kolekcja.find({ wiek: { $gt: 25 } });

Aktualizacja Dokumentów (Update)

Aktualizowanie dokumentów można wykonać za pomocą metod updateOne lub updateMany:


// Aktualizacja pojedynczego dokumentu
db.kolekcja.updateOne(
    { imie: \"Jan\" },
    { $set: { wiek: 31 } }
);

// Aktualizacja wielu dokumentów
db.kolekcja.updateMany(
    { wiek: { $lt: 30 } },
    { $set: { status: \"mlody\" } }
);

Usuwanie Dokumentów (Delete)

Usuwanie dokumentów odbywa się za pomocą metod deleteOne lub deleteMany:


// Usunięcie jednego dokumentu
db.kolekcja.deleteOne({ imie: \"Piotr\" });

// Usunięcie wielu dokumentów
db.kolekcja.deleteMany({ status: \"mlody\" });

Podsumowanie

  • Tworzenie: insertOne, insertMany
  • Odczytywanie: find z opcjonalnymi filtrami
  • Aktualizacja: updateOne, updateMany z operatorem $set
  • Usuwanie: deleteOne, deleteMany
