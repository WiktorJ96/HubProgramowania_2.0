Przykłady Display w Bootstrapie

  • Domyślny display (block)
  • Display inline
  • Display inline-block
  • Display flex
  • Display none
  • Responsywne display

Przykład 1: Domyślny display (block)

Element 1 (display: block)
Element 2 (display: block)

Kod HTML:


<div class="box d-block">Element 1 (display: block)</div>
<div class="box d-block">Element 2 (display: block)</div>

Przykład 2: Display inline

Element 1 (display: inline)
Element 2 (display: inline)

Kod HTML:


<div class="box d-inline">Element 1 (display: inline)</div>
<div class="box d-inline">Element 2 (display: inline)</div>

Przykład 3: Display inline-block

Element 1 (display: inline-block)
Element 2 (display: inline-block)

Kod HTML:


<div class="box d-inline-block">Element 1 (display: inline-block)</div>
<div class="box d-inline-block">Element 2 (display: inline-block)</div>

Przykład 4: Display flex

Element 1 (display: flex)
Element 2 (display: flex)
Element 3 (display: flex)

Kod HTML:


<div class="d-flex justify-content-around">
    <div class="box">Element 1 (display: flex)</div>
    <div class="box">Element 2 (display: flex)</div>
    <div class="box">Element 3 (display: flex)</div>
</div>

Przykład 5: Display none

Element 1 (display: none)
Element 2 (display: none)

Kod HTML:


<div class="box d-none">Element 1 (display: none)</div>
<div class="box d-none">Element 2 (display: none)</div>

Przykład 6: Responsywne display

Element 1 (d-none d-md-block d-lg-inline)
Element 2 (d-none d-md-block d-lg-inline)

Kod HTML:


<div class="box d-none d-md-block d-lg-inline">Element 1 (d-none d-md-block d-lg-inline)</div>
<div class="box d-none d-md-block d-lg-inline">Element 2 (d-none d-md-block d-lg-inline)</div>
