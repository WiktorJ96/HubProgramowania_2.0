Display w Bootstrap

Bootstrap oferuje szereg klas do zarządzania wyświetlaniem elementów HTML na stronie, co umożliwia szybką zmianę stylu wyświetlania bez pisania dodatkowego CSS. Klasy display można używać do kontrolowania wyświetlania elementów jako blokowych, inline, ukrytych i innych.

Podstawowe klasy display

Bootstrap posiada zestaw klas d-*, które odpowiadają właściwościom CSS display. Możemy z ich pomocą szybko ustawić, w jaki sposób elementy będą wyświetlane.

  • d-block – Ustawia element jako blokowy (ang. *block*)
  • d-inline – Ustawia element jako inline (ang. *inline*)
  • d-inline-block – Ustawia element jako inline-block
  • d-none – Ukrywa element (display: none)

Przykład użycia klas display


<!-- Przykład elementu blokowego -->
<div class="d-block">Ten element jest blokowy</div>

<!-- Przykład elementu inline -->
<span class="d-inline">Ten element jest inline</span>

<!-- Przykład ukrytego elementu -->
<div class="d-none">Ten element jest ukryty</div>

Display responsywny

Możesz również kontrolować wyświetlanie elementów na różnych rozdzielczościach ekranu za pomocą responsywnych klas d-*, które używają punktów przerwania Bootstrap. Na przykład, d-sm-block ustawi element jako blokowy tylko na małych ekranach i większych.

Przykład responsywnego wyświetlania


<!-- Element widoczny tylko na dużych ekranach -->
<div class="d-none d-lg-block">Widoczny tylko na ekranach o szerokości ≥ 992px</div>

<!-- Element ukryty na mniejszych ekranach -->
<div class="d-block d-md-none">Ukryty na średnich i większych ekranach</div>

W powyższym przykładzie pierwszy element jest ukryty na małych i średnich ekranach, ale staje się widoczny na dużych ekranach (d-lg-block). Drugi element jest widoczny na małych ekranach, ale znika na ekranach średnich i większych (d-md-none).

Inne klasy display

Bootstrap udostępnia również dodatkowe klasy do bardziej zaawansowanego zarządzania wyświetlaniem elementów:

  • d-flex – Ustawia element jako kontener Flexbox
  • d-inline-flex – Ustawia element jako kontener inline Flexbox
  • d-grid – Ustawia element jako kontener Grid Layout
  • d-inline-grid – Ustawia element jako inline Grid Layout

Przykład użycia Flexboxa


<!-- Element Flexbox -->
<div class="d-flex justify-content-center">
    <div>Flexbox Kolumna 1</div>
    <div>Flexbox Kolumna 2</div>
</div>

Przykład użycia Grid Layout


<!-- Element Grid -->
<div class="d-grid gap-2">
    <div>Kolumna Grid 1</div>
    <div>Kolumna Grid 2</div>
</div>

W powyższych przykładach klasy d-flex i d-grid zmieniają sposób wyświetlania elementów, ustawiając kontener Flexbox lub Grid Layout, który umożliwia bardziej zaawansowane zarządzanie układem elementów.

Podsumowanie

  • Klasy d-* w Bootstrapie pozwalają na szybkie zarządzanie wyświetlaniem elementów.
  • Możesz ustawiać elementy jako blokowe, inline, inline-block, flexbox lub grid, a także je ukrywać za pomocą d-none.
  • Klasy d-* są dostępne w wersjach responsywnych, co umożliwia dostosowanie wyświetlania na różnych rozdzielczościach ekranu.
  • Bootstrap udostępnia również klasy Flexbox i Grid Layout, które dają większą kontrolę nad układem elementów na stronie.
