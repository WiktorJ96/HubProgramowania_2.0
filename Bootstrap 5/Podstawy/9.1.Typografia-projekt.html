Przykład 2: Wyrównania tekstu

Tekst wyrównany do lewej.

Tekst wyrównany do środka.

Tekst wyrównany do prawej.

Tekst wyjustowany - wszystkie linie są równomiernie rozłożone na szerokość.

Kod HTML:


<p class="text-start">Tekst wyrównany do lewej.</p>
<p class="text-center">Tekst wyrównany do środka.</p>
<p class="text-end">Tekst wyrównany do prawej.</p>
<p class="text-justify">Tekst wyjustowany.</p>

Przykład 3: Style tekstu

Pogrubiony tekst (fw-bold).

Normalny tekst (fw-normal).

Kursywa (fst-italic).

Normalny styl (fst-normal).

TEKST WIELKIMI LITERAMI.

tekst małymi literami.

Pierwsza litera każdego wyrazu jest duża.

Kod HTML:


<p class="fw-bold">Pogrubiony tekst (fw-bold).</p>
<p class="fw-normal">Normalny tekst (fw-normal).</p>
<p class="fst-italic">Kursywa (fst-italic).</p>
<p class="fst-normal">Normalny styl (fst-normal).</p>
<p class="text-uppercase">TEKST WIELKIMI LITERAMI.</p>
<p class="text-lowercase">tekst małymi literami.</p>
<p class="text-capitalize">Pierwsza litera każdego wyrazu jest duża.</p>

Przykład 4: Listy

  • Pierwszy element listy
  • Drugi element listy
  • Trzeci element listy
  1. Pierwszy element listy numerowanej
  2. Drugi element listy numerowanej
  3. Trzeci element listy numerowanej
  • Lista bez stylu (bez kropek)
  • Drugi element bez stylu
  • Trzeci element bez stylu
  • Element 1 w linii
  • Element 2 w linii
  • Element 3 w linii

Kod HTML:


<ul>
    <li>Pierwszy element listy</li>
    <li>Drugi element listy</li>
    <li>Trzeci element listy</li>
</ul>

<ol>
    <li>Pierwszy element listy numerowanej</li>
    <li>Drugi element listy numerowanej</li>
    <li>Trzeci element listy numerowanej</li>
</ol>

<ul class="list-unstyled">
    <li>Lista bez stylu (bez kropek)</li>
    <li>Drugi element bez stylu</li>
    <li>Trzeci element bez stylu</li>
</ul>

<ul class="list-inline">
    <li class="list-inline-item">Element 1 w linii</li>
    <li class="list-inline-item">Element 2 w linii</li>
    <li class="list-inline-item">Element 3 w linii</li>
</ul>

Przykład 5: Tekst pomocniczy i skróty

Normalny tekst z tekstem pomocniczym.

Tekst ze skrótem: HTML

Kod HTML:


<p>Normalny tekst z <small class="text-muted">tekstem pomocniczym</small>.</p>
<p>Tekst ze skrótem: <abbr title="Hypertext Markup Language">HTML</abbr></p>

Przykład 6: Zawijanie i łamanie tekstu

To jest bardzo długi tekst, który zostanie zawinięty na małym obszarze.
To jest bardzo długi tekst, który nie zostanie zawinięty i wyjdzie poza obszar.
Tojestbardzooooodługiesłowoktórezostaniezłamaneponieważjestznaczniktext-break.

Kod HTML:


<div class="text-wrap">To jest bardzo długi tekst, który zostanie zawinięty.</div>
<div class="text-nowrap">To jest bardzo długi tekst, który nie zostanie zawinięty.</div>
<div class="text-break">Tojestbardzooooodługiesłowo które zostanie złamane.</div>

Przykład 7: Dekoracje tekstu

Podkreślony tekst (underline).

Przekreślony tekst (line-through).

Tekst bez dekoracji (none).

Kod HTML:


<p class="text-decoration-underline">Podkreślony tekst (underline).</p>
<p class="text-decoration-line-through">Przekreślony tekst (line-through).</p>
<p class="text-decoration-none">Tekst bez dekoracji (none).</p>
