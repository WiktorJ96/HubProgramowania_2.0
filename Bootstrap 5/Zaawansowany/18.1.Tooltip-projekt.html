Przykłady

  • Podstawowy tooltip
  • Tooltip z różnymi pozycjami
  • Tooltip wyzwalany przez JavaScript
  • Kolorowe tooltips

Przykład 1: Podstawowy tooltip

Kod HTML:


<button type="button" class="btn btn-primary" data-bs-toggle="tooltip" data-bs-placement="top" title="To jest podstawowy tooltip!">
    Najedź, aby zobaczyć tooltip
</button>

Przykład 2: Tooltip z różnymi pozycjami

Kod HTML:


<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-placement="top" title="Tooltip na górze">Góra</button>
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-placement="right" title="Tooltip po prawej">Prawa</button>
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-placement="bottom" title="Tooltip na dole">Dół</button>
<button type="button" class="btn btn-secondary" data-bs-toggle="tooltip" data-bs-placement="left" title="Tooltip po lewej">Lewa</button>

Przykład 3: Tooltip wyzwalany przez JavaScript

Kod HTML i JavaScript:


<button type="button" class="btn btn-info" id="jsTooltipBtn">Kliknij, aby pokazać tooltip</button>

<script>
var tooltipTrigger = new bootstrap.Tooltip(document.getElementById('jsTooltipBtn'), {
    trigger: 'manual',
    title: 'Tooltip uruchomiony przez JavaScript'
});

document.getElementById('jsTooltipBtn').addEventListener('click', function () {
    tooltipTrigger.show();
});
</script>
