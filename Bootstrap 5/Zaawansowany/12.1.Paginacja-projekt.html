Przykłady

  • Podstawowa paginacja
  • Paginacja z disabled i active
  • Paginacja z ikonami
  • Paginacja w wersji zaokrąglonej

Przykład 1: Podstawowa paginacja

  • Poprzednia
  • 1
  • 2
  • 3
  • Następna

Kod HTML:


<nav aria-label="Page navigation example">
    <ul class="pagination">
        <li class="page-item"><a class="page-link" href="#">Poprzednia</a></li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item"><a class="page-link" href="#">Następna</a></li>
    </ul>
</nav>

Przykład 2: Paginacja z disabled i active

  • Poprzednia
  • 1
  • 2
  • 3
  • Następna

Kod HTML:


<nav aria-label="Page navigation example">
    <ul class="pagination">
        <li class="page-item disabled">
            <a class="page-link" href="#" tabindex="-1" aria-disabled="true">Poprzednia</a>
        </li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item active" aria-current="page">
            <a class="page-link" href="#">2</a>
        </li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item">
            <a class="page-link" href="#">Następna</a>
        </li>
    </ul>
</nav>

Przykład 3: Paginacja z ikonami

  • «
  • 1
  • 2
  • 3
  • »

Kod HTML:


<nav aria-label="Page navigation example">
    <ul class="pagination">
        <li class="page-item">
            <a class="page-link" href="#" aria-label="Poprzednia">
                <span aria-hidden="true">«</span>
            </a>
        </li>
        <li class="page-item"><a class="page-link" href="#">1</a></li>
        <li class="page-item"><a class="page-link" href="#">2</a></li>
        <li class="page-item"><a class="page-link" href="#">3</a></li>
        <li class="page-item">
            <a class="page-link" href="#" aria-label="Następna">
                <span aria-hidden="true">»</span>
            </a>
        </li>
    </ul>
</nav>
