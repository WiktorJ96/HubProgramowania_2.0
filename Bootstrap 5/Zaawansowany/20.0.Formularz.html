Formularze w Bootstrapie

Komponent Formularze w Bootstrapie pozwala na tworzenie stylizowanych formularzy, które są estetyczne i funkcjonalne. Bootstrap oferuje różne komponenty, takie jak pola tekstowe, checkboxy, przyciski radiowe, dropdowny, suwaki zakresu i wiele innych.

Podstawowy Formularz

Poniżej znajduje się przykład podstawowego formularza z polami tekstowymi, przyciskiem oraz prostą walidacją.


<!-- Podstawowy formularz -->
<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Adres email</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
    <div id="emailHelp" class="form-text">Nigdy nie udostępnimy Twojego emaila.</div>
  </div>
  <div class="mb-3">
    <label for="exampleInputPassword1" class="form-label">Hasło</label>
    <input type="password" class="form-control" id="exampleInputPassword1">
  </div>
  <button type="submit" class="btn btn-primary">Zaloguj się</button>
</form>

Formularz z Grupami Pól (Input Groups)

Grupy pól pozwalają na połączenie różnych elementów formularza, takich jak przyciski i pola tekstowe, aby stworzyć bardziej funkcjonalne układy.


<!-- Grupy pól -->
<div class="input-group mb-3">
  <span class="input-group-text" id="basic-addon1">@</span>
  <input type="text" class="form-control" placeholder="Nazwa użytkownika" aria-label="Nazwa użytkownika" aria-describedby="basic-addon1">
</div>

Checkboxy, Radio, Switch

Bootstrap oferuje klasy do stylizacji checkboxów oraz przycisków radiowych, co umożliwia tworzenie estetycznych list wyboru. Przyciski przełączników (switch) są stylizowanymi checkboxami, które można łatwo włączyć i wyłączyć.


<!-- Checkboxy -->
<div class="form-check">
  <input class="form-check-input" type="checkbox" value="" id="flexCheckDefault">
  <label class="form-check-label" for="flexCheckDefault">
    Domyślny checkbox
  </label>
</div>

<!-- Radio -->
<div class="form-check">
  <input class="form-check-input" type="radio" name="flexRadioDefault" id="flexRadioDefault1">
  <label class="form-check-label" for="flexRadioDefault1">
    Domyślny radio
  </label>
</div>

<!-- Form Switch -->
<div class="form-check form-switch">
  <input class="form-check-input" type="checkbox" id="flexSwitchCheckDefault">
  <label class="form-check-label" for="flexSwitchCheckDefault">Domyślny przełącznik</label>
</div>

Formularz w Układzie Poziomym

Możesz tworzyć formularze w układzie poziomym, gdzie etykiety i pola są umieszczone obok siebie, co oszczędza miejsce.


<!-- Poziomy formularz -->
<form class="row g-3">
  <div class="col-auto">
    <label for="inputPassword2" class="visually-hidden">Hasło</label>
    <input type="password" class="form-control" id="inputPassword2" placeholder="Hasło">
  </div>
  <div class="col-auto">
    <button type="submit" class="btn btn-primary mb-3">Potwierdź</button>
  </div>
</form>

Walidacja Formularzy

Bootstrap pozwala na łatwą walidację formularzy, dodając klasy takie jak .is-invalid oraz .is-valid, które wskazują na błędy lub poprawne dane.


<!-- Walidacja formularza -->
<form class="needs-validation" novalidate>
  <div class="mb-3">
    <label for="validationCustom01" class="form-label">Imię</label>
    <input type="text" class="form-control is-valid" id="validationCustom01" value="Jan" required>
    <div class="valid-feedback">
      Wygląda dobrze!
    </div>
  </div>
  <div class="mb-3">
    <label for="validationCustom02" class="form-label">Nazwisko</label>
    <input type="text" class="form-control is-invalid" id="validationCustom02" required>
    <div class="invalid-feedback">
      Wprowadź prawidłowe nazwisko.
    </div>
  </div>
</form>

Wybór Koloru

Bootstrap obsługuje wybór koloru za pomocą elementu input[type="color"].


<!-- Wybór koloru -->
<div class="mb-3">
  <label for="colorInput" class="form-label">Wybierz kolor</label>
  <input type="color" class="form-control form-control-color" id="colorInput" value="#563d7c" title="Wybierz kolor">
</div>

Form Select (Dropdown)

Możesz łatwo tworzyć rozwijane listy wyboru przy użyciu klasy .form-select.


<!-- Form Select -->
<div class="mb-3">
  <label for="exampleSelect" class="form-label">Wybierz opcję</label>
  <select class="form-select" id="exampleSelect">
    <option selected>Otwórz to menu</option>
    <option value="1">Opcja 1</option>
    <option value="2">Opcja 2</option>
    <option value="3">Opcja 3</option>
  </select>
</div>

Przyciski Check (Button Check)

Możesz także tworzyć przyciski typu checkbox, które wyglądają jak przyciski, przy użyciu klas .btn-check i .btn.


<!-- Button Check -->
<input type="checkbox" class="btn-check" id="btn-check" autocomplete="off">
<label class="btn btn-primary" for="btn-check">Sprawdź mnie</label>

Form File (Wybór Pliku)

Elementy do wybierania plików mogą być łatwo stylizowane przy użyciu klasy .form-control.


<!-- Form File -->
<div class="mb-3">
  <label for="formFile" class="form-label">Wybierz plik</label>
  <input class="form-control" type="file" id="formFile">
</div>

Form Range (Zakres)

Bootstrap oferuje łatwe w użyciu suwaki, które można dostosować do wybranych zakresów.


<!-- Form Range -->
<label for="customRange1" class="form-label">Zakres</label>
<input type="range" class="form-range" id="customRange1">

Podsumowanie

  • Formularze w Bootstrapie oferują różnorodne komponenty, które umożliwiają łatwe tworzenie estetycznych i funkcjonalnych elementów formularza.
  • Bootstrap obsługuje różne typy pól, w tym kolor, przełączniki, przyciski check, wybór plików i suwaki zakresu.
  • Użycie tych komponentów poprawia interakcję użytkownika, dzięki czemu formularze są łatwiejsze do użycia i bardziej funkcjonalne.
