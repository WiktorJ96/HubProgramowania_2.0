  • Nawigacja z dropdown

Przykład 1: Podstawowa nawigacja

  • Aktywna
  • Link
  • Inny link
  • Wyłączony

Kod HTML:


<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" aria-current="page" href="#">Aktywna</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Link</a>
    </li>
    <li class="nav-item">
        <a class="nav-link" href="#">Inny link</a>
    </li>
    <li class="nav-item">
        <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Wyłączony</a>
    </li>
</ul>
<form action="" class="d-flex ms-auto">
    <input type="text" class="form-control mr-2" type="serach" placeholder="Search" aria-label="Search">
    <button class="btn btn-outline-dark" type="submit">Szukaj</button>
</form>

Przykład 2: Responsywna nawigacja z hamburger menu

  • Home
  • Link
  • Inny link
  • Wyłączony

Kod HTML:


<nav class="navbar navbar-expand-lg navbar-dark bg-dark p-3 rounded-2">
    <a class="navbar-brand" href="#"><img src="#" alt="" width="40" height="30"></a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
        <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
        <ul class="navbar-nav">
            <li class="nav-item">
                <a class="nav-link active" aria-current="page" href="#">Home</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Link</a>
            </li>
            <li class="nav-item">
                <a class="nav-link" href="#">Inny link</a>
            </li>
            <li class="nav-item">
                <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Wyłączony</a>
            </li>
        </ul>
        <form action="" class="d-flex ms-auto">
                <input type="text" class="form-control mr-2" type="serach" placeholder="Search" aria-label="Search">
                <button class="btn btn-outline-light ms-1" type="submit">Szukaj</button>
            </form>
    </div>
</nav>

Przykład 5: Nawigacja z dropdown

  • Aktywna
  • Dropdown
    • Akcja
    • Inna akcja
    • Jeszcze inna akcja

Kod HTML:


<ul class="nav">
    <li class="nav-item">
        <a class="nav-link active" aria-current="page" href="#">Aktywna</a>
    </li>
    <li class="nav-item dropdown">
        <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-bs-toggle="dropdown" aria-expanded="false">
            Dropdown
        </a>
        <ul class="dropdown-menu" aria-labelledby="navbarDropdown">
            <li><a class="dropdown-item" href="#">Akcja</a></li>
            <li><a class="dropdown-item" href="#">Inna akcja</a></li>
            <li><hr class="dropdown-divider"></li>
            <li><a class="dropdown-item" href="#">Jeszcze inna akcja</a>
        </ul>
    </li>
</ul>
