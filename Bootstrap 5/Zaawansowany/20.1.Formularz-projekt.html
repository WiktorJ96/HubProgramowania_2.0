Przykład 4: Formularz z walidacją

Wprowadź imię.

Kod HTML:


<form class="needs-validation" novalidate>
    <div class="mb-3">
        <label for="validationCustom01" class="form-label">Imię</label>
        <input type="text" class="form-control" id="validationCustom01" required>
        <div class="invalid-feedback">
            Wprowadź imię.
        </div>
    </div>
    <button class="btn btn-primary" type="submit">Zatwierdź</button>
</form>

Kod Java Script:


        <script>
            (() => {
        'use strict';

        // Pobieramy wszystkie formularze, które wymagają walidacji
        const forms = document.querySelectorAll('.needs-validation');

        // Iterujemy przez formularze i dodajemy obsługę walidacji
        Array.from(forms).forEach(form => {
            form.addEventListener('submit', event => {
                // Jeśli formularz nie jest poprawny, anulujemy wysyłanie
                if (!form.checkValidity()) {
                    event.preventDefault();
                    event.stopPropagation();
                }

                // Dodajemy klasę `was-validated` aby uaktywnić style walidacji Bootstrapa
                form.classList.add('was-validated');
            }, false);
        });
    })();
    </script>

Przykład 7: Switch i range

Kod HTML:


<form>
    <div class="form-check form-switch mb-3">
        <input class="form-check-input" type="checkbox" id="flexSwitchCheckDefault">
        <label class="form-check-label" for="flexSwitchCheckDefault">Tryb nocny</label>
    </div>
    <div class="mb-3">
        <label for="rangeExample" class="form-label">Dostosuj głośność</label>
        <input type="range" min="0" max="100" value="27" class="form-range" id="rangeExample">
    </div>
    <button type="submit" class="btn btn-primary">Zatwierdź</button>
</form>

Kod Java Script:



        <script>
        // Kod dla switcha
            const nightModeSwitch = document.getElementById('flexSwitchCheckDefault');

    nightModeSwitch.addEventListener('change', function() {
        if (this.checked) {
           
            document.body.style.backgroundColor = '#222';
            document.body.style.color = '#ffffff'; 
        } else {
       
            document.body.style.backgroundColor = '#ffffff'; 
            document.body.style.color = '#000000'; 
        }           
    });
        </script>
