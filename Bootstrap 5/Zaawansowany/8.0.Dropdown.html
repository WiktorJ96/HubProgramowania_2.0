Rozwijane menu (Dropdown) w Bootstrapie

Dropdown, czyli rozwijane menu, pozwala na ukrycie listy opcji pod jednym przyciskiem lub linkiem. Menu rozwija się po kliknięciu, a użytkownik może wybrać jedną z dostępnych opcji.

Podstawowy przykład dropdown


<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton" data-bs-toggle="dropdown" aria-expanded="false">
    Rozwijane menu
  </button>
  <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton">
    <li><a class="dropdown-item active" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
    <li><a class="dropdown-item" href="#">Opcja 3</a></li>
  </ul>
</div>

W tym przykładzie przycisk o nazwie "Rozwijane menu" rozwija listę trzech opcji. Użytkownik może wybrać jedną z dostępnych opcji z listy. W powyższym przykładzie użyta jest też klasa active która podświetla aktualnie wybraną opcje.

Dropdown z podzielonym przyciskiem

Podzielony przycisk, gdzie jedna część wykonuje akcję, a druga rozwija dodatkowe opcje.


<div class="btn-group">
  <button type="button" class="btn btn-primary">Akcja</button>
  <button type="button" class="btn btn-primary dropdown-toggle dropdown-toggle-split" data-bs-toggle="dropdown" aria-expanded="false">
    <span class="visually-hidden">Rozwiń menu</span>
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
    <li><a class="dropdown-item" href="#">Opcja 3</a></li>
  </ul>
</div>

Dropdown z divider (separator)

Możesz dodać separator w rozwijanym menu za pomocą klasy .dropdown-divider.


<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton2" data-bs-toggle="dropdown" aria-expanded="false">
    Menu z divider
  </button>
  <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton2">
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
    <li><hr class="dropdown-divider"></li>
    <li><a class="dropdown-item" href="#">Opcja 3</a></li>
  </ul>
</div>

Dropdown z drop left i drop right

Menu rozwijane w lewo lub w prawo za pomocą klas .dropstart (lewo) i .dropend (prawo).

Drop left (menu rozwijane w lewo)


<div class="btn-group dropstart">
  <button type="button" class="btn btn-danger dropdown-toggle" data-bs-toggle="dropdown" aria-expanded="false">
    Menu w lewo
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
  </ul>
</div>

Drop right (menu rozwijane w prawo)


<div class="btn-group dropend">
  <button type="button" class="btn btn-warning dropdown-toggle" data-bs-toggle="dropdown" aria-expanded="false">
    Menu w prawo
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
  </ul>
</div>

Dropdown disabled (nieaktywny przycisk)

Możesz dezaktywować rozwijane menu, dodając atrybut disabled do przycisku.


<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton3" data-bs-toggle="dropdown" aria-expanded="false" disabled>
    Nieaktywny przycisk
  </button>
</div>

Dropdown header (nagłówek rozwijanego menu)

Możesz dodać nagłówek do rozwijanego menu za pomocą klasy .dropdown-header.


<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button" id="dropdownMenuButton4" data-bs-toggle="dropdown" aria-expanded="false">
    Menu z nagłówkiem
  </button>
  <ul class="dropdown-menu" aria-labelledby="dropdownMenuButton4">
    <li><h6 class="dropdown-header">Nagłówek menu</h6></li>
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
  </ul>
</div>

Podsumowanie

  • Dropdown umożliwia tworzenie rozwijanych list opcji w różnych kierunkach (w prawo, w lewo).
  • Można dodawać podzielone przyciski, separatory (divider), nagłówki i wyłączać przyciski (disabled).
  • Popper.js jest wymagany, aby dropdown działał prawidłowo, a Bootstrap automatycznie go ładuje.
