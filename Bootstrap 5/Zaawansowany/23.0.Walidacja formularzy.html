Walidacja formularzy w Bootstrapie

Walidacja formularzy w Bootstrapie pozwala na łatwe dodanie reguł sprawdzających poprawność danych wprowadzanych przez użytkowników. Dzięki temu można upewnić się, że formularze są wypełniane poprawnie przed ich wysłaniem. Bootstrap wspiera zarówno podstawową walidację HTML5, jak i niestandardowe komunikaty.

Podstawowa Walidacja HTML5

Walidacja HTML5 jest domyślnie wspierana przez przeglądarki i można ją łatwo włączyć, dodając atrybuty takie jak required, type czy pattern.


<!-- Podstawowa walidacja HTML5 -->
<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Adres email</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp" required>
    <div class="invalid-feedback">Proszę wprowadzić prawidłowy adres email.</div>
  </div>
  <button type="submit" class="btn btn-primary">Wyślij</button>
</form>

Niemożliwe do Pominęcia Pola (Required)

Możesz oznaczyć pola jako obowiązkowe, dodając atrybut required, co zapobiegnie wysłaniu formularza, jeśli pole nie jest wypełnione.


<!-- Pola obowiązkowe -->
<form>
  <div class="mb-3">
    <label for="inputName" class="form-label">Imię i nazwisko</label>
    <input type="text" class="form-control" id="inputName" required>
    <div class="invalid-feedback">To pole jest wymagane.</div>
  </div>
  <button type="submit" class="btn btn-primary">Zatwierdź</button>
</form>

Walidacja Niestandardowa z Bootstrapem

Bootstrap pozwala na dodanie niestandardowych komunikatów walidacyjnych za pomocą klas .is-valid oraz .is-invalid. Użyj tych klas, aby dodać wizualną wskazówkę o poprawności pola.


<!-- Niestandardowa walidacja -->
<form class="needs-validation" novalidate>
  <div class="mb-3">
    <label for="validationCustomUsername" class="form-label">Nazwa użytkownika</label>
    <div class="input-group has-validation">
      <span class="input-group-text" id="inputGroupPrepend">@</span>
      <input type="text" class="form-control is-invalid" id="validationCustomUsername" aria-describedby="inputGroupPrepend" required>
      <div class="invalid-feedback">Wprowadź prawidłową nazwę użytkownika.</div>
    </div>
  </div>
  <button class="btn btn-primary" type="submit">Zatwierdź</button>
</form>

Przykład Dynamicznej Walidacji

Możesz dodać niestandardową walidację po stronie klienta, korzystając z JavaScript, aby dynamicznie aktualizować stan pól formularza. Poniżej znajduje się przykładowy kod.


<!-- Dynamiczna walidacja -->
<form class="needs-validation" novalidate>
  <div class="row">
    <div class="col-md-4 mb-3">
      <label for="validationCustom01" class="form-label">Imię</label>
      <input type="text" class="form-control" id="validationCustom01" value="Jan" required>
      <div class="valid-feedback">Wygląda dobrze!</div>
    </div>
    <div class="col-md-4 mb-3">
      <label for="validationCustom02" class="form-label">Nazwisko</label>
      <input type="text" class="form-control" id="validationCustom02" value="Kowalski" required>
      <div class="valid-feedback">Wygląda dobrze!</div>
    </div>
    <div class="col-md-4 mb-3">
      <label for="validationCustomUsername" class="form-label">Nazwa użytkownika</label>
      <div class="input-group has-validation">
        <span class="input-group-text" id="inputGroupPrepend">@</span>
        <input type="text" class="form-control" id="validationCustomUsername" aria-describedby="inputGroupPrepend" required>
        <div class="invalid-feedback">Proszę wprowadzić nazwę użytkownika.</div>
      </div>
    </div>
  </div>
  <button class="btn btn-primary" type="submit">Prześlij</button>
</form>

Dynamiczna walidacja wymaga dodania skryptu JavaScript, który zapobiegnie wysłaniu formularza, jeśli dane są niepoprawne.

Przykładowy Skrypt JavaScript


// JavaScript do dynamicznej walidacji formularza
(function () {
  'use strict'
  var forms = document.querySelectorAll('.needs-validation')
  Array.prototype.slice.call(forms).forEach(function (form) {
    form.addEventListener('submit', function (event) {
      if (!form.checkValidity()) {
        event.preventDefault()
        event.stopPropagation()
      }
      form.classList.add('was-validated')
    }, false)
  })
})();

Podsumowanie

  • Walidacja Formularzy w Bootstrapie pozwala na łatwe dodanie reguł sprawdzających poprawność danych.
  • Możesz używać podstawowej walidacji HTML5 lub niestandardowych komunikatów walidacyjnych z klasami .is-valid i .is-invalid.
  • Dynamiczna walidacja po stronie klienta może zostać wdrożona przy pomocy JavaScript, co pozwala na natychmiastowe reagowanie na błędy użytkownika.
