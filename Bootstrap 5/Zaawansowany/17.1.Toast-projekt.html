Przykład 1: Podstawowy toast

Bootstrap Just now
Hello, this is a basic toast!

Kod HTML:


<div class="toast-container">
    <div class="toast" role="alert" aria-live="assertive" aria-atomic="true">
        <div class="toast-header">
            <strong class="me-auto">Bootstrap</strong>
            <small class="text-muted">Just now</small>
            <button type="button" class="btn-close" data-bs-dismiss="toast" aria-label="Close"></button>
        </div>
        <div class="toast-body">
            Hello, this is a basic toast!
        </div>
    </div>
</div>

Kod JavaScript


                <script>
                let toast1 = document.querySelector('#toast1');
                let t1 = new bootstrap.Toast(toast1);
                t1.show();
                </script>

Przykład 2: Toast z automatycznym ukrywaniem

Auto-Hide Toast 5 seconds
This toast will disappear after 5 seconds!

Kod HTML:


<div class="toast" role="alert" aria-live="assertive" aria-atomic="true" data-bs-autohide="true" data-bs-delay="5000">
    <div class="toast-header">
        <strong class="me-auto">Auto-Hide Toast</strong>
        <small class="text-muted">5 seconds</small>
        <button type="button" class="btn-close" data-bs-dismiss="toast" aria-label="Close"></button>
    </div>
    <div class="toast-body">
        This toast will disappear after 5 seconds!
    </div>
</div>

Kod JavaScript:


                <script>
                let toast2 = document.querySelector('#toast2');
                let t2 = new bootstrap.Toast(toast2);
                t2.show();
                </script>

Przykład 4: Toast z podpiętym JS

JavaScript Toast
This toast is triggered via JavaScript!

Kod HTML i JavaScript:


<button type="button" class="btn btn-info" id="showToastBtn">Pokaż toasta</button>
<div class="toast" role="alert" aria-live="assertive" aria-atomic="true" id="jsToast">
    <div class="toast-header">
        <strong class="me-auto">JavaScript Toast</strong>
        <button type="button" class="btn-close" data-bs-dismiss="toast" aria-label="Close"></button>
    </div>
    <div class="toast-body">
        This toast is triggered via JavaScript!
    </div>
</div>

Kod JavaScript:


<script>
document.getElementById('showToastBtn').addEventListener('click', function () {
    const toastEl = document.getElementById('jsToast');
    const toast = new bootstrap.Toast(toastEl);
    toast.show();
});
</script>
