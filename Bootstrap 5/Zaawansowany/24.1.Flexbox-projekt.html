Przykład 8: Fill, Grow, Shrink

Element A (flex-fill) Lorem ipsum dolor sit, amet consectetur adipisicing elit. Ea harum suscipit quas, repudiandae quisquam maiores perferendis explicabo mollitia nemo impedit voluptatibus earum alias eius nisi, ex eligendi, consectetur veritatis error?
Element B (flex-grow-1) Lorem ipsum dolor sit amet consectetur adipisicing elit. Hic illum inventore dolore placeat magnam! Deserunt officia perspiciatis iste explicabo earum amet perferendis, hic voluptatum rerum dolores architecto culpa mollitia ad.
Element C (flex-shrink-1)

Kod HTML:


            <!--flex-fill: Sprawia, że element zajmuje dostępne miejsce, ale inne elementy z flex-grow mogą ją "zabrać".
                flex-grow-1: Umożliwia Elementowi B zajmowanie większej ilości miejsca, gdy jest dodatkowa przestrzeń.
                flex-shrink-0: Powoduje, że Element C nie będzie się kurczyć.-->
<div class="d-flex" style="border: 1px solid #ccc; padding: 10px; width: 100%;">
    <div class="flex-example flex-fill">Element A (flex-fill) Lorem*1</div>
    <div class="flex-example flex-grow-1">Element B (flex-grow-1) Lorem*1</div>
    <div class="flex-example flex-shrink-0">Element C (flex-shrink-0)</div>
</div>

Przykład: Różne opcje Flex Wrap

Bez Flex Wrap (`flex-nowrap`)

Element A
Element B
Element C
Element D

Z Flex Wrap (`flex-wrap`)

Element A
Element B
Element C
Element D

Z Flex Wrap Reverse (`flex-wrap-reverse`)

Element A
Element B
Element C
Element D

Kod HTML:


            <!-- Bez Flex Wrap -->
            <div class="d-flex flex-nowrap">
                <div class="flex-example col">Element A</div>
                <div class="flex-example col">Element B</div>
                <div class="flex-example col">Element C</div>
                <div class="flex-example col">Element D</div>
            </div>
                        
            <!-- Z Flex Wrap -->
            <div class="d-flex flex-wrap">
                <div class="flex-example col">Element A</div>
                <div class="flex-example col">Element B</div>
                <div class="flex-example col">Element C</div>
                <div class="flex-example col">Element D</div>
            </div>
                        
            <!-- Z Flex Wrap Reverse -->
            <div class="d-flex flex-wrap-reverse">
                <div class="flex-example col">Element A</div>
                <div class="flex-example col">Element B</div>
                <div class="flex-example col">Element C</div>
                <div class="flex-example col">Element D</div>
            </div>
