Przykład 1: Podstawowy alert

To jest przykładowy podstawowy alert Bootstrap.

Kod HTML:


<div class="alert alert-primary" role="alert">
    To jest przykładowy podstawowy alert Bootstrap.
</div>

Przykład 2: Różne style alertów

To jest zielony alert sukcesu!
To jest czerwony alert błędu!
To jest żółty alert ostrzeżenia!
To jest niebieski alert informacyjny!

Kod HTML:


<div class="alert alert-success" role="alert">
    To jest zielony alert sukcesu!
</div>
<div class="alert alert-danger" role="alert">
    To jest czerwony alert błędu!
</div>
<div class="alert alert-warning" role="alert">
    To jest żółty alert ostrzeżenia!
</div>
<div class="alert alert-info" role="alert">
    To jest niebieski alert informacyjny!
</div>

Przykład 3: Alert z przyciskiem zamykania

Uwaga! To jest alert z przyciskiem zamykania.

Kod HTML:


<div class="alert alert-warning alert-dismissible fade show" role="alert">
    <strong>Uwaga!</strong> To jest alert z przyciskiem zamykania.
    <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>

Przykład 4: Dynamiczny alert tworzony za pomocą JavaScript

Kod HTML i JavaScript:


<button class="btn btn-success mb-3" id="createAlertBtn">Utwórz alert</button>
<div id="alert-container"></div>

<script>
document.getElementById('createAlertBtn').addEventListener('click', function() {
    const alertContainer = document.getElementById('alert-container');

    // Tablica z różnymi typami alertów Bootstrap
    const alertTypes = ['alert-info', 'alert-success', 'alert-warning', 'alert-danger'];

    // Tablica z różnymi wiadomościami
    const alertMessages = [
        'To jest informacyjny alert!',
        'Operacja zakończona sukcesem!',
        'Ostrzeżenie: coś poszło nie tak!',
        'Błąd: wystąpił problem!'
    ];

    // Losowe wybieranie typu alertu i wiadomości
    const randomAlertType = alertTypes[Math.floor(Math.random() * alertTypes.length)];
    const randomMessage = alertMessages[Math.floor(Math.random() * alertMessages.length)];

    // Tworzenie elementu alertu
    const alert = document.createElement('div');
    alert.className = `alert ${randomAlertType} alert-dismissible fade show`;
    alert.role = 'alert';
    alert.innerHTML = `${randomMessage} `;

    // Dodanie alertu do kontenera
    alertContainer.appendChild(alert);
});
</script>
