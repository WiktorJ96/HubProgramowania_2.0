Layout formularzy w Bootstrapie

Bootstrap oferuje różne sposoby na organizację układu formularzy. Możesz tworzyć formularze pionowe, poziome oraz inline, dostosowując je do różnych potrzeb użytkowników.

Pionowy Layout Formularza

Jest to domyślny układ formularza, w którym elementy są ustawione jeden pod drugim. Jest to najbardziej klasyczny i uniwersalny układ.


<!-- Pionowy Layout Formularza -->
<form>
  <div class="mb-3">
    <label for="exampleInputEmail1" class="form-label">Adres email</label>
    <input type="email" class="form-control" id="exampleInputEmail1" aria-describedby="emailHelp">
    <div id="emailHelp" class="form-text">Nigdy nie udostępnimy Twojego emaila.</div>
  </div>
  <div class="mb-3">
    <label for="exampleInputPassword1" class="form-label">Hasło</label>
    <input type="password" class="form-control" id="exampleInputPassword1">
  </div>
  <button type="submit" class="btn btn-primary">Zaloguj się</button>
</form>

Poziomy Layout Formularza

W układzie poziomym etykiety i pola są ustawione obok siebie, co oszczędza miejsce i wygląda bardziej kompaktowo. Aby utworzyć taki układ, używamy klas .row oraz .col-*.


<!-- Poziomy Layout Formularza -->
<form>
  <div class="row mb-3">
    <label for="inputEmail3" class="col-sm-2 col-form-label">Email</label>
    <div class="col-sm-10">
      <input type="email" class="form-control" id="inputEmail3">
    </div>
  </div>
  <div class="row mb-3">
    <label for="inputPassword3" class="col-sm-2 col-form-label">Hasło</label>
    <div class="col-sm-10">
      <input type="password" class="form-control" id="inputPassword3">
    </div>
  </div>
  <fieldset class="row mb-3">
    <legend class="col-form-label col-sm-2 pt-0">Płeć</legend>
    <div class="col-sm-10">
      <div class="form-check">
        <input class="form-check-input" type="radio" name="gridRadios" id="gridRadios1" value="option1" checked>
        <label class="form-check-label" for="gridRadios1">
          Mężczyzna
        </label>
      </div>
      <div class="form-check">
        <input class="form-check-input" type="radio" name="gridRadios" id="gridRadios2" value="option2">
        <label class="form-check-label" for="gridRadios2">
          Kobieta
        </label>
      </div>
    </div>
  </fieldset>
  <button type="submit" class="btn btn-primary">Zatwierdź</button>
</form>

Inline Layout Formularza

Formularze inline są stosowane, gdy chcesz, aby elementy formularza były ustawione w jednej linii, np. w paskach wyszukiwania lub szybkich formularzach logowania. Używamy klasy .row g-3 oraz .col-auto do tworzenia inline form.


<!-- Inline Layout Formularza -->
<form class="row g-3">
  <div class="col-auto">
    <label for="staticEmail2" class="visually-hidden">Email</label>
    <input type="text" class="form-control" id="staticEmail2" placeholder="Adres email">
  </div>
  <div class="col-auto">
    <label for="inputPassword2" class="visually-hidden">Hasło</label>
    <input type="password" class="form-control" id="inputPassword2" placeholder="Hasło">
  </div>
  <div class="col-auto">
    <button type="submit" class="btn btn-primary mb-3">Potwierdź</button>
  </div>
</form>

Rozciągliwe Elementy Formularza

Możesz użyć klas .w-100 oraz .w-auto, aby kontrolować szerokość elementów formularza, co pozwala na lepszą kontrolę nad ich rozmieszczeniem.


<!-- Rozciągliwe Elementy Formularza -->
<form class="d-flex">
  <input class="form-control me-2 w-50" type="search" placeholder="Szukaj" aria-label="Szukaj">
  <button class="btn btn-outline-success" type="submit">Szukaj</button>
</form>

Podsumowanie

  • Layout Formularzy w Bootstrapie oferuje elastyczne i estetyczne sposoby na organizację elementów formularza.
  • Można łatwo tworzyć układy pionowe, poziome oraz inline, co pozwala dostosować formularze do różnych potrzeb użytkowników.
  • Użycie odpowiednich klas, takich jak .row, .col-*, .d-flex i innych, zapewnia pełną kontrolę nad układem formularzy.
