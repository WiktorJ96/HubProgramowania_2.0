Przykłady

  • Podstawowy Popover
  • Popover z tytułem
  • Popover w różnych kierunkach

Przykład 1: Podstawowy Popover

Kod HTML:


<button type="button" class="btn btn-primary" data-bs-toggle="popover" data-bs-content="To jest przykładowy popover!">
    Kliknij, aby zobaczyć Popover
</button>

Kod JS:


<script>
document.addEventListener('DOMContentLoaded', function () {
            const popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'));
            const popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
                return new bootstrap.Popover(popoverTriggerEl);
            });
        });
</script>
