Listy w Bootstrapie

Komponent List Group w Bootstrapie jest przydatny do tworzenia grupowanych list elementów, takich jak nawigacja, menu lub listy zadań. Oferuje on wsparcie dla różnych stylizacji, układów i zawartości, takich jak tekst, obrazy czy elementy formularzy.

Podstawowy przykład listy


<ul class="list-group">
  <li class="list-group-item">Element listy 1</li>
  <li class="list-group-item">Element listy 2</li>
  <li class="list-group-item">Element listy 3</li>
</ul>

Podstawowy przykład listy grupowej z trzema elementami. Każdy element listy jest otoczony klasą .list-group-item.

Flush list (Lista flush)

Komponent List Group może być wyświetlany bez marginesów i obramowań między elementami, używając klasy .list-group-flush.


<ul class="list-group list-group-flush">
  <li class="list-group-item">Element 1</li>
  <li class="list-group-item">Element 2</li>
  <li class="list-group-item">Element 3</li>
</ul>

W tym przykładzie lista flush usuwa marginesy i obramowania między elementami listy, tworząc bardziej zwartą i uproszczoną wersję listy.

Listy poziome (Horizontal)

Lista pozioma umożliwia wyświetlanie elementów w jednym wierszu. Używa klasy .list-group-horizontal, ale działa ona tylko na szerokich ekranach.


<ul class="list-group list-group-horizontal">
  <li class="list-group-item">Element 1</li>
  <li class="list-group-item">Element 2</li>
  <li class="list-group-item">Element 3</li>
</ul>

Lista pozioma sprawia, że elementy są wyświetlane obok siebie, zamiast jeden pod drugim. Na urządzeniach mobilnych lista powróci do układu pionowego.

Responsywność listy poziomej

Możesz również tworzyć listy poziome, które będą działać tylko w określonych rozmiarach ekranu, np. .list-group-horizontal-sm, .list-group-horizontal-md, .list-group-horizontal-lg, itd.


<ul class="list-group list-group-horizontal-md">
  <li class="list-group-item">Element 1</li>
  <li class="list-group-item">Element 2</li>
  <li class="list-group-item">Element 3</li>
</ul>

W tym przykładzie lista pozioma działa tylko na średnich i większych ekranach (klasa .list-group-horizontal-md).

Kolorowe listy

Każdy element listy można pokolorować za pomocą klas takich jak list-group-item-primary, list-group-item-secondary, itd., podobnie jak w przypadku komunikatów błyskawicznych.


<ul class="list-group">
  <li class="list-group-item list-group-item-primary">Element niebieski (primary)</li>
  <li class="list-group-item list-group-item-secondary">Element szary (secondary)</li>
  <li class="list-group-item list-group-item-info">Element niebieski (info)</li>
</ul>

W tym przykładzie zastosowano różne klasy kolorystyczne, aby zmienić wygląd elementów listy.

Elementy formularza w listach (np. checkbox)

Można używać elementów formularza takich jak checkbox lub radio wewnątrz list grupowych, co umożliwia tworzenie interaktywnych list z wyborem.


<ul class="list-group">
  <li class="list-group-item">
    <input class="form-check-input me-1" type="checkbox" value=""> Element 1
  </li>
  <li class="list-group-item">
    <input class="form-check-input me-1" type="checkbox" value=""> Element 2
  </li>
  <li class="list-group-item">
    <input class="form-check-input me-1" type="checkbox" value=""> Element 3
  </li>
</ul>

W tym przykładzie elementy listy zawierają checkboxy, co umożliwia użytkownikowi zaznaczanie wybranych elementów.

Podsumowanie

  • List Group to elastyczny komponent Bootstrapu, który może być używany do tworzenia list z różnorodną zawartością.
  • Listy mogą być poziome, kolorowe, responsywne i mogą zawierać elementy formularza, takie jak checkboxy.
  • Komponent List Group wspiera różne style i układy, co czyni go idealnym do tworzenia złożonych interfejsów użytkownika.
