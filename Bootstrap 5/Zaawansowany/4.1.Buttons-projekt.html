Przykłady

  • Podstawowe przyciski
  • Przyciski outline
  • Przyciski nieaktywne
  • Podpięcie JS do przycisku

Przykład 1: Podstawowe przyciski

Kod HTML:


<button type="button" class="btn btn-primary">Przycisk podstawowy</button>
<button type="button" class="btn btn-secondary">Przycisk dodatkowy</button>
<button type="button" class="btn btn-success">Przycisk sukces</button>

Przykład 2: Przyciski w wersji outline

Kod HTML:


<button type="button" class="btn btn-outline-primary">Przycisk outline primary</button>
<button type="button" class="btn btn-outline-secondary">Przycisk outline secondary</button>
<button type="button" class="btn btn-outline-success">Przycisk outline success</button>

Przykład 3: Przyciski nieaktywne (disabled)

Kod HTML:


<button type="button" class="btn btn-primary" disabled>Przycisk primary (disabled)</button>
<button type="button" class="btn btn-secondary" disabled>Przycisk secondary (disabled)</button>
<button type="button" class="btn btn-success" disabled>Przycisk success (disabled)</button>

Przykład 4: Podpięcie JS do przycisku za pomocą Bootstrap

To jest przykładowy alert!

Kod HTML i JavaScript:


<button type="button" class="btn btn-info" id="alertButton">Pokaż alert</button>
<div class="alert alert-warning alert-dismissible fade show mt-3 d-none" role="alert" id="alertExample">
    To jest przykładowy alert!
    <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
</div>

<script>
document.getElementById('alertButton').addEventListener('click', function() {
    const alertEl = document.getElementById('alertExample');
    alertEl.classList.remove('d-none');
    new bootstrap.Alert(alertEl);
});
</script>
