Przykłady Grup Wejściowych w Bootstrapie

  • Prosta grupa wejściowa
  • Grupa z dodatkowymi tekstami
  • Grupa z przyciskiem
  • Grupa z przyciskiem rozwijanym
  • Grupa z plikiem wejściowym

Przykład 1: Prosta grupa wejściowa

@

Kod HTML:


<div class="input-group mb-3">
    <span class="input-group-text">@</span>
    <input type="text" class="form-control" placeholder="Nazwa użytkownika">
</div>

Przykład 2: Grupa z dodatkowymi tekstami

https://example.com
zł

Kod HTML:


<div class="input-group mb-3">
    <span class="input-group-text">https://example.com</span>
    <input type="text" class="form-control" placeholder="Twoja strona">
</div>
<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Kwota">
    <span class="input-group-text">zł</span>
</div>

Przykład 3: Grupa z przyciskiem

Kod HTML:


<div class="input-group mb-3">
    <input type="text" class="form-control" placeholder="Odbiorca">
    <button class="btn btn-outline-secondary" type="button">Wyślij</button>
</div>

Przykład 4: Grupa z przyciskiem rozwijanym

  • Opcja 1
  • Opcja 2
  • Opcja 3

Kod HTML:


<div class="input-group mb-3">
    <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">Akcja</button>
    <ul class="dropdown-menu">
        <li><a class="dropdown-item" href="#">Opcja 1</a></li>
        <li><a class="dropdown-item" href="#">Opcja 2</a></li>
        <li><a class="dropdown-item" href="#">Opcja 3</a></li>
    </ul>
    <input type="text" class="form-control" placeholder="Wybierz akcję">
</div>

Przykład 5: Grupa z plikiem wejściowym

Kod HTML:


<div class="input-group mb-3">
    <input type="file" class="form-control" id="inputGroupFile02">
    <label class="input-group-text" for="inputGroupFile02">Wgraj</label>
</div>
