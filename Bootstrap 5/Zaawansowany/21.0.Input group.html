Input Group w Bootstrapie

Komponent Input Group w Bootstrapie pozwala na połączenie różnych elementów formularza, takich jak pola tekstowe, przyciski, dropdowny i inne, aby stworzyć bardziej funkcjonalne układy. Ułatwia to tworzenie złożonych formularzy z jednolitą estetyką.

Podstawowa Input Group

Przykład podstawowej grupy pól, w której dodano przedrostek do pola tekstowego.


<!-- Podstawowa Input Group -->
<div class="input-group mb-3">
  <span class="input-group-text" id="basic-addon1">@</span>
  <input type="text" class="form-control" placeholder="Nazwa użytkownika" aria-label="Nazwa użytkownika" aria-describedby="basic-addon1">
</div>

Input Group z Przyciskiem

Można łatwo połączyć przycisk z polem tekstowym.


<!-- Input Group z przyciskiem -->
<div class="input-group mb-3">
  <input type="text" class="form-control" placeholder="Odbiorca" aria-label="Odbiorca">
  <button class="btn btn-outline-secondary" type="button">Wyślij</button>
</div>

Input Group z Dropdown

Input Group może również zawierać rozwijane menu (dropdown).


<!-- Input Group z dropdown -->
<div class="input-group mb-3">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">Opcje</button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Opcja 1</a></li>
    <li><a class="dropdown-item" href="#">Opcja 2</a></li>
    <li><a class="dropdown-item" href="#">Opcja 3</a></li>
  </ul>
  <input type="text" class="form-control" aria-label="Text input with dropdown button">
</div>

Input Group z Przedrostkiem i Przyrostkiem

Można dodać zarówno przedrostek, jak i przyrostek do pola tekstowego, co jest przydatne np. w przypadku walut lub jednostek.


<!-- Input Group z przedrostkiem i przyrostkiem -->
<div class="input-group mb-3">
  <span class="input-group-text">$</span>
  <input type="text" class="form-control" aria-label="Kwota (dolar amerykański)">
  <span class="input-group-text">.00</span>
</div>

Input Group z Wyborem Pliku

Można łatwo dodać pole wyboru pliku do Input Group, aby umożliwić użytkownikom dodawanie plików.


<!-- Input Group z wyborem pliku -->
<div class="input-group mb-3">
  <input type="file" class="form-control" id="inputGroupFile02">
  <label class="input-group-text" for="inputGroupFile02">Prześlij</label>
</div>

Wielokrotne Elementy w Input Group

Bootstrap pozwala na dodanie wielu elementów do jednej grupy, co jest przydatne w bardziej złożonych formularzach.


<!-- Wielokrotne elementy w Input Group -->
<div class="input-group">
  <span class="input-group-text">Imię i nazwisko</span>
  <input type="text" aria-label="Imię" class="form-control">
  <input type="text" aria-label="Nazwisko" class="form-control">
</div>

Podsumowanie

  • Input Group w Bootstrapie umożliwia łatwe łączenie różnych elementów formularza w jedną funkcjonalną grupę.
  • Można używać przedrostków, przycisków, dropdownów, przyrostków i pól wyboru plików, aby stworzyć bardziej intuicyjne formularze.
  • Input Group jest elastyczny i pozwala na różne kombinacje elementów, co ułatwia tworzenie bardziej złożonych interfejsów.
