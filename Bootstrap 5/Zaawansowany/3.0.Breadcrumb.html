Nawigacja breadcrumb w Bootstrapie

Nawigacja okruszkowa (breadcrumb) pozwala użytkownikowi śledzić swoją pozycję w strukturze strony. Jest szczególnie przydatna na stronach z wielopoziomowymi ścieżkami nawigacyjnymi, np. w sklepach internetowych czy stronach z dokumentacją.

Podstawowe użycie breadcrumb

Aby stworzyć podstawową nawigację okruszkową, użyj elementu <nav> i klasy .breadcrumb. Poszczególne kroki nawigacji są oddzielone za pomocą <li> i klasy .breadcrumb-item. Bootstrap automatycznie stylizuje separator pomiędzy krokami (domyślnie jest to znak "/").

Przykład podstawowej nawigacji okruszkowej


<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Strona główna</a></li>
    <li class="breadcrumb-item"><a href="#">Biblioteka</a></li>
    <li class="breadcrumb-item active" aria-current="page">Dane</li>
  </ol>
</nav>

W powyższym przykładzie, dwie pierwsze pozycje breadcrumb są klikalne, a ostatnia pozycja jest oznaczona jako bieżąca lokalizacja za pomocą klasy .active i atrybutu aria-current="page".

Zamiana separatora

Domyślny separator pomiędzy pozycjami breadcrumb to ukośnik (/). Możesz zmienić separator, modyfikując CSS lub stosując dodatkowe klasy. Poniżej znajduje się przykład zamiany separatora na ">".

Przykład zmiany separatora


<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Strona główna</a></li>
    <li class="breadcrumb-item"><a href="#">Biblioteka</a></li>
    <li class="breadcrumb-item active" aria-current="page">Dane</li>
  </ol>
</nav>

<style>
.breadcrumb-item + .breadcrumb-item::before {
  content: ">";
  padding: 0 5px;
  color: #6c757d;
}
</style>

W powyższym przykładzie, znak ukośnika jest zastąpiony znakiem ">", co można uzyskać za pomocą selektora .breadcrumb-item + .breadcrumb-item::before.

Responsywność breadcrumb

Nawigacja okruszkowa jest domyślnie responsywna, ponieważ dostosowuje się do szerokości ekranu, co oznacza, że dobrze działa zarówno na urządzeniach mobilnych, jak i stacjonarnych.

Przykład responsywnej breadcrumb


<nav aria-label="breadcrumb">
  <ol class="breadcrumb">
    <li class="breadcrumb-item"><a href="#">Strona główna</a></li>
    <li class="breadcrumb-item"><a href="#">Sklep</a></li>
    <li class="breadcrumb-item"><a href="#">Elektronika</a></li>
    <li class="breadcrumb-item active" aria-current="page">Telewizory</li>
  </ol>
</nav>

Nawigacja breadcrumb automatycznie dostosuje się do zmieniającej się szerokości ekranu, dzięki czemu jest czytelna na urządzeniach mobilnych.

Inne style breadcrumb

Można stylizować breadcrumb za pomocą własnych stylów CSS lub dodatkowych klas. Na przykład, można zmienić kolor tła, czcionkę czy odstępy między elementami breadcrumb.

Przykład breadcrumb z niestandardowym stylem


<nav aria-label="breadcrumb">
  <ol class="breadcrumb bg-light p-3 rounded">
    <li class="breadcrumb-item"><a href="#">Strona główna</a></li>
    <li class="breadcrumb-item"><a href="#">Kategorie</a></li>
    <li class="breadcrumb-item active" aria-current="page">Produkty</li>
  </ol>
</nav>

W tym przykładzie dodano klasy Bootstrap .bg-light, .p-3, i .rounded dla zmiany tła, dodania wypełnienia i zaokrąglenia krawędzi breadcrumb.

Podsumowanie

  • Breadcrumb to prosty sposób na nawigację użytkownika po hierarchii strony.
  • Bootstrap oferuje proste klasy do stylizacji breadcrumb, a także możliwość zmiany separatorów i dodawania własnych stylów.
  • Breadcrumb jest w pełni responsywny i łatwo dostosowuje się do różnych rozdzielczości ekranu.
