Przykłady

  • Podstawowa lista
  • Flush lista
  • Pozioma lista
  • Responsywna lista
  • Kolorowe listy
  • Lista z elementami formularza

Przykład 1: Podstawowa lista

  • Pierwszy element
  • Drugi element
  • Trzeci element

Kod HTML:


<ul class="list-group">
    <li class="list-group-item">Pierwszy element</li>
    <li class="list-group-item">Drugi element</li>
    <li class="list-group-item">Trzeci element</li>
</ul>

Przykład 2: Flush lista

  • Element 1
  • Element 2
  • Element 3

Kod HTML:


<ul class="list-group list-group-flush">
    <li class="list-group-item">Element 1</li>
    <li class="list-group-item">Element 2</li>
    <li class="list-group-item">Element 3</li>
</ul>

Przykład 3: Pozioma lista

  • Element 1
  • Element 2
  • Element 3

Kod HTML:


<ul class="list-group list-group-horizontal">
    <li class="list-group-item">Element 1</li>
    <li class="list-group-item">Element 2</li>
    <li class="list-group-item">Element 3</li>
</ul>

Przykład 4: Responsywna lista

  • Element 1
  • Element 2
  • Element 3

Na mniejszych ekranach lista będzie wyświetlana pionowo.

Kod HTML:


<ul class="list-group list-group-horizontal-md">
    <li class="list-group-item">Element 1</li>
    <li class="list-group-item">Element 2</li>
    <li class="list-group-item">Element 3</li>
</ul>

Przykład 5: Kolorowe listy

  • Primary element
  • Success element
  • Danger element

Kod HTML:


<ul class="list-group">
    <li class="list-group-item list-group-item-primary">Primary element</li>
    <li class="list-group-item list-group-item-success">Success element</li>
    <li class="list-group-item list-group-item-danger">Danger element</li>
</ul>

Przykład 6: Lista z elementami formularza (checkbox)

Kod HTML:


<ul class="list-group">
    <li class="list-group-item">
        <input class="form-check-input me-1" type="checkbox" value="" id="firstCheckbox">
        <label class="form-check-label" for="firstCheckbox">Pierwszy element</label>
    </li>
    <li class="list-group-item">
        <input class="form-check-input me-1" type="checkbox" value="" id="secondCheckbox">
        <label class="form-check-label" for="secondCheckbox">Drugi element</label>
    </li>
    <li class="list-group-item">
        <input class="form-check-input me-1" type="checkbox" value="" id="thirdCheckbox">
        <label class="form-check-label" for="thirdCheckbox">Trzeci element</label>
    </li>
</ul>
