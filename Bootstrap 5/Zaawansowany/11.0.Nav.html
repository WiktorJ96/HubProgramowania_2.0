Nav w Bootstrapie

Komponent Nav w Bootstrapie umożliwia tworzenie nawigacji poziomych i pionowych, zakładek (tabs), rozwijanych menu oraz nawigacji z nieaktywnymi elementami. Może być używany do budowy menu stron, zakładek w panelach administracyjnych lub nawigacji po sekcjach na stronie.

Podstawowa nawigacja pozioma

Tworzenie podstawowej nawigacji poziomej za pomocą komponentu Nav.


<ul class="nav">
  <li class="nav-item">
    <a class="nav-link active" href="#">Aktywna</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Inny link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Nieaktywny</a>
  </li>
</ul>

W tym przykładzie mamy prostą nawigację poziomą z aktywnym elementem, dwoma zwykłymi linkami oraz jednym nieaktywnym linkiem, używając klasy .disabled.

Podstawowa nawigacja pionowa

Przykład nawigacji pionowej, używając klasy .flex-column.


<ul class="nav flex-column">
  <li class="nav-item">
    <a class="nav-link active" href="#">Aktywna</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Inny link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link disabled" href="#" tabindex="-1" aria-disabled="true">Nieaktywny</a>
  </li>
</ul>

Nawigacja pionowa jest tworzona za pomocą klasy .flex-column, która ustawia elementy w kolumnie.

Tabs (nawigacja w formie zakładek)

Nawigacja w formie zakładek, gdzie aktywny element reprezentuje obecnie wybraną zakładkę.


<ul class="nav nav-tabs">
  <li class="nav-item">
    <a class="nav-link active" aria-current="page" href="#">Zakładka 1</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Zakładka 2</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Zakładka 3</a>
  </li>
</ul>

Nawigacja z zakładkami (tabs) używa klasy .nav-tabs, która tworzy wizualny układ w formie zakładek, typowy dla interfejsów aplikacji.

Nawigacja justyfikowana (rozciągnięta na całą szerokość)

Justyfikowana nawigacja, która rozciąga elementy na całą szerokość kontenera.


<ul class="nav nav-pills nav-justified">
  <li class="nav-item">
    <a class="nav-link active" href="#">Aktywna</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Link</a>
  </li>
  <li class="nav-item">
    <a class="nav-link" href="#">Inny link</a>
  </li>
</ul>

Justyfikowana nawigacja (.nav-justified) powoduje, że elementy zajmują równą szerokość, rozciągając się na całą dostępność przestrzeń.

Nawigacja z dropdown

Tworzenie rozwijanego menu w nawigacji za pomocą komponentu .dropdown.


<ul class="nav nav-pills">
  <li class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown" href="#" role="button" aria-expanded="false">Menu</a>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">Opcja 1</a></li>
      <li><a class="dropdown-item" href="#">Opcja 2</a></li>
      <li><a class="dropdown-item" href="#">Opcja 3</a></li>
    </ul>
  </li>
</ul>

Responsywna nawigacja z hamburger menu

W przypadku mniejszych ekranów nawigacja może zostać ukryta i wyświetlana jako rozwijane hamburger menu. Poniższy przykład pokazuje, jak to osiągnąć za pomocą komponentu Navbar w Bootstrapie.


<nav class="navbar navbar-expand-lg navbar-light bg-light">
  <div class="container-fluid">
    <a class="navbar-brand" href="#">Logo</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarNav">
      <ul class="navbar-nav">
        <li class="nav-item">
          <a class="nav-link active" aria-current="page" href="#">Home</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Features</a>
        </li>
        <li class="nav-item">
          <a class="nav-link" href="#">Pricing</a>
        </li>
        <li class="nav-item dropdown">
          <a class="nav-link dropdown-toggle" href="#" id="navbarDropdown" role="button" data-bs-toggle="dropdown" aria-expanded="false">
            Menu rozwijane
          </a>
          <ul class="dropdown-menu" aria-labelledby="navbarDropdown">
            <li><a class="dropdown-item" href="#">Opcja 1</a></li>
            <li><a class="dropdown-item" href="#">Opcja 2</a></li>
            <li><hr class="dropdown-divider"></li>
            <li><a class="dropdown-item" href="#">Inna opcja</a></li>
          </ul>
        </li>
      </ul>
    </div>
  </div>
</nav>

W tym przykładzie używamy klasy navbar-expand-lg, co oznacza, że nawigacja jest rozwinięta tylko na większych ekranach. Na mniejszych ekranach jest ukryta za ikoną hamburgera. Komponent navbar-toggler służy do rozwijania menu.

Ten przykład pokazuje, jak zintegrować rozwijane menu w nawigacji, używając klasy .dropdown i .dropdown-menu dla opcji w menu.

Podsumowanie

  • Nav w Bootstrapie jest elastycznym komponentem do budowania nawigacji poziomych, pionowych, justyfikowanych, zakładek (tabs), a także z dropdownami.
  • Można dodawać nieaktywne linki (disabled) oraz rozwijane menu do nawigacji.
  • Użycie klas .nav-tabs i .nav-pills pozwala na tworzenie zakładek i nawigacji w formie "pigułek".
