Tworzenie wielostronicowych aplikacji za pomocą React Router

W React, możemy tworzyć **wielostronicowe aplikacje** dzięki bibliotece **React Router**. React Router umożliwia nawigację między różnymi widokami (stronami) bez potrzeby przeładowywania całej aplikacji. Dzięki temu użytkownicy mogą nawigować po stronie w bardziej płynny sposób.

Instalacja React Router

Aby używać React Router w swojej aplikacji, musisz najpierw zainstalować pakiet react-router-dom. Możesz to zrobić, wpisując następującą komendę w terminalu:


        npm install react-router-dom

Podstawowa struktura React Router

Podstawowym elementem React Router jest komponent Router, który zarządza trasami (ścieżkami) w aplikacji. Dwa najczęściej używane komponenty to BrowserRouter i Route.

  • BrowserRouter – używany do zarządzania trasami w oparciu o adresy URL (najczęściej używany rodzaj routera).
  • Route – definiuje ścieżki, które renderują określone komponenty, w zależności od adresu URL.

Tworzenie podstawowej aplikacji z React Router

Poniżej znajduje się przykład prostej aplikacji wielostronicowej z trzema stronami: Home, About i Contact.


        import React from 'react';
        import { BrowserRouter as Router, Route, Switch } from 'react-router-dom';

        function Home() {
          return <h2>Strona Główna</h2>;
        }

        function About() {
          return <h2>O nas</h2>;
        }

        function Contact() {
          return <h2>Kontakt</h2>;
        }

        function App() {
          return (
            <Router>
              <div>
                <nav>
                  <ul>
                    <li>
                      <a href="/">Strona Główna</a>
                    </li>
                    <li>
                      <a href="/about">O nas</a>
                    </li>
                    <li>
                      <a href="/contact">Kontakt</a>
                    </li>
                  </ul>
                </nav>

                <Switch>
                  <Route path="/about">
                    <About />
                  </Route>
                  <Route path="/contact">
                    <Contact />
                  </Route>
                  <Route path="/" exact>
                    <Home />
                  </Route>
                </Switch>
              </div>
            </Router>
          );
        }

        export default App;

W tym przykładzie używamy komponentu Router do zarządzania trasami. Każda trasa jest definiowana przez komponent Route, a Switch zapewnia, że tylko jedna trasa jest renderowana jednocześnie. Każda trasa renderuje inny komponent: Home, About lub Contact.

Linki nawigacyjne

Zamiast używać standardowych linków HTML (<a>), React Router oferuje komponent Link, który pozwala na nawigację między stronami bez przeładowywania strony. Oto jak możemy zamienić linki na Link:


        import { Link } from 'react-router-dom';

        function App() {
          return (
            <Router>
              <div>
                <nav>
                  <ul>
                    <li>
                      <Link to="/">Strona Główna</Link>
                    </li>
                    <li>
                      <Link to="/about">O nas</Link>
                    </li>
                    <li>
                      <Link to="/contact">Kontakt</Link>
                    </li>
                  </ul>
                </nav>

                <Switch>
                  <Route path="/about">
                    <About />
                  </Route>
                  <Route path="/contact">
                    <Contact />
                  </Route>
                  <Route path="/" exact>
                    <Home />
                  </Route>
                </Switch>
              </div>
            </Router>
          );
        }

Komponent Link zapewnia lepszą nawigację, ponieważ nie przeładowuje całej strony, a jedynie zmienia adres URL i renderuje odpowiednią trasę.

Nawigacja dynamiczna

React Router obsługuje również dynamiczne ścieżki, które mogą się zmieniać w zależności od parametrów URL. Oto przykład, w którym dynamicznie renderujemy stronę profilu użytkownika na podstawie ID w URL.


        import React from 'react';
        import { BrowserRouter as Router, Route, Switch, useParams } from 'react-router-dom';

        function UserProfile() {
          const { id } = useParams();
          return <h2>Profil użytkownika: {id}</h2>;
        }

        function App() {
          return (
            <Router>
              <div>
                <Switch>
                  <Route path="/user/:id">
                    <UserProfile />
                  </Route>
                </Switch>
              </div>
            </Router>
          );
        }

        export default App;

W tym przykładzie komponent UserProfile korzysta z hooka useParams, aby uzyskać ID użytkownika z dynamicznej ścieżki. Jeśli użytkownik wejdzie na stronę /user/123, zostanie wyświetlony komunikat "Profil użytkownika: 123".

Przekierowania

React Router pozwala również na przekierowywanie użytkowników z jednej trasy na inną. Możemy to osiągnąć za pomocą komponentu Redirect.


        import { Redirect } from 'react-router-dom';

        function App() {
          return (
            <Router>
              <Switch>
                <Route path="/home">
                  <Redirect to="/" />
                </Route>
                <Route path="/" exact>
                  <Home />
                </Route>
              </Switch>
            </Router>
          );
        }

        export default App;

W tym przykładzie użytkownik, który wejdzie na ścieżkę /home, zostanie automatycznie przekierowany na stronę główną /.

Podsumowanie

React Router to potężne narzędzie, które umożliwia tworzenie wielostronicowych aplikacji w React. Dzięki takim funkcjom jak dynamiczne trasy, przekierowania czy łatwe zarządzanie linkami nawigacyjnymi, React Router pozwala na efektywne budowanie aplikacji SPA (Single Page Application), które są bardziej intuicyjne i płynne dla użytkowników.
