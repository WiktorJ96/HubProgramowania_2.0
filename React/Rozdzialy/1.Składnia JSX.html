Składnia JSX

JSX to rozszerzenie składni JavaScript, które umożliwia pisanie elementów podobnych do HTML bezpośrednio w kodzie JavaScript. JSX jest często wykorzystywany w React do tworzenia komponentów interfejsu użytkownika.

Wprowadzenie do JSX

JSX pozwala na wstawianie kodu HTML bezpośrednio w JavaScript i dodawanie wyrażeń JavaScript w HTML przy użyciu składni {}. JSX musi być przetworzony przez Babel, aby działał w przeglądarkach.


// JSX pozwala na pisanie kodu przypominającego HTML w JavaScript
const element = <h1>Witaj, świecie!</h1>;

// JSX wspiera wyrażenia JavaScript wewnątrz nawiasów {}
const user = "Jan";
const greeting = <h2>Witaj, {user}!</h2>;
console.log(greeting);

Pisanie komponentów w JSX

Komponenty w React można tworzyć jako funkcje lub klasy. Funkcja komponentu jest prostą funkcją JavaScript, która zwraca JSX. Oto przykład:


// Przykład prostego komponentu w JSX
function Welcome(props) {
    return <h1>Witaj, {props.name}!</h1>;
}

// Użycie komponentu
const element = <Welcome name="Jan" />;
console.log(element);

Wstawianie wyrażeń JavaScript w JSX

W JSX można wstawiać wyrażenia JavaScript w nawiasach klamrowych {}. Jest to przydatne, gdy chcemy dynamicznie generować zawartość.


// Przykład wstawiania wyrażeń w JSX
const user = { firstName: 'Anna', lastName: 'Kowalska' };

function formatName(user) {
    return user.firstName + ' ' + user.lastName;
}

const element = <h1>Witaj, {formatName(user)}!</h1>;
console.log(element);

Atrybuty w JSX

Podobnie jak w HTML, w JSX można używać atrybutów, ale niektóre z nich mają inne nazwy, jak na przykład className zamiast class.


// Dodawanie atrybutów w JSX
const element = <img src="zdjecie.jpg" alt="Przykładowe zdjęcie" className="obrazek" />;
