React - React Router

React Router to biblioteka, która umożliwia dodanie routingu (nawigacji) w aplikacji React. Dzięki niej możemy tworzyć wielostronicowe aplikacje, gdzie różne adresy URL wyświetlają różne komponenty bez przeładowania strony.

Instalacja React Router

Aby używać React Router, należy najpierw zainstalować pakiet react-router-dom:


npm install react-router-dom

Podstawowa Konfiguracja

Podstawowe elementy React Router to BrowserRouter, Routes, i Route. BrowserRouter jest kontenerem dla całej aplikacji, a Routes zawiera poszczególne Route, które określają, jakie komponenty mają być renderowane dla określonych ścieżek.


import React from 'react';
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Home from './Home';
import About from './About';
import Contact from './Contact';

function App() {
    return (
        <BrowserRouter>
            <Routes>
                <Route path="/" element=<Home /> />
                <Route path="/about" element=<About /> />
                <Route path="/contact" element=<Contact /> />
            </Routes>
        </BrowserRouter>
    );
}

export default App;

W powyższym przykładzie każda ścieżka (path) renderuje inny komponent. "/" wyświetla komponent Home, "/about" wyświetla About, a "/contact" wyświetla Contact.

Nawigacja

Aby poruszać się między różnymi ścieżkami w aplikacji, używamy komponentu Link zamiast tradycyjnego HTML-owego linku <a>. Pozwala to na zmianę ścieżek bez przeładowania strony.


import React from 'react';
import { Link } from 'react-router-dom';

function Navbar() {
    return (
        <nav>
            <ul>
                <li><Link to="/">Strona główna</Link></li>
                <li><Link to="/about">O nas</Link></li>
                <li><Link to="/contact">Kontakt</Link></li>
            </ul>
        </nav>
    );
}

export default Navbar;

Komponent Link zmienia adres URL i renderuje odpowiedni komponent, bez odświeżania całej strony.

Komponent Navigate

React Router umożliwia również przekierowywanie użytkowników na inną stronę za pomocą komponentu Navigate. Przykładem może być przekierowanie na stronę główną po wykonaniu akcji:


import React from 'react';
import { Navigate } from 'react-router-dom';

function RedirectToHome() {
    return <Navigate to="/" />;
}

W tym przykładzie komponent RedirectToHome automatycznie przekieruje użytkownika na stronę główną ("/").
