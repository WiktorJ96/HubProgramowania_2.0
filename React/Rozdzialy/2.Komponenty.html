Komponenty w React - Funkcyjne i Klasowe

Komponenty są podstawowymi blokami budującymi aplikacje w React. W React możemy tworzyć komponenty funkcyjne lub klasowe.

Komponenty Funkcyjne

Komponenty funkcyjne to proste funkcje JavaScript, które przyjmują props jako argument i zwracają JSX. Są preferowane w nowoczesnych aplikacjach React.


// Przykład komponentu funkcyjnego
function Welcome(props) {
    return <h1>Witaj, {props.name}!</h1>;
}

// Użycie komponentu
const element = <Welcome name="Jan" />;

Komponenty Klasowe

Komponenty klasowe to starszy sposób tworzenia komponentów w React. Muszą dziedziczyć po React.Component i zawierać metodę render(), która zwraca JSX.


// Przykład komponentu klasowego
import React, { Component } from 'react';

class Welcome extends Component {
    render() {
        return <h1>Witaj, {this.props.name}!</h1>;
    }
}

// Użycie komponentu
const element = <Welcome name="Jan" />;
