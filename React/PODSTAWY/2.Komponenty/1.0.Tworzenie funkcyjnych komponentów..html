Tworzenie Funkcyjnych Komponentów w React

Funkcyjne komponenty to jeden z podstawowych sposobów tworzenia komponentów w React. Są to proste funkcje, które przyjmują dane wejściowe (tzw. props) i zwracają elementy JSX. Dzięki funkcjom można łatwo zarządzać interfejsem użytkownika i dynamicznie generować elementy strony.

Prosty komponent funkcyjny

Funkcyjny komponent React to funkcja, która zwraca element JSX. Oto przykład:


// Prosty komponent funkcyjny
function Welcome(props) {
  return <h1>Witaj, {props.name}!</h1>;
}

// Użycie komponentu
ReactDOM.render(<Welcome name="Kamil" />, document.getElementById('root'));

Komponenty z props

Funkcyjne komponenty mogą przyjmować argumenty w postaci props, które pozwalają na przekazywanie danych do komponentu. Oto przykład:


// Komponent z props
function Greeting(props) {
  return <h1>Cześć, {props.name}! Jak się masz?</h1>;
}

// Użycie komponentu
ReactDOM.render(<Greeting name="Magda" />, document.getElementById('root'));

Zarządzanie stanem w komponentach funkcyjnych

Dzięki hookowi useState komponenty funkcyjne mogą zarządzać stanem. Oto przykład:


// Importowanie hooka useState
import React, { useState } from 'react';

function Counter() {
  // Deklaracja stanu z początkową wartością 0
  const [count, setCount] = useState(0);

  return (
    <div>
      <p>Kliknięto {count} razy</p>
      <button onClick={() => setCount(count + 1)}>Kliknij mnie</button>
    </div>
  );
}

// Użycie komponentu
ReactDOM.render(<Counter />, document.getElementById('root'));

Efekty uboczne w komponentach funkcyjnych

W komponentach funkcyjnych można używać hooka useEffect do zarządzania efektami ubocznymi. Oto przykład:


// Importowanie hooków useState i useEffect
import React, { useState, useEffect } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  // Hook useEffect do aktualizacji tytułu dokumentu
  useEffect(() => {
    document.title = `Kliknięto {count} razy`;
  });

  return (
    <div>
      <p>Kliknięto {count} razy</p>
      <button onClick={() => setCount(count + 1)}>Kliknij mnie</button>
    </div>
  );
}

// Użycie komponentu
ReactDOM.render(<Counter />, document.getElementById('root'));

Kiedy używać komponentów funkcyjnych?

Komponenty funkcyjne są zazwyczaj preferowane w nowoczesnych aplikacjach React, ponieważ są prostsze, bardziej zwięzłe i mogą zarządzać stanem oraz efektami ubocznymi dzięki hookom.

Podsumowanie

Funkcyjne komponenty to podstawowy sposób tworzenia komponentów w React. Są proste, elastyczne i umożliwiają zarządzanie stanem oraz efektami ubocznymi dzięki hookom, takim jak useState i useEffect. Dzięki temu są preferowane w nowoczesnych aplikacjach React.
