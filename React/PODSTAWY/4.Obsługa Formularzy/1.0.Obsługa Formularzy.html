Tworzenie prostego formularza w React

Tworzenie formularzy w React jest łatwe dzięki możliwości zarządzania stanem i reagowania na zmiany w polach formularza. Formularze są kluczowym elementem w każdej aplikacji, która wymaga interakcji z użytkownikiem.

Podstawowy formularz

Poniżej znajduje się przykład prostego formularza React, który pozwala użytkownikowi na wpisanie imienia. Komponent zarządza stanem, aby zapisać wartość wprowadzoną w polu tekstowym.


// Importowanie hooka useState
import React, { useState } from 'react';

function SimpleForm() {
  // Inicjalizacja stanu
  const [name, setName] = useState('');

  // Funkcja obsługująca zmiany w polu tekstowym
  const handleChange = (event) => {
    setName(event.target.value);
  };

  // Funkcja obsługująca przesłanie formularza
  const handleSubmit = (event) => {
    event.preventDefault();
    alert(`Witaj, ${name}!`);
  };

  return (
    <form onSubmit={handleSubmit}>
      <label>
        Imię:
        <input type="text" value={name} onChange={handleChange} />
      </label>
      <button type="submit">Wyślij</button>
    </form>
  );
}

// Renderowanie formularza
ReactDOM.render(<SimpleForm />, document.getElementById('root'));

Objaśnienie działania

W powyższym przykładzie:

  • Stworzono stan name, który przechowuje aktualną wartość wpisaną przez użytkownika w polu tekstowym.
  • Funkcja handleChange jest wywoływana za każdym razem, gdy użytkownik wpisuje coś w pole. Aktualizuje ona stan name.
  • Funkcja handleSubmit obsługuje przesłanie formularza. Zamiast odświeżać stronę, wyświetla alert z wpisanym imieniem.

Dodawanie dodatkowych pól do formularza

Formularze mogą zawierać więcej niż jedno pole. Możemy rozszerzyć powyższy przykład, dodając pole na wiek. Każde pole będzie miało swoje własne zarządzanie stanem:


// Importowanie hooka useState
import React, { useState } from 'react';

function ExtendedForm() {
  // Inicjalizacja stanu dla imienia i wieku
  const [name, setName] = useState('');
  const [age, setAge] = useState('');

    setName(event.target.value);

  };

  // Funkcja obsługująca przesłanie formularza
  const handleSubmit = (event) => {
    event.preventDefault();
    alert(`Witaj, ${name}! Masz ${age} lat.`);
  };

<form onSubmit={handleSubmit}>
  <label>
    Imię:
    <input type="text" value={name} onChange={handleNameChange} />
  </label>
  <br />
  <label>
    Wiek:
    <input type="number" value={age} onChange={handleAgeChange} />
  </label>
  <br />
  <button type="submit">Wyślij</button>
</form>



  );

Objaśnienie dodatkowych pól

W tym przykładzie dodano nowe pole Wiek. Każde pole ma własną funkcję zarządzającą zmianą stanu, która aktualizuje odpowiednią wartość w momencie wprowadzania danych.

Obsługa wielu pól za pomocą jednego stanu

Zamiast używać oddzielnego stanu dla każdego pola, można zarządzać wieloma polami za pomocą jednego obiektu w stanie. Oto przykład:


// Importowanie hooka useState
import React, { useState } from 'react';

function UnifiedForm() {
  // Jeden stan dla wszystkich pól
  const [formData, setFormData] = useState({ name: '', age: '' });

  // Funkcja obsługująca zmiany w polach
  const handleChange = (event) => {
    const { name, value } = event.target;
    setFormData((prevData) => ({
      ...prevData,
      [name]: value,
    }));
  };

  // Funkcja obsługująca przesłanie formularza
  const handleSubmit = (event) => {
    event.preventDefault();
    alert(`Witaj, ${formData.name}! Masz ${formData.age} lat.`);
  };

  return (
    

<form onSubmit={handleSubmit}>
  <label>
    Imię:
    <input
      type="text"
      name="name"
      value={formData.name}
      onChange={handleChange}
    />
  </label>
  <br />
  <label>
    Wiek:
    <input
      type="number"
      name="age"
      value={formData.age}
      onChange={handleChange}
    />
  </label>
  <br />
  <button type="submit">Wyślij</button>
</form>



  );
}

// Renderowanie formularza
ReactDOM.render(, document.getElementById('root'));

W tym przykładzie zarządzamy danymi wszystkich pól za pomocą jednego stanu formData, który jest obiektem. Funkcja handleChange dynamicznie aktualizuje odpowiednie pole, korzystając z nazwy pola jako klucza obiektu.

Kiedy używać formularzy w React?

Formularze są niezbędne w aplikacjach, które wymagają interakcji z użytkownikiem, np. rejestracja, logowanie, ankiety, itp. React pozwala na łatwe zarządzanie danymi w formularzach dzięki wykorzystaniu stanu i dynamicznej aktualizacji zawartości pól.

Podsumowanie

Tworzenie formularzy w React jest proste i elastyczne dzięki zarządzaniu stanem komponentu. Możemy w łatwy sposób dodać pola, zarządzać ich wartościami oraz reagować na interakcje użytkownika. Przykłady pokazują różne sposoby zarządzania formularzami w zależności od potrzeb aplikacji.
