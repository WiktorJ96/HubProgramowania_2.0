Wstawianie Wyrażeń JavaScript w JSX

JSX to rozszerzona składnia JavaScript używana w React, która umożliwia pisanie struktury komponentów w sposób przypominający HTML. Jedną z najważniejszych cech JSX jest możliwość wstawiania wyrażeń JavaScript wewnątrz klamr {}.

Proste wstawienie wyrażenia JavaScript w JSX

Aby wstawić wyrażenie JavaScript w JSX, wystarczy użyć klamr {}. Można w ten sposób wyświetlać wartości zmiennych, wywoływać funkcje czy wykonywać operacje matematyczne.


        function MyComponent() {
          const name = "Jan";
          const age = 30;
          return (
            <div>
              <p>Cześć, {name}! Masz {age} lat.</p>
            </div>
          );
        }

W tym przykładzie wyrażenia {name} oraz {age} są wstawiane do JSX, a React renderuje ich wartości w interfejsie użytkownika.

Wywoływanie funkcji w JSX

Możesz również wywoływać funkcje w JSX. Poniższy przykład pokazuje, jak wywołać funkcję zwracającą wynik w JSX.


        function greetUser(name) {
          return `Cześć, ${name}!`;
        }

        function MyComponent() {
          const user = "Anna";
          return (
            <div>
              <p>{greetUser(user)}</p>
            </div>
          );
        }

Funkcja greetUser jest wywoływana w klamrach, a jej wynik jest wyświetlany w JSX.

Operacje matematyczne i logiczne w JSX

JSX pozwala również na wykonywanie operacji matematycznych oraz logicznych. Przykład:


        function MyComponent() {
          const a = 10;
          const b = 20;
          return (
            <div>
              <p>Wynik dodawania: {a + b}</p>
              <p>Czy a jest większe od b? {a > b ? 'Tak' : 'Nie'}</p>
            </div>
          );
        }

W powyższym przykładzie JSX wykonuje operacje matematyczne oraz sprawdza warunki logiczne, a wynik jest dynamicznie wyświetlany w interfejsie użytkownika.

Wstawianie złożonych struktur danych w JSX

Możesz także wstawiać złożone struktury danych, takie jak tablice czy obiekty. Na przykład, możesz mapować elementy tablicy na komponenty JSX.


        function MyComponent() {
          const items = ['jabłko', 'banan', 'gruszka'];
          return (
            <ul>
              {items.map(item => <li key={item}>{item}</li>)}
            </ul>
          );
        }

W tym przykładzie każdy element tablicy items jest renderowany jako osobny element <li> w JSX.

Podsumowanie

JSX pozwala na wstawianie wyrażeń JavaScript w klamrach {}, co umożliwia dynamiczne renderowanie treści, wywoływanie funkcji, wykonywanie operacji logicznych i matematycznych, a także manipulację tablicami i obiektami. Dzięki temu aplikacje React mogą być bardziej interaktywne i elastyczne.
