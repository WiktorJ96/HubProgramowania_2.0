Tworzenie prostych elementów JSX w React

JSX (JavaScript XML) to specjalna składnia, która umożliwia pisanie komponentów React w stylu przypominającym HTML. JSX pozwala na tworzenie elementów interfejsu użytkownika w prosty i czytelny sposób. Każdy element JSX zostaje przekształcony przez React w wywołania funkcji JavaScript, co pozwala na dynamiczne generowanie struktury interfejsu.

Tworzenie elementu JSX

Aby stworzyć prosty element JSX, można użyć składni zbliżonej do HTML. Wystarczy zdefiniować tagi podobnie jak w HTML, ale cała struktura znajduje się w kodzie JavaScript.


        import React from 'react';

        function App() {
          return (
            <div>
              <h1>Witaj w React!</h1>
              <p>To jest prosty element JSX.</p>
            </div>
          );
        }

        export default App;

W tym przykładzie tworzymy element div, który zawiera nagłówek h1 i akapit p. Każdy element JSX jest zagnieżdżony w strukturze komponentu React.

JSX jako wyrażenia

JSX to wyrażenie, co oznacza, że można je przypisywać do zmiennych, przekazywać jako argumenty do funkcji, a nawet zwracać z funkcji. Każdy element JSX jest przekształcany do wywołania React.createElement().


        import React from 'react';

        function App() {
          const element = <h1>Witaj w React!</h1>;  {/* JSX jako wyrażenie */}

          return (
            <div>
              {element}  {/* Wstawianie elementu JSX */}
              <p>To jest prosty element JSX.</p>
            </div>
          );
        }

        export default App;

W powyższym przykładzie zmienna element przechowuje JSX, który jest następnie wstawiony do struktury HTML przy użyciu nawiasów klamrowych {}.

Zagnieżdżanie elementów JSX

JSX wspiera zagnieżdżanie elementów, co oznacza, że jeden element JSX może zawierać inne elementy wewnątrz siebie.


        import React from 'react';

        function App() {
          return (
            <div>
              <h1>Witaj!</h1>
              <div>
                <p>To jest zagnieżdżony element JSX.</p>
              </div>
            </div>
          );
        }

        export default App;

W tym przykładzie element div zawiera inny element div, który z kolei zawiera akapit p. JSX pozwala na budowanie złożonych struktur poprzez proste zagnieżdżanie elementów.

Dodawanie atrybutów do elementów JSX

JSX umożliwia dodawanie atrybutów, takich jak klasy CSS, style inline, a także niestandardowe atrybuty. W React niektóre atrybuty mają inne nazwy niż w HTML (np. class to className, a for to htmlFor).


        import React from 'react';

        function App() {
          return (
            <div className="container">
              <h1 style={ { color: 'blue' } }>Witaj w React!</h1>  {/* Styl inline */}
              <p>Ucz się, jak dodawać atrybuty do elementów JSX.</p>
            </div>
          );
        }

        export default App;

W tym przykładzie używamy atrybutu className do dodania klasy CSS oraz style do zdefiniowania stylu inline. W JSX style inline są definiowane jako obiekt JavaScript.

Warunkowe renderowanie elementów JSX

JSX pozwala na warunkowe renderowanie elementów, co oznacza, że można wyświetlać różne elementy w zależności od wartości zmiennych lub wyrażeń logicznych.


        import React from 'react';

        function App() {
          const isLoggedIn = true;

          return (
            <div>
              <h1>Witaj!</h1>
              <p>{isLoggedIn ? 'Jesteś zalogowany.' : 'Zaloguj się.'}</p>  {/* Operator warunkowy */}
            </div>
          );
        }

        export default App;

W powyższym przykładzie używamy operatora warunkowego (ternary), aby wyświetlić różne treści w zależności od wartości zmiennej isLoggedIn.

Wstawianie wyrażeń JavaScript w JSX

Do JSX można wstawiać wyrażenia JavaScript za pomocą nawiasów klamrowych {}. Można w nich używać zmiennych, funkcji, a nawet operacji matematycznych.


        import React from 'react';

        function App() {
          const name = "Anna";
          const age = 28;

          return (
            <div>
              <h1>Witaj, {name}!</h1>  {/* Wstawianie zmiennej */}
              <p>Masz {age} lat.</p>  {/* Wstawianie wyrażenia */}
              <p>Za 5 lat będziesz mieć {age + 5} lat.</p>  {/* Operacja matematyczna */}
            </div>
          );
        }

        export default App;

W tym przykładzie wstawiamy wyrażenia JavaScript do JSX, aby dynamicznie generować treść opartą na zmiennych i operacjach matematycznych.

Podsumowanie

JSX w React to potężna składnia, która pozwala na pisanie kodu przypominającego HTML, ale z pełnymi możliwościami JavaScript. Dzięki JSX można tworzyć złożone interfejsy użytkownika w prosty sposób, łącząc elementy, atrybuty i wyrażenia JavaScript. JSX jest kluczowym elementem React, który sprawia, że praca z komponentami staje się łatwa i intuicyjna.
