Podstawy Next.js

Next.js to popularny framework oparty na React, który umożliwia budowanie aplikacji z renderowaniem po stronie serwera (SSR) oraz statycznym generowaniem stron (SSG). Poniżej znajdziesz informacje o strukturze projektu, instalacji, konfiguracji oraz podstawowych komendach.

Struktura projektu

Podstawowa struktura projektu Next.js wygląda następująco:


my-next-app/
├── pages/          
├── public/         
├── styles/         
├── components/     
├── .next/          
├── package.json    
├── node_modules/

Folder pages/ zawiera pliki, które automatycznie odpowiadają różnym ścieżkom URL w aplikacji.

Instalacja

Aby zainstalować Next.js, musisz mieć zainstalowanego Node.js. Następnie wykonaj poniższe kroki:


# Tworzenie nowego projektu Next.js
npx create-next-app@latest my-next-app

# Przejście do katalogu projektu
cd my-next-app

Komenda npx create-next-app utworzy nowy projekt z domyślną konfiguracją i podstawowymi plikami.

Konfiguracja

Plik next.config.js pozwala dostosować konfigurację projektu. Przykład:


module.exports = {
  reactStrictMode: true,
  images: {
    domains: ['example.com'], // Zezwolenie na ładowanie obrazów z określonych domen
  },
};

Możesz w nim ustawiać opcje, takie jak optymalizacja obrazów, routing czy wsparcie dla innych technologii.

Podstawowe komendy

Next.js oferuje kilka podstawowych komend do zarządzania projektem:

  • npm run dev: Uruchamia projekt w trybie deweloperskim (na żywo z aktualizacjami).
  • npm run build: Tworzy zoptymalizowaną wersję produkcyjną aplikacji.
  • npm start: Uruchamia wcześniej zbudowaną wersję aplikacji w trybie produkcyjnym.

# Tryb deweloperski
npm run dev

# Budowanie aplikacji
npm run build

# Start w trybie produkcyjnym
npm start

Komenda npm run dev jest najbardziej przydatna podczas codziennej pracy, ponieważ automatycznie odświeża aplikację po zapisaniu zmian w plikach.

Podsumowanie

Next.js to potężne narzędzie do budowania nowoczesnych aplikacji webowych. Dzięki możliwościom SSR i SSG pozwala na tworzenie szybkich i zoptymalizowanych stron. Pamiętaj, aby zapoznać się z dokumentacją na oficjalnej stronie Next.js, aby dowiedzieć się więcej!
