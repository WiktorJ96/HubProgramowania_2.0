Deploy Next.js na Vercel i Netlify

Next.js można łatwo wdrożyć na platformach takich jak Vercel i Netlify. Obie te usługi oferują darmowy hosting, automatyczne wdrażanie i integrację z GitHub.

1. Deploy na Vercel

Vercel to platforma stworzona przez twórców Next.js, oferująca natywną obsługę tego frameworka.

Kroki wdrożenia:

  • Utwórz konto na Vercel.
  • Połącz swoje repozytorium GitHub/GitLab/Bitbucket.
  • Wybierz repozytorium z aplikacją Next.js.
  • Vercel automatycznie wykryje Next.js i skonfiguruje projekt.
  • Po wdrożeniu otrzymasz adres URL aplikacji.

Deploy z terminala:

Można również wdrożyć aplikację ręcznie za pomocą CLI:


npm install -g vercel
vercel

Po uruchomieniu vercel aplikacja zostanie wdrożona, a Vercel wygeneruje URL.

2. Deploy na Netlify

Netlify to popularna platforma do hostowania aplikacji statycznych i dynamicznych, obsługująca Next.js.

Kroki wdrożenia:

  • Załóż konto na Netlify.
  • Połącz repozytorium z GitHub.
  • Podczas konfiguracji wybierz framework Next.js.
  • W polu Build command wpisz: next build
  • W polu Publish directory wpisz: .next
  • Rozpocznij wdrażanie.

Deploy z terminala:


npm install -g netlify-cli
netlify deploy

3. CI/CD z GitHub Actions

Można skonfigurować automatyczne wdrażanie na każdą zmianę w repozytorium za pomocą GitHub Actions:


name: Deploy to Vercel
on:
  push:
    branches:
      - main
jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - name: Install dependencies
        run: npm install
      - name: Build project
        run: npm run build
      - name: Deploy to Vercel
        run: vercel --token ${{ secrets.VERCEL_TOKEN }}

Plik ten umieszczamy w katalogu .github/workflows/deploy.yml, co pozwala na automatyczny deploy po każdej zmianie w branchu main.

4. Konfiguracja domen

Na Vercel i Netlify można podłączyć własną domenę:

  • Wejdź w ustawienia projektu.
  • Dodaj domenę w sekcji "Custom Domains".
  • Skonfiguruj rekordy DNS w panelu domeny.

Podsumowanie

Next.js można łatwo wdrożyć na Vercel i Netlify. Można także dodać CI/CD do automatycznego wdrażania aplikacji oraz podłączyć własną domenę. Więcej informacji znajdziesz w dokumentacji Next.js.
