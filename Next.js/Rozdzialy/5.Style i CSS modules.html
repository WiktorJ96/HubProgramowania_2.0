Style i CSS Modules w Next.js

Next.js oferuje różne sposoby stylizowania aplikacji, w tym style globalne, CSS Modules, styled-jsx oraz integrację z Tailwind CSS. Dzięki temu możesz wybrać najlepsze rozwiązanie dla swojego projektu.

Konfiguracja stylów globalnych

Style globalne w Next.js definiuje się w plikach CSS, które importuje się w pliku _app.js (lub _app.tsx w TypeScript). Przykład:


// pages/_app.js
import '../styles/globals.css';

export default function MyApp({ Component, pageProps }) {
  return ;
}

Plik globals.css znajduje się w katalogu styles i zawiera stylizację globalną aplikacji.


/* styles/globals.css */
body {
  margin: 0;
  font-family: Arial, sans-serif;
  background-color: #f5f5f5;
}

CSS Modules

CSS Modules to sposób stylizowania, w którym klasy są izolowane do konkretnego komponentu. Aby użyć CSS Modules, utwórz plik z rozszerzeniem .module.css i zaimportuj go do komponentu.


// components/Button.js
import styles from './Button.module.css';

const Button = () => {
  return ;
};

export default Button;
        

/* components/Button.module.css */
.button {
  background-color: #0070f3;
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.button:hover {
  background-color: #005bb5;
}

Integracja z Tailwind CSS

Next.js łatwo integruje się z Tailwind CSS, co umożliwia korzystanie z klasycznych utility-first stylów. Aby zintegrować Tailwind CSS, wykonaj poniższe kroki:


# Instalacja Tailwind CSS
npm install tailwindcss postcss autoprefixer
npx tailwindcss init

W pliku tailwind.config.js skonfiguruj ścieżki do swoich plików:


module.exports = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {},
  },
  plugins: [],
};

Następnie zaimportuj Tailwind CSS w pliku globals.css:


/* styles/globals.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

Po konfiguracji możesz używać klas Tailwind w swoich komponentach:


const TailwindComponent = () => {
  return ;
};

export default TailwindComponent;

Podsumowanie

Next.js oferuje różne możliwości stylizowania aplikacji: od tradycyjnych stylów globalnych, przez CSS Modules, aż po nowoczesne podejścia jak styled-jsx i Tailwind CSS. Wybór metody zależy od potrzeb projektu i preferencji zespołu.

Więcej informacji znajdziesz w dokumentacji Next.js.
