Routing w Next.js

Next.js oferuje system routingu oparty na strukturze plików w katalogu pages. Pliki w tym katalogu automatycznie stają się stronami dostępnymi pod odpowiadającymi im ścieżkami URL.

Tworzenie stron

Aby utworzyć nową stronę, wystarczy dodać nowy plik do katalogu pages. Przykład:


pages/
├── index.js        
├── about.js

Plik about.js będzie odpowiadał ścieżce /about w aplikacji.

Dynamiczne ścieżki

Next.js obsługuje dynamiczne ścieżki za pomocą nazw plików z nawiasami kwadratowymi. Przykład:


pages/
├── product/
│   ├── [id].js

Plik [id].js odpowiada dynamicznym ścieżkom, np. /product/1 lub /product/abc. Wartość id można pobrać za pomocą getStaticProps lub useRouter.


// Przykład dynamicznej strony w [id].js
import { useRouter } from 'next/router';

const ProductPage = () => {
  const router = useRouter();
  const { id } = router.query; // Pobranie dynamicznego id

  return 
Produkt o ID: {id}
;
};

export default ProductPage;

Link

Komponent Link pozwala na nawigację między stronami bez przeładowania strony. Przykład:


import Link from 'next/link';

const HomePage = () => {
  return (
    

      
Strona główna

      
        Przejdź do strony About
      
    

  );
};

export default HomePage;

Użycie Link zapewnia lepszą wydajność dzięki wbudowanemu systemowi optymalizacji Next.js.

useRouter

Hook useRouter daje dostęp do informacji o bieżącej trasie oraz pozwala na programową nawigację. Przykład:


import { useRouter } from 'next/router';

const NavigationExample = () => {
  const router = useRouter();

  const goToAbout = () => {
    router.push('/about'); // Przejście do strony /about
  };

  return (
    

      
    

  );
};

export default NavigationExample;

Metoda router.push() pozwala na przejście do innej strony z poziomu kodu JavaScript.

Nawigacja między podstronami

Next.js umożliwia łatwe przechodzenie między stronami za pomocą komponentu Link oraz programowo za pomocą useRouter. Przykładowa nawigacja:

Podsumowanie

Routing w Next.js jest prosty i intuicyjny dzięki strukturze plików, dynamicznym ścieżkom, komponentowi Link oraz hookowi useRouter. Dzięki temu budowanie aplikacji wielostronicowych jest szybkie i wydajne. Więcej informacji znajdziesz w dokumentacji Next.js.
