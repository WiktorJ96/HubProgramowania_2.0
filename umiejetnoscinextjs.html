Podstawy Next.js

Struktura projektu, instalacja, konfiguracja, podstawowe komendy (dev, build, start).

Routing

Strony (pages), dynamiczne ścieżki (np. [id].js), Link, useRouter, nawigacja między podstronami.

Rendering (SSR/SSG/ISR)

getStaticProps, getServerSideProps, Incremental Static Regeneration, optymalizacja ładowania stron.

API Routes

Tworzenie własnych endpointów (api/), obsługa żądań (GET, POST), autoryzacja.

Style i CSS Modules

Konfiguracja stylów globalnych, pliki .module.css, styled-jsx, integracja z Tailwind CSS.

Optymalizacja Obrazów

Komponent Image, ładowanie obrazów z zewnętrznych źródeł, responsywne rozmiary.

Head i SEO

Komponent Head, meta tagi, tytuł strony, podstawy SEO, Open Graph.

Middleware (pośrednik)

Tworzenie middleware (pośrednika) do kontroli ruchu, autoryzacji, przekierowań w Next.js.

Konfiguracja Next.js

next.config.js, optymalizacja buildu, customowy serwer, zmienne środowiskowe.

Autoryzacja i Sesje

Integracja z next-auth, przechowywanie sesji, ochrona stron (middleware).

Baza Danych i ORM

Podłączenie do bazy (MongoDB, PostgreSQL itp.), użycie ORM (np. Prisma) w projektach Next.js.

Deploy (Vercel, Netlify)

Wdrożenie aplikacji Next.js na Vercel/Netlify, podstawy CI/CD, konfiguracja domen.

Optymalizacja i Wydajność

Analiza rozmiaru bundla, lazy loading (opóźnione ładowanie), code splitting, Lighthouse.

Testowanie w Next.js

Integracja z Jest, Cypress, React Testing Library, testy e2e.
