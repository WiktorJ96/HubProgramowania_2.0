Parsowanie i Serializacja JSON

Konwersja między obiektami JavaScript a JSON

W JavaScript JSON jest używany do przesyłania danych między serwerem a klientem, co wymaga konwersji między obiektami JavaScript a JSON.

Dwie podstawowe metody JavaScript do pracy z JSON to:

  • JSON.parse() - służy do konwersji ciągu znaków JSON na obiekt JavaScript.
  • JSON.stringify() - służy do konwersji obiektu JavaScript na ciąg znaków JSON.

Metoda JSON.parse()

Metoda JSON.parse() konwertuje ciąg JSON na obiekt JavaScript, co jest przydatne, gdy dane są otrzymywane jako tekst JSON i muszą być przetworzone w aplikacji.

Składnia: JSON.parse(text)

Przykład:


const jsonString = '{"name": "Alice", "age": 25}';
const jsObject = JSON.parse(jsonString);
console.log(jsObject.name); // Wyjście: "Alice"

W tym przykładzie, JSON.parse() przekształca tekst JSON w obiekt JavaScript, który można używać w kodzie.

Metoda JSON.stringify()

Metoda JSON.stringify() konwertuje obiekt JavaScript na ciąg JSON, co jest przydatne, gdy chcemy przesłać dane do serwera w formacie JSON.

Składnia: JSON.stringify(value)

Przykład:


const jsObject = { name: "Alice", age: 25 };
const jsonString = JSON.stringify(jsObject);
console.log(jsonString); // Wyjście: '{"name":"Alice","age":25}'

W tym przykładzie, JSON.stringify() konwertuje obiekt JavaScript na tekst JSON, który można wysłać do serwera.

Przykłady Zastosowań

  • Parsowanie danych: Przetwarzanie danych pobranych z API, które zwracają wyniki w formacie JSON.
  • Serializacja danych: Przekształcanie obiektów JavaScript na JSON w celu ich zapisania, np. w plikach lub bazach danych.
  • Przesyłanie danych: Wysyłanie danych z aplikacji klienta na serwer w formacie JSON.
