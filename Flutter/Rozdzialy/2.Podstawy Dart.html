Podstawy Dart

Dart to język programowania opracowany przez Google, który jest używany głównie do tworzenia aplikacji w frameworku Flutter. Posiada silne typowanie, wsparcie dla programowania obiektowego oraz asynchroniczności.

Składnia języka

Podstawowe elementy składni Dart obejmują deklaracje zmiennych, funkcje oraz struktury sterujące.


void main() {
  print('Witaj w Dart!');
  int liczba = 10;
  double przecinek = 3.14;
  String tekst = 'Przykładowy string';
  bool prawda = true;
  print('$liczba, $przecinek, $tekst, $prawda');
}

Funkcje

Funkcje w Dart mogą mieć opcjonalne parametry oraz wartości domyślne.


int suma(int a, int b) {
  return a + b;
}

void main() {
  print(suma(5, 7));
}

Klasy i obiekty

Dart wspiera programowanie obiektowe z użyciem klas i obiektów.


class Osoba {
  String imie;
  int wiek;

  Osoba(this.imie, this.wiek);

  void przedstawSie() {
    print('Cześć, jestem $imie i mam $wiek lat.');
  }
}

void main() {
  var osoba = Osoba('Jan', 25);
  osoba.przedstawSie();
}

Typy danych

Dart posiada różne typy danych, w tym liczby, napisy, listy i mapy.


void main() {
  List liczby = [1, 2, 3, 4, 5];
  Map oceny = {'Jan': 5, 'Anna': 4};
  print(liczby);
  print(oceny);
}

Obsługa asynchroniczności (async/await)

Dart obsługuje operacje asynchroniczne za pomocą Future i async/await.


Future pobierzDane() async {
  await Future.delayed(Duration(seconds: 2));
  return 'Dane zostały pobrane';
}

void main() async {
  print('Rozpoczynam pobieranie...');
  String dane = await pobierzDane();
  print(dane);
}
