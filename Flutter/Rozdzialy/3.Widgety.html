Widżety w Flutterze

Flutter to framework umożliwiający tworzenie aplikacji na wiele platform przy użyciu jednego kodu źródłowego. Kluczowym elementem Fluttera są widżety, które budują interfejs użytkownika.

StatelessWidget vs StatefulWidget

Widżety w Flutterze dzielą się na dwa podstawowe typy:

  • StatelessWidget – widżety bezstanowe, które nie zmieniają swojego stanu po utworzeniu.
  • StatefulWidget – widżety stanowe, których stan może się dynamicznie zmieniać.

Przykład StatelessWidget

Widżet StatelessWidget nadaje się do statycznych komponentów, które nie wymagają zmian w trakcie działania aplikacji.


import 'package:flutter/material.dart';

class MyStatelessWidget extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Center(
      child: Text('To jest StatelessWidget'),
    );
  }
}

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text('StatelessWidget')),
      body: MyStatelessWidget(),
    ),
  ));
}

Przykład StatefulWidget

Widżet StatefulWidget może zmieniać swój stan za pomocą metody setState. Przykładem jest licznik, który aktualizuje wartość po naciśnięciu przycisku.


import 'package:flutter/material.dart';

class MyStatefulWidget extends StatefulWidget {
  @override
  _MyStatefulWidgetState createState() => _MyStatefulWidgetState();
}

class _MyStatefulWidgetState extends State {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Text('Licznik: $_counter', style: TextStyle(fontSize: 24)),
        SizedBox(height: 10),
        ElevatedButton(
          onPressed: _incrementCounter,
          child: Text('Zwiększ licznik'),
        ),
      ],
    );
  }
}

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text('StatefulWidget')),
      body: Center(child: MyStatefulWidget()),
    ),
  ));
}

Budowanie interfejsu użytkownika

Flutter korzysta z hierarchii widżetów do budowania interfejsu. Można je komponować i układać, tworząc zaawansowane interfejsy.


import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Budowanie UI w Flutterze')),
        body: Column(
          children: [
            Text('Witaj w Flutter!', style: TextStyle(fontSize: 24)),
            ElevatedButton(
              onPressed: () {},
              child: Text('Kliknij mnie'),
            ),
          ],
        ),
      ),
    );
  }
}

Modyfikacja stanu

Widżety stanowe pozwalają na dynamiczną aktualizację UI. Przykładem jest zmiana tekstu po naciśnięciu przycisku.


import 'package:flutter/material.dart';

class TextChanger extends StatefulWidget {
  @override
  _TextChangerState createState() => _TextChangerState();
}

class _TextChangerState extends State {
  String _text = 'Pierwotny tekst';

  void _changeText() {
    setState(() {
      _text = 'Nowy tekst po naciśnięciu przycisku';
    });
  }

  @override
  Widget build(BuildContext context) {
    return Column(
      mainAxisAlignment: MainAxisAlignment.center,
      children: [
        Text(_text, style: TextStyle(fontSize: 20)),
        SizedBox(height: 10),
        ElevatedButton(
          onPressed: _changeText,
          child: Text('Zmień tekst'),
        ),
      ],
    );
  }
}

void main() {
  runApp(MaterialApp(
    home: Scaffold(
      appBar: AppBar(title: Text('Modyfikacja stanu')),
      body: Center(child: TextChanger()),
    ),
  ));
}
