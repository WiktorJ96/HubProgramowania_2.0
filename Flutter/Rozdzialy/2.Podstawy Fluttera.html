Podstawy Fluttera

Flutter to framework open-source stworzony przez Google do tworzenia wieloplatformowych aplikacji mobilnych, webowych i desktopowych. Umożliwia szybki rozwój aplikacji dzięki wykorzystaniu języka Dart oraz hot-reload.

Instalacja Fluttera

Aby zacząć pracę z Flutterem, należy najpierw zainstalować środowisko. Oficjalne instrukcje znajdują się na stronie flutter.dev. Najlepiej pobrać stabilną wersję.


# Sprawdzenie wersji Fluttera po instalacji
flutter --version

Po instalacji, komenda flutter --version pozwoli sprawdzić, czy Flutter został poprawnie zainstalowany.

Podstawowe komendy

Flutter oferuje kilka podstawowych komend do zarządzania projektami i testowania aplikacji. Oto niektóre z nich:

  • flutter create [nazwa_projektu]: Tworzy nowy projekt Fluttera.
  • flutter run: Uruchamia aplikację na emulatorze lub podłączonym urządzeniu.
  • flutter pub get: Pobiera zależności projektu.

# Tworzenie nowego projektu Fluttera
flutter create my_app

# Uruchomienie aplikacji
cd my_app
flutter run

# Pobranie zależności
flutter pub get

Hot Reload i Hot Restart

Flutter oferuje funkcję hot reload, która pozwala na szybkie podglądanie zmian w kodzie bez konieczności ponownego uruchamiania aplikacji.


# Uruchomienie aplikacji z obsługą hot reload
flutter run

Podczas działania aplikacji można modyfikować kod i zapisać plik – zmiany zostaną natychmiast odzwierciedlone w interfejsie użytkownika.

Struktura projektu Fluttera

Typowy projekt Fluttera składa się z kilku kluczowych folderów:

  • lib/: Główny katalog kodu źródłowego aplikacji.
  • pubspec.yaml: Plik konfiguracyjny zarządzający zależnościami.
  • android/ i ios/: Foldery zawierające natywny kod dla Androida i iOS.

Przykładowa aplikacja Flutter

Poniżej znajduje się przykład prostego kodu Fluttera, który tworzy aplikację z przyciskiem zwiększającym licznik.


import 'package:flutter/material.dart';

void main() {
  runApp(MyApp());
}

class MyApp extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: Text('Flutter Demo')),
        body: CounterWidget(),
      ),
    );
  }
}

class CounterWidget extends StatefulWidget {
  @override
  _CounterWidgetState createState() => _CounterWidgetState();
}

class _CounterWidgetState extends State {
  int _counter = 0;

  void _incrementCounter() {
    setState(() {
      _counter++;
    });
  }

  @override
  Widget build(BuildContext context) {
    return Center(
      child: Column(
        mainAxisAlignment: MainAxisAlignment.center,
        children: [
          Text('Licznik:', style: TextStyle(fontSize: 24)),
          Text('$_counter', style: TextStyle(fontSize: 32)),
          SizedBox(height: 20),
          ElevatedButton(
            onPressed: _incrementCounter,
            child: Text('Zwiększ licznik'),
          ),
        ],
      ),
    );
  }
}

Powyższy kod tworzy aplikację, w której użytkownik może nacisnąć przycisk, aby zwiększyć licznik.
