Tworzenie nowych elementów DOM w JavaScript

W JavaScript możemy dynamicznie tworzyć i dodawać nowe elementy do DOM. Poniżej znajdziesz różne metody tworzenia i manipulacji elementami DOM w czasie rzeczywistym.

Tworzenie nowego elementu za pomocą document.createElement()

Metoda createElement() pozwala na tworzenie nowego elementu DOM na podstawie nazwy tagu, np. <div> lub <p>.


// Tworzenie nowego elementu 

const newDiv = document.createElement('div');

// Ustawienie treści tekstowej w nowym elemencie
newDiv.textContent = 'To jest nowy element DIV!';

// Dodanie klasy do nowego elementu
newDiv.classList.add('new-div');

// Dodanie elementu do dokumentu
document.body.appendChild(newDiv);

W powyższym przykładzie tworzymy nowy element <div>, przypisujemy do niego treść tekstową i klasę, a następnie dodajemy go do <body> dokumentu.

Ustawianie atrybutów elementu

Za pomocą metody setAttribute() możemy ustawiać atrybuty HTML dla nowo utworzonego elementu.


// Tworzenie nowego linku
const newLink = document.createElement('a');

// Ustawienie atrybutów href i target
newLink.setAttribute('href', 'https://www.example.com');
newLink.setAttribute('target', '_blank');

// Ustawienie tekstu linku
newLink.textContent = 'Kliknij tutaj!';

// Dodanie linku do 
document.body.appendChild(newLink);

W tym przykładzie tworzymy nowy element <a> (link), ustawiamy jego atrybuty href i target, a następnie dodajemy go do dokumentu.

Usuwanie elementów z DOM

Możemy usunąć elementy z DOM za pomocą metody remove():


// Pobieranie elementu do usunięcia
const elementToRemove = document.querySelector('.new-div');

// Usunięcie elementu
elementToRemove.remove();

W tym przykładzie usuwamy element <div> o klasie new-div.

Podsumowanie

  • document.createElement() - Służy do tworzenia nowych elementów DOM.
  • setAttribute() - Umożliwia ustawianie atrybutów HTML dla elementu.
  • appendChild() - Dodaje nowo utworzony element do drzewa DOM.
  • remove() - Usuwa element z drzewa DOM.
