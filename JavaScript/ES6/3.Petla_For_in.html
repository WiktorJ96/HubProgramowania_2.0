Pętla for...in w JavaScript

Pętla for...in służy do iteracji po właściwościach obiektu.

Składnia:

for (let key in object) { // kod do wykonania }

Cechy pętli for...in:

  • Iteruje po wszystkich wyliczalnych właściwościach obiektu.
  • Obejmuje właściwości z łańcucha prototypów.
  • Kolejność iteracji nie jest gwarantowana.
  • Nie zalecana do iteracji po tablicach.

Zastosowania:

  • Przeglądanie właściwości obiektów.
  • Debugowanie - wyświetlanie wszystkich właściwości obiektu.
  • Kopiowanie lub klonowanie obiektów.

Uwagi:

  • Używaj Object.hasOwnProperty() do sprawdzania, czy właściwość należy bezpośrednio do obiektu.
  • Dla tablic, lepiej używać standardowej pętli for lub metod tablicowych.
