Definiowanie i Wywoływanie Funkcji

Funkcje w JavaScript są podstawowym sposobem na organizację i ponowne używanie kodu. Funkcje mogą przyjmować parametry, zwracać wartości oraz być deklarowane na różne sposoby. Poniżej znajdziesz omówienie różnych metod definiowania i wywoływania funkcji.

Deklaracja funkcji

Najprostszym sposobem zdefiniowania funkcji jest użycie deklaracji function. Deklaracja funkcji składa się z nazwy funkcji, listy parametrów (w nawiasach) oraz bloku kodu (w klamrach), który zostanie wykonany po wywołaniu funkcji.


function greet(name) {
    console.log("Cześć, " + name + "!");
}

greet("Ala"); // Wywołanie funkcji, wynik: "Cześć, Ala!"

W powyższym przykładzie funkcja greet przyjmuje jeden parametr name i wyświetla wiadomość powitalną w konsoli.

Wyrażenia funkcyjne

Funkcję można również przypisać do zmiennej, co nazywamy wyrażeniem funkcyjnym. W takim przypadku funkcja jest anonimowa (bez nazwy).


const sayHello = function(name) {
    console.log("Cześć, " + name + "!");
};

sayHello("Jan"); // Wywołanie funkcji, wynik: "Cześć, Jan!"

Funkcje zadeklarowane w ten sposób można wywoływać w podobny sposób jak funkcje nazwane, przypisując je do zmiennej.

Funkcje strzałkowe

Funkcje strzałkowe (ang. arrow functions) to krótsza składnia definiowania funkcji. Zamiast słowa kluczowego function używa się symbolu strzałki (=>).


const add = (a, b) => {
    return a + b;
};

console.log(add(5, 3)); // Wynik: 8

Jeśli funkcja zawiera tylko jedną instrukcję, można pominąć klamry oraz słowo return:


const multiply = (a, b) => a * b;

console.log(multiply(4, 2)); // Wynik: 8

Zwracanie wartości

Funkcja może zwracać wynik swojej pracy przy pomocy instrukcji return. Po napotkaniu return, funkcja kończy swoje działanie i zwraca wartość.


function sum(a, b) {
    return a + b; // Funkcja zwraca sumę dwóch argumentów
}

let result = sum(10, 5);
console.log(result); // Wynik: 15

W powyższym przykładzie funkcja sum zwraca sumę dwóch liczb, a wynik jest przechowywany w zmiennej result.

Wartości domyślne parametrów

Można przypisać wartości domyślne do parametrów funkcji. Jeśli podczas wywołania funkcji nie podamy wartości dla danego parametru, zostanie użyta wartość domyślna.


function greet(name = "Gość") {
    console.log("Cześć, " + name + "!");
}

greet();          // Wynik: "Cześć, Gość!"
greet("Kasia");   // Wynik: "Cześć, Kasia!"

W tym przypadku, jeśli funkcja greet zostanie wywołana bez argumentu, wartość domyślna "Gość" zostanie użyta.
