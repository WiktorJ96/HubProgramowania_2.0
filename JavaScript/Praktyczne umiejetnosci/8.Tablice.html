Tablice

Tablice to struktury danych, które pozwalają przechowywać wiele wartości w jednej zmiennej. Są one szczególnie przydatne, gdy chcemy przechowywać listy elementów. Tablice w JavaScript są dynamiczne i mogą zawierać elementy różnych typów danych.

Tworzenie tablic

Tablicę można stworzyć za pomocą nawiasów kwadratowych []. Elementy tablicy są oddzielane przecinkami.


const fruits = ["jabłko", "banan", "gruszka"];
console.log(fruits); // ["jabłko", "banan", "gruszka"]

Tablicę można porównać do półki, na której przechowujemy różne owoce. Każdy owoc to element tablicy, a półka (tablica) może zawierać ich dowolną liczbę.

Metody tablicowe

JavaScript udostępnia wiele wbudowanych metod do pracy z tablicami. Oto najczęściej używane z nich:

push() i pop()

Metoda push() dodaje element na końcu tablicy, a pop() usuwa ostatni element.


const numbers = [1, 2, 3];
numbers.push(4); // Dodaje 4 na końcu tablicy
console.log(numbers); // [1, 2, 3, 4]

numbers.pop(); // Usuwa ostatni element (4)
console.log(numbers); // [1, 2, 3]

Wyobraź sobie tablicę jako pudełko, do którego można dokładać kolejne elementy (push()) lub wyciągać ostatni dodany element (pop()).

map()

Metoda map() tworzy nową tablicę, stosując funkcję do każdego elementu tablicy.


const numbers = [1, 2, 3];
const doubled = numbers.map(number => number * 2); // Funkcja przyjmuje element tablicy (number) i mnoży go przez 2
console.log(doubled); // [2, 4, 6]

// number - reprezentuje każdy element tablicy 'numbers'
// number * 2 - działanie, które ma być zastosowane na każdym elemencie (w tym przypadku mnożenie przez 2)

Metodę map() można porównać do malowania każdego elementu w tablicy na inny kolor — każdy element zostaje przekształcony według podanej funkcji.

filter()

Metoda filter() tworzy nową tablicę z elementami, które spełniają określony warunek.


const numbers = [1, 2, 3, 4, 5];
const even = numbers.filter(number => number % 2 === 0); 
// number % 2 === 0 - warunek, który sprawdza, czy liczba jest parzysta (tylko te elementy zostaną zachowane w nowej tablicy)
console.log(even); // [2, 4]

// number - reprezentuje każdy element tablicy 'numbers'
// number % 2 === 0 - warunek filtrowania, który przepuszcza tylko parzyste liczby

Metoda filter() działa jak sitko, które przepuszcza tylko te elementy, które spełniają określone kryteria.

reduce()

Metoda reduce() przekształca tablicę do jednej wartości, stosując funkcję do każdego elementu i "zbierając" wynik.


const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((total, number) => total + number, 0); 
// total - to suma zgromadzonych wartości podczas iteracji po tablicy
// number - to aktualny element tablicy w danej iteracji
// 0 - to wartość początkowa, od której zaczyna się sumowanie
console.log(sum); // 10

// total - wartość, która przechowuje sumę (na początku jest 0)
// number - każdy kolejny element tablicy, który jest dodawany do total
// total + number - operacja, która sumuje wszystkie elementy tablicy
// 0 - początkowa wartość total, od której zaczyna się liczenie

Metodę reduce() można porównać do skarbonki, do której wrzucamy każdy element tablicy, sumując ich wartości.

Iteracja po tablicach

Istnieje wiele sposobów na iterowanie po tablicach. Oto kilka z nich:

for

Pętla for pozwala na przechodzenie przez każdy element tablicy.


const fruits = ["jabłko", "banan", "gruszka"];

for (let i = 0; i < fruits.length; i++) { 
    // i - zmienna indeksu, zaczyna się od 0, oznacza pozycję w tablicy
    // i < fruits.length - warunek, który sprawdza, czy indeks jest mniejszy niż długość tablicy
    // fruits[i] - dostęp do elementu tablicy na danym indeksie
    console.log(fruits[i]); // Wyświetla każdy owoc z tablicy
}

for...of

Pętla for...of to wygodny sposób na iterowanie po tablicach, bez potrzeby dostępu do indeksów.


const fruits = ["jabłko", "banan", "gruszka"];

for (const fruit of fruits) { 
    // fruit - reprezentuje każdy kolejny element tablicy 'fruits' podczas iteracji
    console.log(fruit); // Wyświetla każdy owoc z tablicy
}

forEach()

Metoda forEach() wykonuje podaną funkcję dla każdego elementu tablicy.


const fruits = ["jabłko", "banan", "gruszka"];

fruits.forEach(fruit => { 
    // fruit - reprezentuje każdy element tablicy 'fruits'
    // Funkcja strzałkowa (arrow function) wykonuje operację (console.log) na każdym elemencie tablicy
    console.log(fruit); // Wyświetla każdy owoc z tablicy
});

Podsumowanie

  • Tablice przechowują wiele wartości w jednej zmiennej i mogą zawierać elementy różnych typów danych.
  • Metody takie jak push(), pop(), map(), filter() i reduce() pozwalają na łatwą manipulację tablicami.
  • Pętle for, for...of oraz metoda forEach() pozwalają na iterowanie po tablicach.
