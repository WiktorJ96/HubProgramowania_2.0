Operatory i Wyrażenia

Operatory w JavaScript to symbole, które wykonują operacje na wartościach. Wyrażenia są to kombinacje wartości i operatorów, które JavaScript potrafi ocenić (obliczyć).

Operatory arytmetyczne

Operatory arytmetyczne wykonują podstawowe operacje matematyczne, takie jak dodawanie, odejmowanie, mnożenie i dzielenie.

  • Dodawanie: +
  • Odejmowanie: -
  • Mnożenie: *
  • Dzielenie: /
  • Modulo: % (reszta z dzielenia)
  • Inkrementacja: ++ (zwiększa wartość o 1)
  • Dekrementacja: -- (zmniejsza wartość o 1)

let a = 10;
let b = 3;
console.log(a + b); // Dodawanie: 13
console.log(a - b); // Odejmowanie: 7
console.log(a * b); // Mnożenie: 30
console.log(a / b); // Dzielenie: 3.33
console.log(a % b); // Modulo: 1

Operatory porównania

Operatory porównania sprawdzają relację pomiędzy wartościami i zwracają wartość logiczną (true lub false).

  • Równość: ==
  • Ścisła równość: === (sprawdza zarówno wartość, jak i typ danych)
  • Różność: !=
  • Ścisła różność: !==
  • Większe niż: >
  • Mniejsze niż: <
  • Większe lub równe: >=
  • Mniejsze lub równe: <=

let x = 5;
let y = 10;
console.log(x == y);  // false, bo 5 nie równa się 10
console.log(x === 5); // true, bo wartość i typ są takie same
console.log(x !== y); // true, bo 5 nie jest równe 10
console.log(x < y);   // true, bo 5 jest mniejsze niż 10

Operatory przypisania

Operatory przypisania służą do przypisywania wartości do zmiennych.

  • Przypisanie: =
  • Dodaj i przypisz: +=
  • Odejmij i przypisz: -=
  • Mnożenie i przypisz: *=
  • Dzielenie i przypisz: /=
  • Modulo i przypisz: %=

let num = 10;
num += 5;  // num = num + 5; wartość num to teraz 15
num -= 2;  // num = num - 2; wartość num to teraz 13
num *= 3;  // num = num * 3; wartość num to teraz 39

Operatory logiczne

Operatory logiczne służą do łączenia warunków i zwracają wartość logiczną (true lub false).

  • AND (i): &&
  • OR (lub): ||
  • NOT (negacja): !

let isTrue = true;
let isFalse = false;
console.log(isTrue && isFalse);  // false, bo jeden z warunków jest false
console.log(isTrue || isFalse);  // true, bo jeden z warunków jest true
console.log(!isTrue);            // false, negacja true to false

Wyrażenia warunkowe

Wyrażenia warunkowe pozwalają na wykonywanie różnych działań w zależności od spełnienia określonych warunków.

Instrukcja if


let age = 20;

if (age >= 18) {
    console.log("Jesteś pełnoletni");
} else {
    console.log("Nie jesteś pełnoletni");
}

W powyższym przykładzie sprawdzamy, czy wartość age jest większa lub równa 18, a jeśli tak, wyświetlamy komunikat o pełnoletności. Jeśli warunek nie jest spełniony, wykonywany jest blok else.

Operator warunkowy (?:)

Jest to skrócona wersja instrukcji if, znana jako operator trójargumentowy.


let canDrive = (age >= 18) ? "Możesz prowadzić" : "Nie możesz prowadzić";
console.log(canDrive); // Wyświetla "Możesz prowadzić", jeśli wiek to 18 lub więcej
