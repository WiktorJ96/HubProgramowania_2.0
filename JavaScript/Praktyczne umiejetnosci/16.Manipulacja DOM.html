Manipulacja DOM

DOM (Document Object Model) to struktura dokumentu HTML reprezentowana w formie drzewa, gdzie każdy element HTML jest węzłem. Manipulacja DOM polega na modyfikowaniu struktury, zawartości i atrybutów elementów za pomocą JavaScript.

Pobieranie elementów

Aby manipulować elementami DOM, musimy najpierw je pobrać. Możemy to zrobić na kilka sposobów:

getElementById()

Metoda getElementById() zwraca element na podstawie jego unikalnego identyfikatora (ID).


const header = document.getElementById('main-header');
console.log(header);

querySelector()

Metoda querySelector() zwraca pierwszy element, który pasuje do podanego selektora CSS.


const firstParagraph = document.querySelector('p');
console.log(firstParagraph);

querySelectorAll() z indeksem

Metoda querySelectorAll() zwraca wszystkie elementy, które pasują do podanego selektora CSS, jako listę NodeList. Możemy uzyskać dostęp do konkretnego elementu na podstawie jego indeksu w tej liście.


// Pobranie drugiego elementu 
 (indeks 1, bo indeksowanie zaczyna się od zera)
const secondParagraph = document.querySelectorAll('p')[1];
console.log(secondParagraph); // Wyświetli drugi element 


// Zmieniamy tekst w drugim elemencie 

secondParagraph.textContent = "To jest drugi akapit";

W przykładzie używamy querySelectorAll(), aby pobrać listę wszystkich elementów p, a następnie bezpośrednio odwołujemy się do drugiego elementu poprzez indeks, zmieniając jego treść.

Wyobraź sobie pobieranie elementów DOM jako przeszukiwanie drzewa. getElementById() to jak szukanie konkretnej gałęzi po jej etykiecie, a querySelector() to szukanie pierwszej gałęzi, która spełnia dany warunek (np. wszystkie gałęzie oznaczone jako 'p').

Modyfikacja treści i atrybutów

Po pobraniu elementu możemy zmieniać jego zawartość, style i atrybuty.

Modyfikacja tekstu

Możemy zmienić tekst wewnątrz elementu, używając właściwości textContent lub innerHTML.


// Zmiana tekstu nagłówka
const header = document.getElementById('main-header');
header.textContent = 'Nowy nagłówek';

Modyfikacja atrybutów

Możemy zmienić atrybuty elementu, takie jak src, href, czy alt, za pomocą metody setAttribute() lub bezpośredniej modyfikacji właściwości.


// Zmiana atrybutu obrazu
const image = document.querySelector('img');
image.setAttribute('src', 'nowy-obrazek.jpg');

// Lub bezpośrednio
image.src = 'nowy-obrazek.jpg';

Modyfikacja stylów

Style elementu można zmienić, używając właściwości style.


const header = document.getElementById('main-header');
header.style.color = 'blue';
header.style.fontSize = '24px';

Zmiana treści i atrybutów jest jak modyfikowanie opisu w katalogu – zmieniamy to, co użytkownik widzi, lub jak dany element zachowuje się na stronie.

Tworzenie i usuwanie elementów

Możemy dynamicznie tworzyć nowe elementy, dodawać je do strony oraz usuwać istniejące elementy.

Tworzenie nowych elementów

Aby utworzyć nowy element, używamy metody document.createElement(), a następnie dodajemy go do DOM za pomocą appendChild().


// Tworzenie nowego akapitu
const newParagraph = document.createElement('p');
newParagraph.textContent = 'To jest nowy akapit';

// Dodanie akapitu do elementu body
document.body.appendChild(newParagraph);

Usuwanie elementów

Aby usunąć element, używamy metody remove() lub removeChild().


// Usunięcie elementu
const paragraphToRemove = document.querySelector('p');
paragraphToRemove.remove();

Tworzenie nowych elementów to jak dodawanie nowych gałęzi do drzewa, a ich usuwanie to jak obcinanie gałęzi.

Podsumowanie

  • Możemy pobierać elementy DOM za pomocą metod takich jak getElementById() i querySelector().
  • Możemy zmieniać treść, atrybuty i style elementów za pomocą JavaScript.
  • Możemy tworzyć nowe elementy za pomocą createElement() i usuwać istniejące elementy za pomocą remove().
