Struktury Kontrolne

Struktury kontrolne w JavaScript pozwalają na podejmowanie decyzji i wielokrotne wykonywanie kodu w zależności od określonych warunków. Poniżej znajdziesz omówienie instrukcji warunkowych oraz pętli w JavaScript.

Instrukcje warunkowe

Instrukcja if

Instrukcja if pozwala na wykonanie pewnej części kodu, jeśli określony warunek jest spełniony (prawdziwy). Jeśli warunek jest fałszywy, można użyć instrukcji else lub else if do wykonania innego bloku kodu.


let age = 18;

if (age >= 18) {
    console.log("Jesteś pełnoletni");
} else {
    console.log("Nie jesteś pełnoletni");
}

Instrukcja switch

Instrukcja switch jest używana, gdy mamy wiele możliwych wartości do sprawdzenia dla jednej zmiennej. Zamiast wielu instrukcji if, używamy jednej instrukcji switch.


let color = "red";

switch (color) {
    case "red":
        console.log("Kolor to czerwony");
        break;
    case "blue":
        console.log("Kolor to niebieski");
        break;
    default:
        console.log("Inny kolor");
}

W tym przykładzie, jeśli wartość zmiennej color to "red", zostanie wyświetlona wiadomość "Kolor to czerwony". Instrukcja break kończy wykonywanie bloku switch, zapobiegając sprawdzeniu kolejnych przypadków.

Pętle

Pętle pozwalają na wielokrotne wykonywanie określonego bloku kodu, dopóki spełniony jest pewien warunek.

Pętla for

Pętla for jest najczęściej używaną pętlą w JavaScript. Składa się z trzech części: inicjalizacji zmiennej, warunku, który musi być spełniony, aby kontynuować pętlę, oraz instrukcji, która jest wykonywana po każdej iteracji (np. inkrementacja zmiennej).


for (let i = 0; i < 5; i++) {
    console.log(i);  // Wyświetla liczby od 0 do 4
}

Pętla while

Pętla while działa, dopóki warunek w nawiasach jest spełniony. Warunek jest sprawdzany przed każdą iteracją pętli.


let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

Pętla do...while

Pętla do...while działa podobnie do while, z tą różnicą, że kod w bloku do jest wykonywany przynajmniej raz, nawet jeśli warunek nie jest spełniony.


let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Pętla for...of

Pętla for...of iteruje po elementach dowolnej iterowalnej struktury danych, takiej jak tablica.


let fruits = ["apple", "banana", "cherry"];
for (let fruit of fruits) {
    console.log(fruit); // Wyświetla kolejno "apple", "banana", "cherry"
}

Pętla forEach

Metoda forEach jest stosowana do iteracji po elementach tablicy. Wykonuje podaną funkcję dla każdego elementu tablicy.


let numbers = [1, 2, 3, 4, 5];
numbers.forEach(function(number) {
    console.log(number); // Wyświetla kolejno 1, 2, 3, 4, 5
});
