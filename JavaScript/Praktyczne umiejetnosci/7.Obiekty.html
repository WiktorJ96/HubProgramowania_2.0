Obiekty

Obiekty w JavaScript to struktury, które pozwalają przechowywać dane w postaci par klucz-wartość. Dzięki obiektom możemy tworzyć bardziej złożone struktury danych, które zawierają różne właściwości i funkcje (nazywane metodami).

Tworzenie obiektów

Obiekt w JavaScript można stworzyć za pomocą literału obiektowego ({}). Wewnątrz obiektu możemy zdefiniować różne właściwości, które są przechowywane w postaci pary klucz: wartość.


const car = {
    brand: "Toyota",
    model: "Corolla",
    year: 2020
};
console.log(car);

Wyobraź sobie obiekt jako pudełko z etykietkami (kluczami). Każda etykieta wskazuje, co znajduje się w pudełku. W naszym przypadku mamy klucze brand, model i year, a wartości to odpowiednio "Toyota", "Corolla" i "2020".

Manipulacja obiektami

Do właściwości obiektu możemy odwołać się za pomocą kropki lub nawiasów kwadratowych.


// Odwołanie się do właściwości obiektu
console.log(car.brand); // Wynik: "Toyota"

// Modyfikacja właściwości obiektu
car.year = 2021;
console.log(car.year); // Wynik: 2021

// Dodanie nowej właściwości
car.color = "Red";
console.log(car["color"]); // Wynik: "Red"

W analogii pudełka, manipulacja obiektem jest jak zmiana zawartości pudełka: możemy zmieniać istniejące przedmioty (modyfikować właściwości) lub dodać nowe (dodanie właściwości).

Metody obiektów

Obiekty mogą również przechowywać funkcje jako swoje właściwości. Te funkcje nazywamy metodami.


const person = {
    name: "Jan",
    age: 30,
    greet: function() {
        console.log("Cześć, mam na imię " + this.name);
    }
};

person.greet(); // Wynik: "Cześć, mam na imię Jan"

Metody w obiekcie działają jak specjalne narzędzia w pudełku. W przypadku osoby (person), narzędziem jest funkcja greet, która wykonuje pewne zadanie (wyświetla wiadomość powitalną).

Iteracja po obiektach

Istnieje kilka sposobów na iterowanie (przechodzenie) po właściwościach obiektu. Najbardziej powszechną metodą jest użycie pętli for...in.


const book = {
    title: "Władca Pierścieni",
    author: "J.R.R. Tolkien",
    year: 1954
};

for (let key in book) {
    console.log(key + ": " + book[key]);
}

Pętla for...in przechodzi przez wszystkie właściwości obiektu book.

Wyobraź sobie, że obiekt book to dom z trzema szufladami. Każda szuflada ma etykietkę (klucz) na zewnątrz, która mówi, co znajduje się w środku, i w każdej szufladzie znajduje się jakiś przedmiot (wartość). Pętla for...in to osoba, która przechodzi przez wszystkie szuflady w domu i otwiera każdą z nich, czytając etykietkę i sprawdzając, co jest w środku.

W przykładzie powyżej:

  • key to etykietka na szufladzie (np. "title", "author", "year").
  • book[key] to zawartość tej szuflady (np. "Władca Pierścieni", "J.R.R. Tolkien", 1954).

Pętla for...in działa tak, że osoba (pętla) otwiera każdą szufladę, odczytuje jej etykietkę i sprawdza, co jest w środku. Na końcu wypisuje informacje o każdej szufladzie: etykietkę (klucz) i jej zawartość (wartość).

W tym przypadku pętla for...in wypisze:

  • Pierwsza szuflada: etykietka "title" i zawartość "Władca Pierścieni".
  • Druga szuflada: etykietka "author" i zawartość "J.R.R. Tolkien".
  • Trzecia szuflada: etykietka "year" i zawartość 1954.

Podsumowanie

  • Obiekty w JavaScript to struktury przechowujące dane w postaci par klucz-wartość.
  • Możemy odwoływać się do właściwości obiektów za pomocą notacji kropkowej lub nawiasów kwadratowych.
  • Obiekty mogą zawierać funkcje, które nazywamy metodami.
  • Pętla for...in pozwala na iterowanie po wszystkich właściwościach obiektu.
