Składnia, Zmienne i Typy Danych

JavaScript to język skryptowy, który używany jest głównie do tworzenia interaktywnych stron internetowych. Poniżej omówione zostaną podstawowe elementy JavaScript, takie jak składnia, zmienne, komentarze, typy danych oraz literały.

Składnia

Składnia JavaScript odnosi się do reguł, według których piszemy kod. Każda instrukcja kończy się zazwyczaj średnikiem (;), choć nie jest to zawsze wymagane. Poniżej znajduje się prosty przykład składni JavaScript:


        let name = "John"; // Deklaracja zmiennej i przypisanie wartości
        console.log(name);  // Wyświetlenie wartości zmiennej w konsoli

Komentarze

Komentarze to sposób na dodanie notatek w kodzie, które nie są wykonywane przez program. Można używać komentarzy jednolinijkowych lub wielolinijkowych:


// To jest komentarz jednolinijkowy

/*
To jest komentarz
wielolinijkowy
*/

Typy Danych

JavaScript posiada dwa główne typy danych: proste (prymitywne) i referencyjne. Typy proste obejmują liczby, teksty, wartość logiczną i inne, podczas gdy typy referencyjne odnoszą się do bardziej złożonych struktur jak obiekty i tablice.

Typy proste

  • Number: Przechowuje liczby (np. let age = 25;).
  • String: Przechowuje tekst (np. let name = "Alice";).
  • Boolean: Przechowuje wartość logiczną true lub false (np. let isStudent = true;).
  • Null: Specjalna wartość oznaczająca brak wartości (np. let empty = null;).
  • Undefined: Oznacza zmienną, która została zadeklarowana, ale nie przypisano jej wartości (np. let notDefined;).

Typy referencyjne

  • Obiekty: Struktury przechowujące pary klucz-wartość (np. let person = { name: "John", age: 30 };).
  • Tablice: Lista elementów (np. let colors = ["red", "green", "blue"];).

Zmienne

Zmienna to nazwana przestrzeń, w której przechowywana jest wartość. Możemy zadeklarować zmienne za pomocą var, let oraz const.

  • var: Zmienne zadeklarowane za pomocą var mają zasięg funkcyjny, co oznacza, że są dostępne w całej funkcji, w której są zadeklarowane.
  • let: Zmienne zadeklarowane za pomocą let mają zasięg blokowy, co oznacza, że są dostępne tylko w bloku kodu, w którym zostały zadeklarowane.
  • const: Zmienne zadeklarowane za pomocą const również mają zasięg blokowy, ale ich wartości nie można zmienić po przypisaniu.

Literały

Literały to wartości, które pojawiają się bezpośrednio w kodzie. Oto przykłady literałów:

  • Literał liczbowy: let x = 10;
  • Literał łańcuchowy: let text = "Hello, World!";
  • Literał tablicowy: let fruits = ["apple", "banana", "cherry"];
  • Literał obiektowy: let car = {model: "Toyota", year: 2020};
