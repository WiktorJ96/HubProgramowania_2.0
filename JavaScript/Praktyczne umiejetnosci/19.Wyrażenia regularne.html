Wyrażenia Regularne

Wyrażenia regularne (ang. regular expressions, w skrócie regex) to wzorce, które pozwalają na wyszukiwanie, dopasowywanie i manipulowanie ciągami znaków. W JavaScript wyrażenia regularne mogą być tworzone za pomocą literałów lub obiektów klasy RegExp.

Tworzenie wyrażeń regularnych

Wyrażenia regularne można tworzyć na dwa sposoby:

  • Za pomocą literałów regex: /wzorzec/
  • Za pomocą konstruktora RegExp: new RegExp('wzorzec')

Przykład: Tworzenie wyrażenia regularnego


// Literał wyrażenia regularnego
const regex1 = /abc/;

// Konstruktor RegExp
const regex2 = new RegExp('abc');

Metody testowania wyrażeń regularnych

JavaScript udostępnia kilka metod do pracy z wyrażeniami regularnymi, takich jak test() i match(). Poniżej wyjaśniono ich działanie.

Metoda test()

Metoda test() sprawdza, czy wyrażenie regularne pasuje do podanego ciągu. Zwraca true lub false.

Przykład: Użycie test()


const regex = /kot/;
const text = "Mam kota w domu";

// Sprawdzamy, czy w tekście znajduje się słowo "kot"
const result = regex.test(text);
console.log(result); // true

Metoda match()

Metoda match() zwraca tablicę wszystkich dopasowań do wzorca w ciągu. Jeśli nie znajdzie dopasowań, zwraca null.

Przykład: Użycie match()


const text = "123-456-7890 to mój numer telefonu";
const regex = /\d{3}-\d{3}-\d{4}/;

const matchResult = text.match(regex);
console.log(matchResult); // ["123-456-7890"]

W tym przykładzie wyrażenie regularne /\d{3}-\d{3}-\d{4}/ szuka numeru telefonu w formacie XXX-XXX-XXXX. Metoda match() zwraca dopasowanie jako tablicę.

Metoda replace()

Metoda replace() pozwala zamieniać dopasowane ciągi na inny tekst. Jest to przydatne do modyfikowania tekstu na podstawie wzorca.

Przykład: Użycie replace()


const text = "Numer telefonu: 123-456-7890";
const regex = /\d{3}-\d{3}-\d{4}/;

// Zamieniamy numer telefonu na "XXX-XXX-XXXX"
const newText = text.replace(regex, 'XXX-XXX-XXXX');
console.log(newText); // "Numer telefonu: XXX-XXX-XXXX"

Flagi wyrażeń regularnych

Flagi modyfikują sposób działania wyrażeń regularnych. Możemy ich używać z literałami regex, dodając je po ostatnim ukośniku. Oto najczęściej używane flagi:

  • g – Globalne wyszukiwanie, czyli dopasowywanie wszystkich wystąpień w ciągu.
  • i – Ignorowanie wielkości liter.
  • m – Wieloliniowe dopasowanie (uwzględnia znaki nowej linii).

Przykład: Użycie flag


const text = "Ala ma Kota. ala ma psa.";
const regex = /ala/gi; // 'g' - globalne wyszukiwanie, 'i' - ignorowanie wielkości liter

const matches = text.match(regex);
console.log(matches); // ["Ala", "ala"]

Przydatne wzorce w wyrażeniach regularnych

  • \d – Dowolna cyfra.
  • \w – Dowolna litera, cyfra lub podkreślnik.
  • \s – Dowolna spacja.
  • ^ – Początek ciągu.
  • $ – Koniec ciągu.
  • [abc] – Dowolny znak z grupy a, b, c.
  • (abc) – Grupowanie znaków.
  • {n} – Dokładnie n wystąpień poprzedzającego elementu.
  • {n,m} – Od n do m wystąpień poprzedzającego elementu.

Przykład: Sprawdzenie formatu e-maila


const email = "jan.kowalski@example.com";
const emailRegex = /^[\w.-]+@[a-zA-Z\d.-]+\.[a-zA-Z]{2,}$/;

const isValidEmail = emailRegex.test(email);
console.log(isValidEmail); // true

Wyrażenie regularne /^[\w.-]+@[a-zA-Z\d.-]+\.[a-zA-Z]{2,}$/ sprawdza, czy ciąg jest w formacie e-maila.

Podsumowanie

  • Wyrażenia regularne służą do wyszukiwania, dopasowywania i manipulacji ciągami znaków.
  • Możemy tworzyć wyrażenia za pomocą literałów regex lub obiektu RegExp.
  • Metody test(), match() i replace() pozwalają na pracę z wyrażeniami regularnymi w JavaScript.
  • Flagi, takie jak g i i, modyfikują działanie wyrażeń regularnych.
