Obiekt Number i jego Właściwości w JavaScript

W JavaScript Number jest wbudowanym obiektem służącym do pracy z wartościami liczbowymi. Obejmuje on zarówno liczby całkowite, jak i zmiennoprzecinkowe (liczby z ułamkami). Obiekt Number dostarcza wiele przydatnych właściwości i metod do pracy z liczbami.

Właściwości obiektu Number

Obiekt Number ma kilka wbudowanych stałych, które są przydatne do pracy z liczbami w JavaScript:

  • Number.MAX_VALUE - Największa możliwa wartość liczby, jaką można reprezentować w JavaScript.
  • Number.MIN_VALUE - Najmniejsza dodatnia wartość liczby większa niż 0.
  • Number.POSITIVE_INFINITY - Reprezentuje wartość nieskończoną, wynik dzielenia przez 0.
  • Number.NEGATIVE_INFINITY - Reprezentuje wartość ujemną nieskończoności.
  • Number.NaN - Reprezentuje wartość Not-a-Number (nie jest liczbą), wynik operacji matematycznych, które nie zwracają liczby.

Przykłady użycia właściwości


// Wartości maksymalne i minimalne
console.log(Number.MAX_VALUE); // 1.7976931348623157e+308
console.log(Number.MIN_VALUE); // 5e-324

// Przykłady nieskończoności
console.log(Number.POSITIVE_INFINITY); // Infinity
console.log(Number.NEGATIVE_INFINITY); // -Infinity

// Sprawdzanie wartości NaN
let result = 0 / 0;
console.log(result === Number.NaN); // false (NaN nie jest równe sobie)
console.log(isNaN(result)); // true

W powyższych przykładach pokazano, jak można uzyskać dostęp do wartości skrajnych w JavaScript, takich jak maksymalna i minimalna liczba, a także jak pracować z nieskończonościami oraz sprawdzać, czy wynik operacji matematycznej jest NaN.

Metody obiektu Number

Obiekt Number zawiera kilka metod pomocnych w pracy z liczbami:

  • Number.isFinite() - Sprawdza, czy wartość jest liczbą skończoną (czy nie jest Infinity).
  • Number.isInteger() - Sprawdza, czy wartość jest liczbą całkowitą.
  • Number.isNaN() - Sprawdza, czy wartość jest NaN.
  • Number.parseFloat() - Konwertuje ciąg znaków na liczbę zmiennoprzecinkową.
  • Number.parseInt() - Konwertuje ciąg znaków na liczbę całkowitą.

Przykłady użycia metod


// Sprawdzanie, czy liczba jest skończona
console.log(Number.isFinite(10)); // true
console.log(Number.isFinite(Infinity)); // false

// Sprawdzanie, czy liczba jest całkowita
console.log(Number.isInteger(10)); // true
console.log(Number.isInteger(10.5)); // false

// Sprawdzanie, czy wartość jest NaN
console.log(Number.isNaN(NaN)); // true
console.log(Number.isNaN(10)); // false

// Konwersja ciągu znaków na liczbę zmiennoprzecinkową
console.log(Number.parseFloat('3.14')); // 3.14

// Konwersja ciągu znaków na liczbę całkowitą
console.log(Number.parseInt('42')); // 42
console.log(Number.parseInt('42px')); // 42 (ignoruje tekst po liczbie)

Metody takie jak Number.isFinite(), Number.isInteger() i Number.isNaN() pozwalają na walidację liczb, podczas gdy parseFloat() i parseInt() konwertują ciągi znaków na liczby.

Podsumowanie

  • Obiekt Number oferuje szereg przydatnych właściwości, które pozwalają na pracę z liczbami, w tym maksymalne i minimalne wartości oraz nieskończoności.
  • Metody obiektu Number umożliwiają walidację liczb oraz ich konwersję z ciągów znaków.
  • Number.NaN jest specjalną wartością oznaczającą wynik operacji, która nie zwraca liczby, a Number.isNaN() pozwala na jej sprawdzenie.
