Przeglądanie Właściwości Obiektu w JavaScript

Obiekty w JavaScript to dynamiczne struktury danych, które mogą przechowywać właściwości w formie par klucz-wartość. Istnieje wiele metod, które pozwalają przeglądać te właściwości, m.in. Object.keys(), Object.values(), Object.entries() oraz pętla for...in. Przyjrzymy się bliżej każdej z tych metod.

Przeglądanie kluczy obiektu: Object.keys()


        const car = {
            brand: "Tesla",
            model: "Model S",
            year: 2021
        };

        const keys = Object.keys(car);
        console.log(keys); // ['brand', 'model', 'year']

Metoda Object.keys() zwraca tablicę zawierającą wszystkie klucze (nazwy właściwości) obiektu. Jest to przydatne, gdy chcesz iterować po nazwach kluczy obiektu.

Przeglądanie wartości obiektu: Object.values()


        const values = Object.values(car);
        console.log(values); // ['Tesla', 'Model S', 2021]

Metoda Object.values() zwraca tablicę wartości właściwości obiektu. W tym przypadku zwracane są wartości odpowiadające poszczególnym kluczom w obiekcie car.

Przeglądanie par klucz-wartość: Object.entries()


        const entries = Object.entries(car);
        console.log(entries); // [['brand', 'Tesla'], ['model', 'Model S'], ['year', 2021]]

Metoda Object.entries() zwraca tablicę, której elementy są tablicami dwuelementowymi (pary klucz-wartość). Dzięki temu można łatwo przeglądać zarówno klucze, jak i ich odpowiadające wartości.

Iteracja po właściwościach obiektu: for...in


        for (let key in car) {
            console.log(key + ": " + car[key]);
        }
        // Output:
        // brand: Tesla
        // model: Model S
        // year: 2021

Pętla for...in pozwala iterować po wszystkich enumerowalnych właściwościach obiektu. Działa zarówno dla własnych właściwości obiektu, jak i tych odziedziczonych z prototypu, dlatego często używa się jej z filtrowaniem za pomocą hasOwnProperty().

Sprawdzanie własnych właściwości obiektu: hasOwnProperty()


        for (let key in car) {
            if (car.hasOwnProperty(key)) {
                console.log(key + ": " + car[key]);
            }
        }

Metoda hasOwnProperty() sprawdza, czy właściwość jest bezpośrednio związana z obiektem (czyli nie pochodzi z jego prototypu). Jest to przydatne przy iterowaniu po obiektach, aby uniknąć przetwarzania odziedziczonych właściwości.

Metody wbudowane do manipulacji obiektami

Oprócz przeglądania właściwości, JavaScript oferuje wiele metod do manipulacji obiektami. Oto kilka przydatnych:

  • Object.assign(target, ...sources) - Kopiuje wszystkie własne właściwości z jednego lub więcej obiektów do obiektu docelowego.
  • Object.freeze(obj) - "Zamraża" obiekt, zapobiegając modyfikacjom jego właściwości.
  • Object.seal(obj) - Uszczelnia obiekt, zapobiegając dodawaniu nowych właściwości, ale pozwala na modyfikację istniejących.
