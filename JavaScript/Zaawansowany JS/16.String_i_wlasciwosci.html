Obiekt String i jego Właściwości w JavaScript

W JavaScript obiekt String reprezentuje ciągi znaków i dostarcza wielu metod umożliwiających manipulację tekstem. Pomimo że stringi są niemutowalne (nie można ich zmieniać po stworzeniu), obiekt String oferuje liczne narzędzia do pracy z tekstem.

Podstawowe właściwości obiektu String

Obiekt String posiada kilka ważnych właściwości, które pomagają pracować z danymi tekstowymi:

  • length - Zwraca długość ciągu znaków.

Przykład użycia właściwości length


// Przykład użycia length
let text = "JavaScript";
console.log(text.length); // 10

W powyższym przykładzie text.length zwraca liczbę znaków w ciągu tekstowym "JavaScript".

Najczęściej używane metody obiektu String

  • toUpperCase() - Konwertuje wszystkie litery w ciągu na wielkie.
  • toLowerCase() - Konwertuje wszystkie litery w ciągu na małe.
  • charAt(index) - Zwraca znak na określonej pozycji w ciągu.
  • includes(substring) - Sprawdza, czy ciąg zawiera określony podciąg.
  • slice(start, end) - Wyodrębnia część ciągu między podanymi indeksami.
  • split(separator) - Dzieli ciąg na części, tworząc tablicę elementów.
  • trim() - Usuwa białe znaki (spacje) z początku i końca ciągu.

Przykłady użycia metod


// Konwersja na wielkie litery
let lowerText = "hello world";
console.log(lowerText.toUpperCase()); // "HELLO WORLD"

// Sprawdzanie, czy ciąg zawiera podciąg
let text = "JavaScript is great!";
console.log(text.includes("great")); // true

// Wyodrębnianie części ciągu
let str = "JavaScript";
console.log(str.slice(0, 4)); // "Java"

// Dzielenie ciągu na części
let sentence = "Learn, code, repeat";
let words = sentence.split(", ");
console.log(words); // ["Learn", "code", "repeat"]

// Usuwanie białych znaków
let spacedText = "  JavaScript  ";
console.log(spacedText.trim()); // "JavaScript"

Metody takie jak toUpperCase() i toLowerCase() są używane do zmiany wielkości liter. Metody includes(), slice(), i split() umożliwiają manipulację ciągami, a trim() usuwa zbędne spacje.

Tworzenie nowych ciągów znaków

Ponieważ stringi w JavaScript są niemutowalne, operacje na ciągach zawsze tworzą nowy ciąg, a oryginalny tekst pozostaje niezmieniony. Na przykład:


let originalText = "hello";
let upperText = originalText.toUpperCase();

console.log(originalText); // "hello" (oryginalny ciąg niezmieniony)
console.log(upperText); // "HELLO" (nowy ciąg)

Podsumowanie

  • String to wbudowany obiekt w JavaScript, który umożliwia pracę z danymi tekstowymi.
  • Stringi są niemutowalne – wszelkie operacje na stringach zwracają nowe ciągi, a oryginał pozostaje niezmieniony.
  • Obiekt String oferuje wiele metod, takich jak toUpperCase(), slice(), i split(), które ułatwiają manipulację tekstem.
