Enumeracja kluczy i wartości w obiektach JavaScript

W JavaScript obiekty składają się z par klucz-wartość, gdzie kluczem jest ciąg znaków (string), a wartością może być dowolny typ danych. Aby przejść przez wszystkie klucze lub wartości w obiekcie, możemy użyć wbudowanych metod takich jak Object.keys(), Object.values(), i Object.entries().

Metoda Object.keys()

Metoda Object.keys() zwraca tablicę zawierającą wszystkie klucze (nazwy właściwości) obiektu. Jest szczególnie przydatna do iteracji po kluczach obiektu:


const car = { brand: 'Ford', model: 'Mustang', year: 1969 };
const keys = Object.keys(car);
console.log(keys); // ['brand', 'model', 'year']

W tym przykładzie metoda Object.keys() zwraca tablicę wszystkich kluczy (nazw właściwości) obiektu car.

Metoda Object.values()

Metoda Object.values() zwraca tablicę wszystkich wartości obiektu. Jest użyteczna, gdy chcemy iterować po wartościach właściwości obiektu:


const car = { brand: 'Ford', model: 'Mustang', year: 1969 };
const values = Object.values(car);
console.log(values); // ['Ford', 'Mustang', 1969]

Metoda Object.values() zwraca tablicę wszystkich wartości obiektu car.

Metoda Object.entries()

Metoda Object.entries() zwraca tablicę par [klucz, wartość], co pozwala na iterację po zarówno kluczach, jak i wartościach obiektu:


const car = { brand: 'Ford', model: 'Mustang', year: 1969 };
const entries = Object.entries(car);
console.log(entries); // [['brand', 'Ford'], ['model', 'Mustang'], ['year', 1969]]

Dzięki tej metodzie możemy iterować po parach klucz-wartość, co jest przydatne np. w sytuacjach, gdy musimy jednocześnie operować na kluczach i wartościach.

Iteracja z for...in

Alternatywnie, możemy użyć pętli for...in, aby przechodzić po wszystkich wyliczalnych właściwościach obiektu:


const car = { brand: 'Ford', model: 'Mustang', year: 1969 };

for (let key in car) {
  if (car.hasOwnProperty(key)) {
    console.log(`${key}: ${car[key]}`);
  }
}

Pętla for...in iteruje po wszystkich kluczach obiektu. Metoda hasOwnProperty() sprawdza, czy właściwość należy bezpośrednio do obiektu (a nie została odziedziczona).

Przykłady zaawansowane

Możemy także łączyć metody takie jak Object.entries() z innymi funkcjami wyższego rzędu, takimi jak map():


const car = { brand: 'Ford', model: 'Mustang', year: 1969 };
const formattedEntries = Object.entries(car).map(([key, value]) => `${key}: ${value}`);
console.log(formattedEntries); // ['brand: Ford', 'model: Mustang', 'year: 1969']

W powyższym przykładzie użyliśmy Object.entries(), aby przejść przez wszystkie pary klucz-wartość i sformatować je jako ciągi znaków.

Podsumowanie

  • Object.keys() zwraca tablicę kluczy obiektu.
  • Object.values() zwraca tablicę wartości obiektu.
  • Object.entries() zwraca tablicę par [klucz, wartość].
  • Pętla for...in pozwala na iterację po wszystkich wyliczalnych właściwościach obiektu.
