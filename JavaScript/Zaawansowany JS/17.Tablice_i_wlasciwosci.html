Obiekt Array i jego Właściwości w JavaScript

Tablice (Array) w JavaScript są dynamicznymi strukturami danych, które mogą przechowywać wiele elementów różnych typów danych. Każdy element tablicy jest przypisany do indeksu, co pozwala na szybki dostęp do danych. Obiekt Array w JavaScript oferuje wiele metod i właściwości do manipulacji danymi.

Właściwość length

Właściwość length zwraca liczbę elementów w tablicy:


        let fruits = ['jabłko', 'banan', 'pomarańcza'];
        console.log(fruits.length); // 3

W tym przykładzie length zwraca liczbę elementów w tablicy fruits, która wynosi 3.

Najczęściej używane metody obiektu Array

  • push() - Dodaje nowy element na końcu tablicy.
  • pop() - Usuwa ostatni element z tablicy.
  • shift() - Usuwa pierwszy element z tablicy.
  • unshift() - Dodaje nowy element na początku tablicy.
  • concat() - Łączy dwie lub więcej tablic w jedną.
  • slice() - Wyodrębnia część tablicy, tworząc nową tablicę.
  • splice() - Dodaje, usuwa lub zamienia elementy w tablicy.
  • indexOf() - Zwraca pierwszy indeks, pod którym znajduje się dany element.
  • forEach() - Wykonuje funkcję dla każdego elementu tablicy.
  • map() - Tworzy nową tablicę, wykonując funkcję na każdym elemencie tablicy.
  • filter() - Tworzy nową tablicę z elementami, które spełniają warunki określone w funkcji.
  • reduce() - Zmniejsza tablicę do pojedynczej wartości, wykonując funkcję na kolejnych elementach.

Przykłady użycia metod tablic


// Dodawanie elementów
let numbers = [1, 2, 3];
numbers.push(4);
console.log(numbers); // [1, 2, 3, 4]

// Usuwanie elementów
numbers.pop();
console.log(numbers); // [1, 2, 3]

// Przekształcanie tablicy za pomocą map
let squares = numbers.map(n => n * n);
console.log(squares); // [1, 4, 9]

// Filtrowanie tablicy
let evenNumbers = numbers.filter(n => n % 2 === 0);
console.log(evenNumbers); // [2]

// Łączenie tablic
let moreNumbers = [4, 5];
let combined = numbers.concat(moreNumbers);
console.log(combined); // [1, 2, 3, 4, 5]

// Zmniejszanie tablicy do sumy wszystkich elementów
let sum = numbers.reduce((acc, curr) => acc + curr, 0);
console.log(sum); // 6

W powyższych przykładach pokazano kilka najczęściej używanych metod pracy z tablicami, takich jak dodawanie i usuwanie elementów (push, pop), przekształcanie tablic za pomocą map(), filtrowanie za pomocą filter(), oraz sumowanie elementów za pomocą reduce().

Praca z indeksami w tablicy

Każdy element tablicy jest dostępny przez przypisany indeks, który zaczyna się od 0:


let fruits = ['jabłko', 'banan', 'pomarańcza'];
console.log(fruits[0]); // 'jabłko'
console.log(fruits[1]); // 'banan'

Podsumowanie

  • Array to dynamiczna struktura danych w JavaScript, która może przechowywać wiele elementów.
  • Właściwość length zwraca liczbę elementów w tablicy.
  • Tablice oferują wiele metod do manipulowania danymi, w tym dodawanie, usuwanie, filtrowanie, mapowanie i redukowanie elementów.
  • Elementy w tablicy są dostępne poprzez indeksy, zaczynając od 0.
