Obiekty i Metody na Obiektach w JavaScript

Obiekt w JavaScript to struktura danych, która pozwala przechowywać pary klucz-wartość. Obiekty są fundamentalną częścią języka, a ich zastosowanie obejmuje tworzenie bardziej zaawansowanych struktur, takich jak klasy, które później można wykorzystać do tworzenia instancji obiektów. Obiekty można dynamicznie rozszerzać, dodając nowe właściwości lub metody.

Tworzenie Obiektów


        // Prosty obiekt z właściwościami
        const person = {
            name: "John",
            age: 30,
            city: "New York"
        };

        console.log(person.name); // 'John'

W powyższym przykładzie tworzymy prosty obiekt person z trzema właściwościami: name, age i city. Dostęp do tych właściwości uzyskujemy za pomocą notacji kropkowej.

Dodawanie Metod do Obiektów


        const person = {
            name: "John",
            age: 30,
            greet: function() {
                console.log("Hello, my name is " + this.name);
            }
        };

        person.greet(); // 'Hello, my name is John'

Możemy dodać funkcje do obiektów, które stają się metodami tego obiektu. W powyższym przykładzie dodaliśmy metodę greet, która odwołuje się do właściwości name za pomocą słowa kluczowego this.

Metody Wbudowane w Obiekty

JavaScript dostarcza wiele wbudowanych metod dla obiektów, które ułatwiają pracę z nimi. Oto kilka popularnych metod:

  • Object.keys(obj) - Zwraca tablicę kluczy danego obiektu.
  • Object.values(obj) - Zwraca tablicę wartości danego obiektu.
  • Object.entries(obj) - Zwraca tablicę par [klucz, wartość] dla danego obiektu.

        const person = {
            name: "John",
            age: 30,
            city: "New York"
        };

        console.log(Object.keys(person));   // ['name', 'age', 'city']
        console.log(Object.values(person)); // ['John', 30, 'New York']
        console.log(Object.entries(person));// [['name', 'John'], ['age', 30], ['city', 'New York']]

Wbudowane metody takie jak Object.keys, Object.values i Object.entries pozwalają szybko uzyskać dostęp do kluczy, wartości i par klucz-wartość w obiekcie.

Przykładowe Zastosowania

Obiekty są często używane w JavaScript do reprezentowania rzeczywistych encji, takich jak użytkownicy, produkty, czy dane z formularzy. Dzięki dynamicznemu charakterowi obiektów możemy dodawać, usuwać lub modyfikować ich właściwości w trakcie działania programu.


        const car = {
            brand: "Tesla",
            model: "Model 3",
            year: 2021,
            start: function() {
                console.log(this.brand + " " + this.model + " is starting...");
            }
        };

        car.start(); // 'Tesla Model 3 is starting...'

W tym przykładzie obiekt car reprezentuje samochód z właściwościami i metodą start, która wypisuje komunikat o uruchomieniu samochodu.
