Wyjaśnienie działania operatora spread

Operator spread (...) w JavaScript umożliwia "rozpakowywanie" elementów z tablic, obiektów lub iterowalnych struktur danych na pojedyncze elementy. Operator ten pozwala np. na rozdzielenie tablic na poszczególne wartości, co jest przydatne podczas kopiowania lub łączenia tablic.

Przykład użycia operatora spread


        const arr1 = [1, 2, 3];
        const arr2 = [...arr1, 4, 5, 6];

        console.log(arr2); // [1, 2, 3, 4, 5, 6]

W tym przykładzie operator spread rozpakowuje elementy z tablicy arr1 i dodaje je do nowej tablicy arr2, co umożliwia szybkie kopiowanie i dodawanie elementów.

Rozpakowywanie obiektów


        const person = { name: "John", age: 25 };
        const newPerson = { ...person, city: "New York" };

        console.log(newPerson); // { name: "John", age: 25, city: "New York" }

Podobnie jak w przypadku tablic, operator spread pozwala na rozpakowywanie właściwości obiektu, co ułatwia tworzenie nowych obiektów poprzez kopiowanie istniejących właściwości.

Zastosowanie operatora spread

Operator spread ma wiele zastosowań, takich jak:

  • Kopiowanie tablic lub obiektów bez zmieniania oryginałów.
  • Łączenie wielu tablic w jedną.
  • Przekazywanie elementów tablicy jako argumentów funkcji.
