Obiekt window w JavaScript

Obiekt window w JavaScript jest globalnym obiektem, który reprezentuje przeglądarkę internetową i jej okno. Daje on dostęp do wielu funkcji i metod pozwalających na interakcję z przeglądarką, zarządzanie elementami dokumentu HTML, obsługę zdarzeń i wiele więcej. Każda strona internetowa w przeglądarce ma swój obiekt window, co pozwala na wykonywanie operacji związanych z oknem przeglądarki.

Globalny obiekt


// Przykład globalnego dostępu
console.log(window);
console.log(window === this); // true

Obiekt window jest dostępny globalnie w kontekście przeglądarki. Możesz odnosić się do niego po prostu jako window lub przez this, gdy działasz w kontekście globalnym.

Metody związane z przeglądarką

Obiekt window zapewnia dostęp do metod, które umożliwiają interakcję z oknem przeglądarki, takich jak zmiana rozmiaru, przewijanie, otwieranie nowych okien itp.


// Przewinięcie okna do góry
window.scrollTo(0, 0);

// Otwieranie nowego okna
window.open("https://www.example.com");

// Zamykanie okna
window.close();

Metody takie jak scrollTo(), open(), i close() pozwalają na manipulację oknem przeglądarki. Należy pamiętać, że niektóre z tych metod mogą być ograniczone przez politykę bezpieczeństwa przeglądarki.

Właściwości obiektu window


console.log(window.innerHeight);  // Wysokość okna przeglądarki
console.log(window.innerWidth);   // Szerokość okna przeglądarki

console.log(window.location.href); // URL aktualnej strony
console.log(window.navigator.userAgent); // Informacje o przeglądarce

Obiekt window zawiera wiele właściwości, takich jak innerHeight i innerWidth, które informują o wymiarach okna, oraz location, która daje dostęp do adresu URL i innych informacji związanych z nawigacją. navigator umożliwia dostęp do informacji o przeglądarce, takich jak wersja i system operacyjny.

Praca z obiektem window.document

Obiekt document jest częścią obiektu window, który zapewnia dostęp do całego drzewa DOM (Document Object Model) strony internetowej. Dzięki temu możemy manipulować elementami HTML, dodawać nowe elementy i reagować na zdarzenia.


console.log(window.document.title); // Pobranie tytułu strony

// Zmiana tytułu strony
window.document.title = "Nowy Tytuł";

Obiekt window.document umożliwia dostęp do elementów strony, co pozwala na ich dynamiczną modyfikację.

Metody alert, confirm i prompt


// Wyświetlenie prostego alertu
window.alert("To jest wiadomość alertu!");

// Potwierdzenie przez użytkownika
const result = window.confirm("Czy chcesz kontynuować?");
console.log(result); // true lub false

// Pobranie danych od użytkownika
const name = window.prompt("Jak masz na imię?");
console.log(name);

Obiekt window oferuje także metody do interakcji z użytkownikiem, takie jak alert (wyświetla komunikat), confirm (prosi o potwierdzenie), i prompt (prosi o wprowadzenie danych).

Zdarzenia związane z obiektem window


// Obsługa zmiany rozmiaru okna
window.addEventListener('resize', () => {
    console.log('Rozmiar okna został zmieniony.');
});

// Obsługa przewijania okna
window.addEventListener('scroll', () => {
    console.log('Strona jest przewijana.');
});

Obiekt window pozwala także na reagowanie na różne zdarzenia, takie jak zmiana rozmiaru okna lub przewijanie strony. Do tego celu używamy metody addEventListener, która nasłuchuje na określone zdarzenia.
