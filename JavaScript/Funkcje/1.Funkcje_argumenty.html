Funkcje i Argumenty w JavaScript

W JavaScript funkcje są podstawową metodą grupowania kodu, który możemy wielokrotnie wywoływać. Funkcje mogą przyjmować argumenty, które przekazują dane do funkcji, i zwracać wartości po wykonaniu obliczeń lub operacji.

Definiowanie funkcji

Funkcje mogą być definiowane na różne sposoby. Najpopularniejsze sposoby to deklaracja funkcji za pomocą słowa kluczowego function oraz funkcje strzałkowe (arrow functions). Oto przykłady:

Standardowa funkcja


function greet(name) {
    return 'Cześć, ' + name + '!';
}

console.log(greet('Anna')); // Cześć, Anna!

W powyższym przykładzie funkcja greet przyjmuje jeden argument name i zwraca powitanie.

Funkcja strzałkowa


const greet = (name) => 'Cześć, ' + name + '!';

console.log(greet('Marek')); // Cześć, Marek!

Funkcje strzałkowe są bardziej zwięzłą formą zapisu funkcji, a także mają inne zachowanie w odniesieniu do wartości this.

Argumenty funkcji

Funkcje mogą przyjmować argumenty, które przekazujemy do nich w momencie wywołania. Argumenty są dostępne wewnątrz funkcji i mogą być używane do wykonywania obliczeń.

Przykład z wieloma argumentami


function add(a, b) {
    return a + b;
}

console.log(add(5, 10)); // 15

Funkcja add przyjmuje dwa argumenty a i b i zwraca ich sumę.

Domyślne wartości argumentów

W JavaScript możemy ustawiać domyślne wartości dla argumentów. Jeśli argument nie zostanie przekazany, funkcja użyje wartości domyślnej.


function greet(name = 'Gość') {
    return 'Cześć, ' + name + '!';
}

console.log(greet()); // Cześć, Gość!

Jeśli w powyższym przykładzie nie przekażemy argumentu do funkcji greet, zostanie użyta domyślna wartość 'Gość'.

Argumenty z użyciem operatora rest

Funkcje mogą przyjmować zmienną liczbę argumentów za pomocą operatora rest (...), co pozwala na elastyczne przekazywanie dowolnej liczby wartości.


function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1, 2, 3, 4, 5)); // 15

W tym przykładzie funkcja sum przyjmuje dowolną liczbę argumentów, które są przekazywane jako tablica, a następnie zwraca ich sumę.

Funkcja z argumentami obiektu

Argumenty funkcji mogą być przekazywane jako obiekty, co umożliwia przekazywanie wielu parametrów w jednym wywołaniu.


function createUser({name, age, city}) {
    return `Nazwa: ${name}, Wiek: ${age}, Miasto: ${city}`;
}

console.log(createUser({name: 'Anna', age: 25, city: 'Warszawa'}));
// Nazwa: Anna, Wiek: 25, Miasto: Warszawa

W tym przykładzie przekazujemy jeden argument jako obiekt, co pozwala na bardziej zrozumiałe przekazywanie wielu parametrów do funkcji.

Podsumowanie

  • Funkcje w JavaScript mogą przyjmować argumenty, które przekazują dane do funkcji.
  • Argumenty mogą mieć wartości domyślne, jeśli nie zostaną przekazane przy wywołaniu.
  • Za pomocą operatora rest (...) możemy przekazywać zmienną liczbę argumentów do funkcji.
  • Funkcje mogą także przyjmować argumenty jako obiekty, co umożliwia łatwiejsze zarządzanie większą liczbą parametrów.
