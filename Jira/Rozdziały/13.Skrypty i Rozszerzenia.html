Skrypty i Rozszerzenia

Jira pozwala na rozbudowę swoich funkcji poprzez skrypty i rozszerzenia. Dzięki temu możesz tworzyć niestandardowe funkcje oraz integrować system z zewnętrznymi aplikacjami poprzez REST API. Wśród najpopularniejszych narzędzi znajduje się ScriptRunner.

ScriptRunner

ScriptRunner to potężne rozszerzenie dla Jira, które umożliwia pisanie skryptów w języku Groovy. Umożliwia ono:

  • Tworzenie niestandardowych warunków, walidatorów i akcji w workflow
  • Automatyzację zadań administracyjnych
  • Dodawanie nowych przycisków i elementów interfejsu

Tworzenie Niestandardowych Funkcji

Tworzenie niestandardowych funkcji pozwala dostosować Jira do specyficznych potrzeb Twojego zespołu. Możesz na przykład:

  • Definiować niestandardowe pola kalkulacyjne
  • Rozszerzać możliwości języka zapytań JQL poprzez dodawanie własnych funkcji
  • Automatyzować procesy poprzez uruchamianie skryptów w określonych momentach workflow

Przykładowy skrypt w ScriptRunner (Groovy), który pobiera podsumowanie zadania:


import com.atlassian.jira.component.ComponentAccessor

// Pobieramy menadżera zadań
def issueManager = ComponentAccessor.getIssueManager()
// Pobieramy zadanie o kluczu "PROJ-123"
def issue = issueManager.getIssueByCurrentKey("PROJ-123")
return issue.summary

Powyższy skrypt zwraca podsumowanie zadania o kluczu PROJ-123.

Integracja z REST API Jiry

REST API Jiry umożliwia komunikację między Jira a zewnętrznymi systemami. Dzięki niemu można wykonywać operacje takie jak:

  • Pobieranie danych o zadaniach, projektach, użytkownikach czy workflow
  • Tworzenie, edytowanie i usuwanie zadań za pomocą wywołań HTTP
  • Integrację z narzędziami takimi jak Confluence, Slack, czy własnymi aplikacjami backend

Przykładowe zapytanie CURL do pobrania danych o zadaniu:


curl -u username:password -X GET -H "Content-Type: application/json" "https://your-jira-instance.atlassian.net/rest/api/2/issue/PROJ-123"

Takie zapytanie zwróci dane zadania w formacie JSON, które następnie można wykorzystać w innych aplikacjach lub do analizy.
