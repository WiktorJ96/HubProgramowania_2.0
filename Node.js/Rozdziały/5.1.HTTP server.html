Kod Node.js


        // server.js
const http = require('http');

// Tworzymy serwer HTTP
const server = http.createServer((req, res) => {
    // Ustawiamy nagłówki odpowiedzi
    res.setHeader('Content-Type', 'text/html; charset=utf-8');
    
    // Obsługa różnych ścieżek
    if (req.url === '/') {
        res.statusCode = 200;
        res.end('<h1>Witaj na stronie głównej!</h1><p>To jest prosty serwer HTTP w Node.js</p>');
    } else if (req.url === '/json') {
        res.setHeader('Content-Type', 'application/json');
        res.statusCode = 200;
        res.end(JSON.stringify({ message: 'To jest odpowiedź JSON' }));
    } else if (req.url === '/error') {
        res.statusCode = 404;
        res.end('<h1>Błąd 404</h1><p>Strona nie została znaleziona</p>');
    } else {
        res.statusCode = 404;
        res.end('<h1>Błąd 404</h1><p>Strona nie istnieje</p>');
    }
});

// Uruchamiamy serwer na porcie 3000
server.listen(3000, () => {
    console.log('Serwer działa na http://localhost:3000');
});

Jak działa projekt?

Projekt ten demonstruje tworzenie prostego serwera HTTP w Node.js. Poniżej znajduje się szczegółowy opis przepływu danych i komunikacji:

  1. Frontend (HTML):

    Użytkownik klika jeden z przycisków, aby wysłać żądanie HTTP na określoną ścieżkę (`/`, `/json` lub `/error`). Każdy przycisk przekierowuje użytkownika na odpowiedni endpoint serwera.

  2. Serwer HTTP (Node.js w server.js):

    Serwer w pliku server.js obsługuje różne żądania przychodzące na różne ścieżki URL. Każda ścieżka zwraca inny typ odpowiedzi (tekst HTML, JSON lub komunikat błędu).

Podsumowanie

Ten projekt pokazuje podstawy tworzenia serwera HTTP w Node.js bez użycia dodatkowych bibliotek, co pozwala lepiej zrozumieć, jak działa przetwarzanie żądań HTTP na niskim poziomie.
