Kod Node.js z Express.js


        // server.js
const express = require('express');
const app = express();
const PORT = 3000;

// Endpoint główny
app.get('/', (req, res) => {
    res.send('<h1>Witaj na stronie głównej!</h1><p>To jest serwer Express w Node.js</p>');
});

// Endpoint JSON
app.get('/json', (req, res) => {
    res.json({ message: 'To jest odpowiedź JSON' });
});

// Endpoint błędu
app.get('/error', (req, res) => {
    res.status(404).send('<h1>Błąd 404</h1><p>Strona nie została znaleziona</p>');
});

// Uruchomienie serwera
app.listen(PORT, () => {
    console.log(`Serwer działa na http://localhost:${PORT}`);
});

Jak działa projekt?

Projekt ten demonstruje użycie Express.js w Node.js. Poniżej znajduje się szczegółowy opis przepływu danych i komunikacji:

  1. Frontend (HTML):

    Użytkownik klika jeden z przycisków, aby wysłać żądanie HTTP na określoną ścieżkę (`/`, `/json` lub `/error`). Każdy przycisk przekierowuje użytkownika na odpowiedni endpoint serwera.

  2. Serwer Express (Node.js w server.js):

    Serwer w pliku server.js obsługuje różne żądania przychodzące na różne ścieżki URL. Każda ścieżka zwraca inny typ odpowiedzi (HTML, JSON lub komunikat błędu 404).

Podsumowanie

Ten projekt pokazuje, jak Express.js upraszcza zarządzanie ścieżkami i odpowiedziami HTTP w Node.js. Dzięki Express łatwo tworzyć endpointy, które zwracają różne typy danych i odpowiednio reagują na różne żądania.
