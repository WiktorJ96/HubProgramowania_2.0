Podstawy Express.js

Express.js to popularny framework dla Node.js, który upraszcza tworzenie aplikacji webowych i API. Oferuje narzędzia do szybkiego tworzenia serwerów, obsługi żądań i odpowiedzi oraz zarządzania routingiem.

Tworzenie serwera

Aby utworzyć podstawowy serwer za pomocą Express.js, potrzebujesz zainstalowanego Node.js i pakietu express. Poniżej znajduje się przykład prostego serwera:


const express = require('express');
// Importowanie frameworka Express

const app = express();
// Tworzenie nowej aplikacji Express

const PORT = 3000;
// Definiowanie portu, na którym serwer będzie nasłuchiwał

app.get('/', (req, res) => {
    res.send('Witaj w Express.js!');
    // Obsługa żądania GET na ścieżce '/' i wysyłanie odpowiedzi
});

app.listen(PORT, () => {
    console.log(`Serwer działa na http://localhost:${PORT}`);
    // Uruchomienie serwera i nasłuchiwanie na określonym porcie
});

Powyższy kod tworzy serwer, który odpowiada na żądanie GET do ścieżki głównej / i zwraca wiadomość \"Witaj w Express.js!\".

Obsługa żądań i odpowiedzi

Express.js umożliwia obsługę różnych typów żądań HTTP, takich jak GET, POST, PUT i DELETE. Oto przykład obsługi żądania POST:


app.post('/dane', (req, res) => {
    res.send('Dane zostały odebrane.');
    // Obsługa żądania POST na ścieżce '/dane' i wysyłanie odpowiedzi
});

W ten sposób Express.js umożliwia obsługę różnych endpointów aplikacji.
