Podstawy SASS

SASS (Syntactically Awesome Style Sheets) to rozszerzenie CSS, które pozwala na łatwiejsze zarządzanie kodem stylów poprzez wprowadzenie zmiennych, zagnieżdżania, mixinów i innych funkcji. SASS oferuje dwie składnie: SCSS (najczęściej używana) oraz SASS.

Składnia SCSS

SCSS to składnia, która jest w pełni zgodna z CSS, ale pozwala na użycie dodatkowych funkcji SASS. Składnia SCSS używa nawiasów klamrowych {} i średników ;, podobnie jak CSS.


// Przykład składni SCSS
$primary-color: #3498db;

.container {
    color: $primary-color;
    padding: 10px;
    
    .header {
        font-size: 24px;
        margin-bottom: 15px;
    }
}

Składnia SASS

Składnia SASS jest bardziej uproszczona i nie wymaga użycia nawiasów klamrowych ani średników. Zamiast tego, stosuje wcięcia do struktury kodu, podobnie jak Python.


// Przykład składni SASS
$primary-color: #3498db

.container
    color: $primary-color
    padding: 10px
    
    .header
        font-size: 24px
        margin-bottom: 15px

Kompilacja Plików SASS do CSS

Pliki SASS (z rozszerzeniami .scss lub .sass) muszą być skompilowane do CSS, aby mogły być użyte w przeglądarkach. Do kompilacji można użyć narzędzia sass z linii komend lub wbudowanych narzędzi w edytorach kodu.


# Kompilacja pojedynczego pliku SCSS do CSS
sass styles.scss styles.css

# Kompilacja w trybie watch (automatyczne wykrywanie zmian)
sass --watch styles.scss:styles.css

Po wykonaniu tej komendy plik styles.scss zostanie skompilowany do styles.css, gotowego do użycia w projekcie.

Integracja SASS z Projektem

Aby używać plików SASS w projekcie, trzeba dodać skompilowany plik CSS do HTML lub, w przypadku projektów takich jak React, dodać bezpośrednią obsługę plików SCSS.

Integracja z HTML

Po skompilowaniu można podłączyć plik CSS w kodzie HTML za pomocą standardowego linku:


<link rel="stylesheet" href="styles.css">

Integracja z React

Aby dodać SASS do projektu React, należy najpierw zainstalować pakiet node-sass (w przypadku projektów opartych o webpack, takich jak create-react-app, jest on domyślnie obsługiwany):


npm install node-sass

Następnie można importować pliki .scss bezpośrednio w komponentach React:


import './styles.scss';

function App() {
    return (
        <div className="container">
            <h1>Witaj w aplikacji!</h1>
        </div>
    );
}

export default App;

Podstawowe Funkcje SASS

SASS oferuje szereg funkcji, które ułatwiają zarządzanie stylami:

  • Zagnieżdżanie: umożliwia zagnieżdżanie selektorów.
  • Zmiennie: pozwalają na przechowywanie wartości, takich jak kolory i rozmiary.
  • Mixiny: pozwalają na wielokrotne użycie bloków kodu.
  • Dziedziczenie: umożliwia użycie stylów z innego selektora.
