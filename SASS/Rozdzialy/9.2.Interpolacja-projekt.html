Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// Użycie interpolacji do generowania klas przycisków
$colors: (primary: #3498db, secondary: #2ecc71, danger: #e74c3c);

@each $name, $color in $colors {
    .button-#{$name} {
        background-color: $color;
        color: #fff;
        padding: 10px;
    }
}

// Interpolacja dla niestandardowych właściwości CSS
$theme-color: #ff6f61;

:root {
    --main-color: #{$theme-color};
}

.button-custom {
    background-color: var(--main-color);
    color: #fff;
    padding: 10px 20px;
    border-radius: 5px;
}

// Generowanie kolumn z niebieskim obramowaniem
@for $i from 1 through 3 {
    .column-#{$i} {
        width: #{$i * 10}%;
        border: 2px solid blue;
    }
}

// Responsywne kontenery
$breakpoints: (small: 576px, medium: 768px, large: 992px);

@each $size, $px in $breakpoints {
    @media (min-width: $px) {
        .container-#{$size} {
            max-width: $px;
        }
    }
}
