Nagłówek Modułowy

Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// _variables.scss - plik zmiennych
$default-font: 'Arial, sans-serif';
$primary-color: #3498db;
$secondary-color: #2ecc71;

// _header.scss - plik stylu nagłówka
@use 'variables';

.header {
    background-color: variables.$primary-color;
    color: #fff;
    padding: 20px;
    text-align: center;
    font-family: variables.$default-font;
}

// _button.scss - plik stylu przycisku
@use 'variables';
@use 'mixins';

.button {
    @include mixins.rounded(10px);
    background-color: variables.$secondary-color;
    color: #fff;
    padding: 10px 20px;
    cursor: pointer;
    border: none;
}

// _mixins.scss - plik z mixinami
@mixin rounded($radius) {
    border-radius: $radius;
}

// main.scss - główny plik stylów
@use 'variables';
@use 'header';
@use 'button';
@use 'mixins';
