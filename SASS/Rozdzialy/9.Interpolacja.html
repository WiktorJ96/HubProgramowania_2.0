SASS - Interpolacja

Interpolacja w SASS umożliwia dynamiczne generowanie nazw selektorów, właściwości i wartości. Jest to szczególnie przydatne, gdy chcemy tworzyć elementy CSS bazujące na zmiennych lub iteracjach. Interpolacja jest wykonywana za pomocą składni #{}, która pozwala na wstawienie wartości zmiennej bezpośrednio w nazwie lub właściwości.

Podstawowe Użycie Interpolacji w Nazwach Selektorów

Interpolacja może być użyta do dynamicznego tworzenia nazw klas i identyfikatorów, co jest przydatne, gdy chcemy generować klasy w oparciu o wartości zmiennych.


// Definicja podstawowego koloru
$color-name: "primary";
$primary-color: #3498db;

.button-#{$color-name} {
    background-color: $primary-color;
    color: #fff;
    padding: 10px;
}

W powyższym przykładzie interpolacja #{} tworzy klasę .button-primary, która używa wartości zmiennej $color-name jako części nazwy klasy.

Interpolacja w Nazwach Właściwości CSS

Interpolacja może być również użyta do tworzenia dynamicznych nazw właściwości. Jest to przydatne, gdy chcemy generować niestandardowe właściwości lub style CSS oparte na zmiennych.


$side: "left";
$margin-size: 20px;

.content {
    margin-#{$side}: $margin-size; // Tworzy margin-left
}

W tym przykładzie interpolacja generuje właściwość margin-left z wartością $margin-size na podstawie zmiennej $side.

Interpolacja w Generowaniu Wielu Klas i Właściwości

Interpolacja może być użyta w połączeniu z pętlami @for i @each, co pozwala na dynamiczne tworzenie wielu klas lub właściwości.


@for $i from 1 through 3 {
    .column-#{$i} {
        width: #{$i * 10}% // Tworzy width: 10%, width: 20%, itd.
    }
}

W powyższym przykładzie pętla @for generuje klasy .column-1 do .column-3, z szerokościami od 10% do 30%.

Interpolacja w Niestandardowych Właściwościach CSS

Interpolacja jest także przydatna do tworzenia niestandardowych właściwości CSS (CSS Custom Properties), co jest szczególnie użyteczne w bardziej zaawansowanych projektach.


$theme-color: #ff6f61;

:root {
    --main-color: #{$theme-color}; // Tworzy: --main-color: #ff6f61
}

.button {
    background-color: var(--main-color);
}

W powyższym przykładzie interpolacja generuje niestandardową właściwość --main-color z wartością zmiennej $theme-color.

Interpolacja w Mapach i Listach

Interpolacja może również pomóc w iterowaniu po mapach i listach, co jest przydatne przy tworzeniu dynamicznych stylów bazujących na zestawach danych.


$colors: (primary: #3498db, secondary: #2ecc71, danger: #e74c3c);

@each $name, $color in $colors {
    .text-#{$name} {
        color: $color;
    }
}

W tym przykładzie interpolacja generuje klasy .text-primary, .text-secondary, i .text-danger z kolorami pobranymi z mapy $colors.

Przykład: Tworzenie Responsywnych Klas z Interpolacją

Interpolacja jest także przydatna przy generowaniu responsywnych klas. Poniższy przykład pokazuje, jak dynamicznie tworzyć klasy zależne od punktów przerwania.


$breakpoints: (small: 576px, medium: 768px, large: 992px);

@each $size, $px in $breakpoints {
    @media (min-width: $px) {
        .container-#{$size} {
            max-width: $px;
        }
    }
}

W tym przykładzie interpolacja generuje klasy .container-small, .container-medium, i .container-large z odpowiednimi maksymalnymi szerokościami, co pozwala na szybkie tworzenie responsywnych stylów.
