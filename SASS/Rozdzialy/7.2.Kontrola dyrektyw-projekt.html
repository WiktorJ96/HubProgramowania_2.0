Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// Definicja kolorów i motywu
$theme: dark;

.button {
    @if $theme == dark {
        background-color: #333;
        color: #fff;
    } @else {
        background-color: #fff;
        color: #333;
    }
}

// Generowanie klas dla marginesów
@for $i from 1 through 5 {
    .margin-#{$i} {
        margin: #{$i * 10}px;
    }
}

// Definicja listy kolorów
$colors: (primary: #3498db, success: #2ecc71, danger: #e74c3c);

@each $name, $color in $colors {
    .button-#{$name} {
        background-color: $color;
        color: #fff;
    }
}

// Generowanie klas dla obramowań
$i: 1;

@while $i <= 5 {
    .border-#{$i} {
        border-width: #{$i}px;
    }
    $i: $i + 1;
}

// Generowanie kolumn siatki
$columns: 4;
$gutter: 20px;

.main-container {
    display: flex;
    flex-wrap: wrap;

    @for $i from 1 through $columns {
        .col-#{$i} {
            width: (100% / $columns) * $i - $gutter;
            margin-right: $gutter;
        }
    }
}
