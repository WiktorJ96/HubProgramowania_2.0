SASS - Funkcje Niestandardowe

W SASS można definiować własne funkcje, które przyjmują argumenty i zwracają wynik. Dzięki funkcjom niestandardowym możliwe jest wykonywanie obliczeń lub tworzenie złożonych reguł CSS w sposób bardziej dynamiczny. Funkcje niestandardowe są definiowane za pomocą dyrektywy @function.

Definiowanie Prostej Funkcji Niestandardowej

Aby zdefiniować funkcję, należy użyć @function, nadać jej nazwę, a następnie zwrócić wartość za pomocą @return. Funkcję można wywoływać tak samo, jak wbudowane funkcje SASS.


// Definiowanie funkcji do konwersji pikseli na remy
@function px-to-rem($px, $base-font-size: 16px) {
    @return $px / $base-font-size * 1rem;
}

// Użycie funkcji
.container {
    font-size: px-to-rem(18px); // Konwertuje 18px na rem
}

W powyższym przykładzie funkcja px-to-rem przelicza wartość pikseli na remy na podstawie rozmiaru czcionki bazowej. Wartość jest obliczana i zwracana przy użyciu @return.

Funkcje z Wiele Parametrów

Funkcje mogą przyjmować wiele parametrów, co pozwala na bardziej elastyczne operacje. Poniżej znajduje się przykład funkcji obliczającej odcień koloru na podstawie parametrów koloru i stopnia przyciemnienia.


// Funkcja do przyciemniania koloru
@function darken-color($color, $percentage) {
    @return darken($color, $percentage);
}

// Użycie funkcji
.button {
    background-color: darken-color(#3498db, 20%); // Tworzy ciemniejszy kolor
}

W tym przykładzie funkcja darken-color przyjmuje kolor i procent przyciemnienia, a następnie zwraca nowy odcień koloru.

Walidacja Argumentów w Funkcjach

W funkcjach niestandardowych można stosować warunki @if do walidacji argumentów. Na przykład możemy sprawdzić, czy argument jest poprawnego typu, zanim wykonamy na nim operacje.


// Funkcja z walidacją argumentów
@function square($number) {
    @if type-of($number) != 'number' {
        @error "Argument musi być liczbą";
    }
    @return $number * $number;
}

// Użycie funkcji
.box {
    width: square(5px); // Ustawia szerokość na 25px
}

Funkcja square sprawdza, czy argument jest liczbą. Jeśli nie, wyrzuca błąd przy użyciu @error.

Funkcje Zwracające Listy lub Mapy

Funkcje niestandardowe mogą zwracać złożone struktury danych, takie jak listy i mapy. Na przykład funkcja może zwrócić mapę kolorów na podstawie zdefiniowanej palety.


// Funkcja zwracająca mapę kolorów
@function color-palette() {
    @return (
        primary: #3498db,
        secondary: #2ecc71,
        danger: #e74c3c
    );
}

// Użycie funkcji do pobrania koloru z mapy
.alert {
    background-color: map-get(color-palette(), danger); // Ustawia tło na kolor danger
}

Funkcja color-palette zwraca mapę kolorów, co pozwala na szybki dostęp do poszczególnych kolorów w stylach.

Przykład: Funkcja do Obliczania Odstępów

Poniżej znajduje się przykład funkcji, która oblicza odstępy między elementami na podstawie przekazanego stopnia wielkości (np. small, medium, large).


// Funkcja do obliczania odstępów
@function spacing($size) {
    @if $size == small {
        @return 8px;
    } @else if $size == medium {
        @return 16px;
    } @else if $size == large {
        @return 24px;
    } @else {
        @error "Niewłaściwa wartość dla $size: musi być small, medium lub large";
    }
}

// Użycie funkcji
.container {
    margin-bottom: spacing(medium); // Ustawia odstęp na 16px
}

Funkcja spacing zwraca odpowiednią wartość odstępu w zależności od argumentu $size, co pozwala na stosowanie spójnych odstępów w całym projekcie.
