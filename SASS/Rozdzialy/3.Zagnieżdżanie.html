SASS - Zagnieżdżanie

Zagnieżdżanie w SASS pozwala na bardziej przejrzyste i zorganizowane pisanie kodu, umożliwiając strukturalne tworzenie selektorów wewnątrz innych selektorów. Dzięki temu kod CSS jest łatwiejszy w zarządzaniu, zwłaszcza w złożonych projektach.

Podstawy Zagnieżdżania

W SASS selektory można zagnieżdżać, co oznacza, że możemy pisać reguły CSS wewnątrz innych selektorów w sposób, który odzwierciedla strukturę HTML. Poniższy przykład pokazuje zagnieżdżanie elementów w klasie .container:


.container {
    padding: 20px;
    background-color: #f3f3f3;

    .header {
        font-size: 24px;
        color: #333;
    }

    .content {
        font-size: 16px;
        color: #666;

        a {
            color: #3498db;
            text-decoration: none;
        }
    }
}

W powyższym przykładzie reguły dla .header i .content są zagnieżdżone wewnątrz .container, a styl dla linku <a> wewnątrz .content jest zagnieżdżony dodatkowo, co ułatwia organizację kodu.

Wykorzystanie Zagnieżdżania w Pseudo-klasach i Pseudo-elementach

Zagnieżdżanie można stosować również w pseudo-klasach i pseudo-elementach, co pozwala na stosowanie stylów dla stanów elementów, takich jak :hover i ::after.


.button {
    background-color: #3498db;
    color: #fff;
    padding: 10px;

    &:hover {
        background-color: #2980b9;
    }

    &::after {
        content: ' »';
    }
}

W przykładzie powyżej styl :hover oraz ::after jest zagnieżdżony w .button, co poprawia czytelność i organizację kodu.

Operator & (referencja do rodzica)

Symbol & odnosi się do bieżącego selektora, co jest przydatne przy tworzeniu bardziej złożonych struktur CSS. Na przykład, można go użyć do tworzenia modyfikatorów klasy.


.button {
    background-color: #3498db;
    color: #fff;

    &.is-active {
        background-color: #2ecc71;
    }

    &--large {
        padding: 20px;
        font-size: 18px;
    }
}

W przykładzie & jest użyty do tworzenia modyfikatorów .button.is-active i .button--large, co umożliwia łatwe zarządzanie wariantami klasy .button.

Zagnieżdżanie Media Queries

W SASS można również zagnieżdżać @media wewnątrz selektorów, co pozwala na łatwe definiowanie responsywnych stylów dla poszczególnych elementów.


.card {
    padding: 20px;
    background-color: #fff;
    border: 1px solid #ddd;

    @media (min-width: 768px) {
        padding: 40px;
    }

    .title {
        font-size: 24px;

        @media (min-width: 768px) {
            font-size: 32px;
        }
    }
}

W powyższym przykładzie @media zostało zagnieżdżone wewnątrz .card oraz .title, co ułatwia organizację responsywnych stylów bez konieczności powtarzania pełnych selektorów.

Przykład Zagnieżdżania w Projekcie

Zagnieżdżanie jest szczególnie przydatne, gdy projekt wymaga wielu zagnieżdżonych selektorów. Oto przykład kodu, który przedstawia, jak zagnieżdżanie może pomóc w organizacji stylów dla złożonego komponentu.


.navbar {
    background-color: #333;
    padding: 15px;

    .nav-item {
        color: #fff;
        margin-right: 10px;

        &:hover {
            color: #ddd;
        }

        .icon {
            margin-right: 5px;
        }
    }

    .nav-item--active {
        font-weight: bold;
        color: #ff6f61;
    }
}

W tym przykładzie struktura .navbar zawiera zagnieżdżone style dla elementów nawigacji, takich jak .nav-item, .nav-item:hover oraz .nav-item--active. Dzięki temu kod jest bardziej zorganizowany i łatwiejszy w zarządzaniu.
