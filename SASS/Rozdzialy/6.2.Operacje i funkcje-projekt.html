Karta z Dynamicznymi Stylami

Przykład karty wykorzystującej funkcje i operacje matematyczne SASS.

Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// Ustawienia bazowe
$primary-color: #3498db;
$secondary-color: lighten($primary-color, 20%);
$border-radius: 5px;
$padding: 10px;

// Styl dla przycisku
.button {
    background-color: $primary-color;
    padding: $padding;
    border-radius: $border-radius;
    color: #fff;
    border: none;
    cursor: pointer;

    &:hover {
        background-color: darken($primary-color, 10%);
    }
}

// Styl dla karty
.card {
    background-color: $secondary-color;
    border-radius: $border-radius;
    padding: $padding * 2;
    box-shadow: 0 4px 8px rgba(0, 0, 0, 0.2);
    color: #333;
}

// Styl kontenera
.main-container {
    width: 100% - 2 * $padding;
    margin: $padding auto;
    padding: $padding / 2;
}
