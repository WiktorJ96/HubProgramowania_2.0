Nagłówek z dziedziczonym stylem bazowym

To jest paragraf, który dziedziczy styl bazowy tekstu z użyciem `@extend`.

Podstawowa Karta

Przykład karty z dziedziczonym stylem bazowym `card-base`.

Wyróżniona Karta

Przykład wyróżnionej karty z dziedziczonym stylem bazowym `card-base`.

Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// Definicja placeholdera dla karty
%card-base {
    border: 1px solid #333;
    border-radius: 8px;
    padding: 15px;
    box-shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
}

// Dziedziczenie stylu placeholdera
.card {
    @extend %card-base;
    background-color: #333;
}

.card-highlighted {
    @extend %card-base;
    background-color: #333;
    border-color: #3498db;
}

// Styl podstawowy dla tekstu
%base-text {
    font-family: Arial, sans-serif;
    color: #333;
    line-height: 1.5;
}

// Dziedziczenie stylu tekstu bazowego
.header {
    @extend %base-text;
    font-size: 24px;
}

.paragraph {
    @extend %base-text;
    font-size: 16px;
    margin-bottom: 10px;
}

// Styl przycisków z użyciem @extend
.button {
    padding: 10px 20px;
    font-size: 16px;
    color: #fff;
    background-color: #3498db;
    border: none;
    border-radius: 5px;
}

.button-secondary {
    @extend .button;
    background-color: #2ecc71;
}
