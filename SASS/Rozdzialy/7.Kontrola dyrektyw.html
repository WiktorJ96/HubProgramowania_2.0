SASS - Kontrola Dyrektyw

SASS umożliwia kontrolowanie generacji kodu CSS przy użyciu dyrektyw kontrolnych, takich jak @if, @else, @for, i @each. Dzięki nim można tworzyć dynamiczne reguły CSS w zależności od warunków, iteracji i wartości zmiennych.

Dyrektywa @if i @else

Dyrektywy @if i @else umożliwiają tworzenie warunkowych stylów. Dzięki nim można definiować różne style w zależności od określonych warunków.


// Definicja koloru w zależności od zmiennej
$theme: dark;

.button {
    @if $theme == dark {
        background-color: #333;
        color: #fff;
    } @else {
        background-color: #fff;
        color: #333;
    }
}

W tym przykładzie styl .button zmienia się w zależności od wartości zmiennej $theme. Jeśli motyw jest ciemny, przycisk będzie miał ciemne tło i jasny tekst.

Dyrektywa @for

Dyrektywa @for umożliwia iterację w określonym zakresie liczb, co pozwala generować powtarzające się style dynamicznie. Jest to przydatne np. przy tworzeniu siatek lub wariantów stylów.


// Generowanie klas dla marginesów
@for $i from 1 through 5 {
    .margin-#{$i} {
        margin: #{$i * 10}px;
    }
}

W powyższym przykładzie dyrektywa @for generuje klasy .margin-1 do .margin-5 z wartościami marginesów od 10px do 50px.

Dyrektywa @each

Dyrektywa @each umożliwia iterację po elementach listy lub mapy, co pozwala na generowanie stylów dla każdego elementu na liście. Jest szczególnie przydatna do pracy z listami kolorów, czcionek lub rozmiarów.


// Definicja listy kolorów
$colors: (primary: #3498db, success: #2ecc71, danger: #e74c3c);

@each $name, $color in $colors {
    .button-#{$name} {
        background-color: $color;
        color: #fff;
    }
}

W przykładzie powyżej dyrektywa @each generuje klasy .button-primary, .button-success, i .button-danger na podstawie listy $colors i przypisanych kolorów.

Dyrektywa @while

Dyrektywa @while pozwala na wykonywanie powtarzających się stylów, dopóki warunek jest spełniony. Jest przydatna do bardziej złożonych iteracji i warunków.


$i: 1;

@while $i <= 5 {
    .border-#{$i} {
        border-width: #{$i}px;
    }
    $i: $i + 1;
}

W tym przykładzie dyrektywa @while generuje klasy .border-1 do .border-5, zmieniając szerokość obramowania od 1px do 5px.

Przykład: Tworzenie Siatki za pomocą Dyrektyw Kontrolnych

Poniżej znajduje się bardziej zaawansowany przykład zastosowania dyrektyw kontrolnych do generowania siatki układu.


$columns: 4;
$gutter: 20px;

.container {
    display: flex;
    flex-wrap: wrap;

    @for $i from 1 through $columns {
        .col-#{$i} {
            width: (100% / $columns) * $i - $gutter;
            margin-right: $gutter;
        }
    }
}

W powyższym przykładzie dyrektywa @for generuje klasy .col-1 do .col-4 z odpowiednimi szerokościami, tworząc dynamicznie kolumny siatki. Stylizacja uwzględnia także margines między kolumnami (gutter).
