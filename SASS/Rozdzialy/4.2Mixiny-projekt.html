Podgląd Kodów SASS i HTML

Poniżej znajdują się fragmenty kodów SASS i HTML, z których składa się projekt:

Kod SASS


// Mixin dla zaokrąglonych rogów
@mixin rounded-corners($radius) {
    border-radius: $radius;
    overflow: hidden;
}

// Mixin dla cienia
@mixin box-shadow($color: rgba(0, 0, 0, 0.5), $blur: 10px) {
    box-shadow: 0 0 $blur $color;
}

// Styl przycisku z parametrami
@mixin button-style($bg-color, $text-color: #fff, $padding: 10px) {
    background-color: $bg-color;
    color: $text-color;
    padding: $padding;
    border: none;
    border-radius: 5px;
    cursor: pointer;
    text-transform: uppercase;

    &:hover {
        background-color: darken($bg-color, 10%);
    }
}

// Użycie mixinów
.card {
    @include rounded-corners(10px);
    background-color: #333;
    padding: 20px;
}

.button-primary {
    @include button-style(#3498db);
}

.button-secondary {
    @include button-style(#2ecc71, #333, 15px);
}

.panel {
    @include box-shadow;
    padding: 20px;
    background-color: #333;
    @include rounded-corners(5px);
}

.panel-strong {
    @include box-shadow(rgba(0, 0, 0, 0.8), 15px);
    background-color: #eee;
    padding: 20px;
}
