Iteratory

Iteratory to specjalne metody w Ruby, które umożliwiają przeglądanie elementów kolekcji, takich jak tablice czy hashe. Najbardziej popularnym iteratorem w Ruby jest each, ale istnieje wiele innych, które pozwalają na bardziej zaawansowane operacje.

Podstawowy iterator each

Metoda each jest najczęściej używanym iteratorem w Ruby. Przegląda ona wszystkie elementy kolekcji i wykonuje blok kodu dla każdego elementu.


        fruits = ["jabłko", "banan", "truskawka"]

        # Użycie each do iteracji po tablicy
        fruits.each do |fruit|
          puts "Owoc: #{fruit}"
        end
        # Wyświetla: 
        # Owoc: jabłko
        # Owoc: banan
        # Owoc: truskawka

Iterator map

Metoda map (czasem znana jako collect) przekształca każdy element kolekcji i zwraca nową kolekcję z przekształconymi elementami. Często jest używana, gdy chcemy dokonać zmiany na wszystkich elementach kolekcji.


        numbers = [1, 2, 3, 4]

        # Użycie map do przekształcenia elementów tablicy
        squares = numbers.map { |num| num * num }
        puts squares.inspect  # Wyświetla: [1, 4, 9, 16]

Iterator select

Metoda select filtruje elementy kolekcji, zwracając nową kolekcję z elementami, które spełniają warunki zdefiniowane w bloku kodu.


        numbers = [1, 2, 3, 4, 5, 6]

        # Użycie select do wybierania liczb parzystych
        even_numbers = numbers.select { |num| num.even? }
        puts even_numbers.inspect  # Wyświetla: [2, 4, 6]

Iterator reduce (znany też jako inject)

Metoda reduce jest używana do zredukowania wszystkich elementów kolekcji do jednej wartości, np. sumy wszystkich elementów.


        numbers = [1, 2, 3, 4, 5]

        # Użycie reduce do obliczenia sumy elementów
        sum = numbers.reduce(0) { |total, num| total + num }
        puts sum  # Wyświetla: 15

Inne przydatne iteratory

Ruby oferuje również inne iteratory, które mogą być przydatne w zależności od potrzeb:

  • find – Zwraca pierwszy element, który spełnia warunek.
  • all? – Sprawdza, czy wszystkie elementy kolekcji spełniają warunek.
  • any? – Sprawdza, czy jakikolwiek element spełnia warunek.
  • none? – Sprawdza, czy żaden element nie spełnia warunku.
  • count – Zwraca liczbę elementów, które spełniają warunek.

Przykład metody find


        numbers = [1, 2, 3, 4, 5]

        # Znalezienie pierwszej liczby większej niż 3
        first_large_number = numbers.find { |num| num > 3 }
        puts first_large_number  # Wyświetla: 4

Iteracja z indeksem

Jeśli potrzebujesz zarówno elementów, jak i ich indeksów, możesz użyć metody each_with_index.


        fruits = ["jabłko", "banan", "truskawka"]

        # Użycie each_with_index do iteracji z dostępem do indeksu
        fruits.each_with_index do |fruit, index|
          puts "Owoc nr #{index}: #{fruit}"
        end
        # Wyświetla:
        # Owoc nr 0: jabłko
        # Owoc nr 1: banan
        # Owoc nr 2: truskawka

Podsumowanie

Iteratory w Ruby to potężne narzędzia pozwalające na efektywną pracę z kolekcjami danych. Pozwalają one na elastyczne przetwarzanie danych, od prostego przeglądania po skomplikowane przekształcenia. Dzięki wielu metodom, takim jak each, map czy reduce, można w prosty sposób manipulować danymi w Ruby.
