Tablice

Tablice (Arrays) w Ruby są strukturami danych, które przechowują kolekcje elementów. Elementy te mogą być różnego typu, na przykład liczby, łańcuchy znaków czy inne obiekty. Tablice są dynamiczne, co oznacza, że mogą zmieniać swój rozmiar w trakcie działania programu.

Tworzenie tablicy

Tablicę w Ruby można stworzyć za pomocą nawiasów kwadratowych [], lub używając metody Array.new. Elementy tablicy są indeksowane, co oznacza, że możemy uzyskać dostęp do elementu tablicy na podstawie jego indeksu.


        # Tworzenie tablicy za pomocą nawiasów kwadratowych
        fruits = ["jabłko", "banan", "truskawka"]

        # Tworzenie tablicy za pomocą metody Array.new
        numbers = Array.new(3)  # Tablica o 3 elementach, wypełniona nilami

        puts fruits[0]  # Wyświetla: "jabłko"
        puts numbers[1]  # Wyświetla: nil

Dodawanie i usuwanie elementów

Ruby oferuje różne metody do manipulacji elementami tablicy. Można dodawać elementy na końcu tablicy za pomocą metody push lub operatora <<, a usuwać za pomocą metody pop lub delete.


        fruits = ["jabłko", "banan", "truskawka"]

        # Dodawanie elementu na końcu tablicy
        fruits.push("gruszka")
        fruits << "winogrono"

        # Usuwanie elementu z końca tablicy
        fruits.pop  # Usuwa "winogrono"

        puts fruits.inspect  # Wyświetla: ["jabłko", "banan", "truskawka", "gruszka"]

        # Usuwanie konkretnego elementu
        fruits.delete("banan")
        puts fruits.inspect  # Wyświetla: ["jabłko", "truskawka", "gruszka"]

Iteracja po tablicach

Tablice w Ruby można łatwo przeglądać za pomocą różnych metod iteracyjnych, takich jak each czy map.


        fruits = ["jabłko", "truskawka", "gruszka"]

        # Użycie each do iteracji po elementach
        fruits.each do |fruit|
          puts "Owoc: #{fruit}"
        end

        # Użycie map do przekształcenia elementów tablicy
        uppercase_fruits = fruits.map { |fruit| fruit.upcase }
        puts uppercase_fruits.inspect  # Wyświetla: ["JABŁKO", "TRUSKAWKA", "GRUSZKA"]

Dostęp do elementów tablicy

Do elementów tablicy możemy uzyskać dostęp poprzez indeks. Indeksowanie w Ruby zaczyna się od zera. Istnieje również możliwość dostępu do elementów od końca tablicy za pomocą indeksów ujemnych.


        fruits = ["jabłko", "truskawka", "gruszka"]

        # Dostęp do elementów za pomocą indeksu
        puts fruits[0]  # Wyświetla: "jabłko"
        puts fruits[-1]  # Wyświetla: "gruszka" (ostatni element)

        # Można również wycinać podtablice
        puts fruits[0, 2].inspect  # Wyświetla: ["jabłko", "truskawka"]

Metody przydatne dla tablic

Ruby oferuje wiele przydatnych metod do pracy z tablicami, takich jak sort, reverse, include?, join i wiele innych.


        fruits = ["gruszka", "jabłko", "truskawka"]

        # Sortowanie tablicy
        sorted_fruits = fruits.sort
        puts sorted_fruits.inspect  # Wyświetla: ["gruszka", "jabłko", "truskawka"]

        # Odwracanie kolejności elementów w tablicy
        reversed_fruits = fruits.reverse
        puts reversed_fruits.inspect  # Wyświetla: ["truskawka", "jabłko", "gruszka"]

        # Sprawdzanie, czy tablica zawiera dany element
        puts fruits.include?("jabłko")  # Wyświetla: true

        # Łączenie elementów tablicy w łańcuch znaków
        puts fruits.join(", ")  # Wyświetla: "gruszka, jabłko, truskawka"

Podsumowanie

Tablice są jedną z podstawowych struktur danych w Ruby. Dzięki ich dynamicznemu charakterowi oraz dużej liczbie wbudowanych metod, praca z nimi jest bardzo elastyczna i wydajna. Tablice pozwalają przechowywać, modyfikować i przetwarzać zbiory danych w prosty sposób.
