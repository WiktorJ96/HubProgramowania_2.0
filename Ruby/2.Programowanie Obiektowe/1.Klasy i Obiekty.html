Klasy i Obiekty

Ruby jest językiem obiektowym, co oznacza, że wszystko w Ruby jest obiektem. Klasa jest jak szablon, na podstawie którego tworzone są obiekty. Klasy mogą mieć atrybuty i metody, które definiują ich zachowanie i właściwości.

Tworzenie klasy

Aby zdefiniować klasę w Ruby, używamy słowa kluczowego class. Klasa może zawierać metody, które określają jej zachowanie.


        class Car
          # Definicja metody inicjalizującej
          def initialize(make, model, year)
            @make = make  # Atrybuty obiektu
            @model = model
            @year = year
          end

          # Definicja metody wyświetlającej szczegóły samochodu
          def display_details
            puts "Samochód: #{@make}, Model: #{@model}, Rok: #{@year}"
          end
        end

Tworzenie obiektu

Obiekt to instancja klasy. Obiekty są tworzone za pomocą metody new, która wywołuje metodę initialize odpowiedzialną za przypisanie początkowych wartości atrybutów.


        # Tworzenie nowego obiektu klasy Car
        my_car = Car.new("Toyota", "Corolla", 2020)
        my_car.display_details  # Wywołanie metody obiektu

Atrybuty i metody

W Ruby możemy zdefiniować metody dostępu do atrybutów (getter i setter), aby kontrolować dostęp do zmiennych instancji.

Definiowanie metod dostępu (getter i setter)


        class Person
          def initialize(name, age)
            @name = name
            @age = age
          end

          # Getter dla atrybutu name
          def name
            @name
          end

          # Setter dla atrybutu name
          def name=(new_name)
            @name = new_name
          end

          # Getter dla atrybutu age
          def age
            @age
          end

          # Setter dla atrybutu age
          def age=(new_age)
            @age = new_age
          end
        end

        person = Person.new("Alice", 25)
        puts person.name    # Wyświetla: Alice
        person.name = "Bob" # Zmiana imienia na Bob
        puts person.name    # Wyświetla: Bob

Dziedziczenie

Dziedziczenie pozwala tworzyć klasy, które dziedziczą funkcjonalność innych klas. Używamy do tego znaku < po nazwie klasy bazowej.


        class Animal
          def speak
            puts "Zwierzę wydaje dźwięk"
          end
        end

        # Klasa Dog dziedziczy po klasie Animal
        class Dog < Animal
          def bark
            puts "Pies szczeka"
          end
        end

        dog = Dog.new
        dog.speak  # Wywołanie metody odziedziczonej z klasy Animal
        dog.bark   # Wywołanie metody zdefiniowanej w klasie Dog

Metody klasowe i instancyjne

Ruby rozróżnia metody instancyjne, które działają na konkretnych obiektach, oraz metody klasowe, które można wywołać bez tworzenia obiektu klasy.

Metoda klasowa

Metody klasowe są definiowane za pomocą self.


        class MathHelper
          # Metoda klasowa
          def self.add(a, b)
            a + b
          end
        end

        result = MathHelper.add(5, 3)
        puts result  # Wyświetla: 8

Metoda instancyjna

Metody instancyjne działają na instancjach (obiektach) danej klasy.


        class Calculator
          def initialize(number)
            @number = number
          end

          # Metoda instancyjna
          def square
            @number * @number
          end
        end

        calc = Calculator.new(4)
        puts calc.square  # Wyświetla: 16
