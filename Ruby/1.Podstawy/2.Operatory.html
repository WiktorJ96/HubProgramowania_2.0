Operatory

Ruby obsługuje szeroki wachlarz operatorów, które umożliwiają wykonywanie operacji na zmiennych i wartościach. Poniżej omówione zostaną najczęściej używane operatory w Ruby.

Operatory arytmetyczne

Te operatory służą do wykonywania operacji matematycznych na liczbach.

  • Dodawanie (+): 5 + 3 zwraca 8.
  • Odejmowanie (-): 5 - 3 zwraca 2.
  • Mnożenie (*): 5 * 3 zwraca 15.
  • Dzielenie (/): 6 / 3 zwraca 2.
  • Modulo (%): Zwraca resztę z dzielenia (np. 5 % 2 zwraca 1).
  • Potęgowanie (**): Podnosi liczbę do potęgi (np. 2 ** 3 zwraca 8).

Operatory przypisania

Operatory przypisania służą do przypisywania wartości do zmiennych.

  • = : Przypisuje wartość (np. x = 5).
  • += : Dodaje wartość do zmiennej i przypisuje (np. x += 3 jest równoważne x = x + 3).
  • -= : Odejmowanie z przypisaniem (np. x -= 3).
  • *= : Mnożenie z przypisaniem (np. x *= 3).
  • /= : Dzielenie z przypisaniem (np. x /= 3).
  • %= : Modulo z przypisaniem (np. x %= 3).

Operatory porównania

Operatory te służą do porównywania wartości. Wynikiem porównania jest wartość logiczna true lub false.

  • == : Sprawdza, czy wartości są równe (np. 5 == 5 zwraca true).
  • != : Sprawdza, czy wartości są różne (np. 5 != 3 zwraca true).
  • > : Sprawdza, czy wartość po lewej stronie jest większa (np. 5 > 3).
  • < : Sprawdza, czy wartość po lewej stronie jest mniejsza (np. 3 < 5).
  • >= : Sprawdza, czy wartość po lewej stronie jest większa lub równa (np. 5 >= 3).
  • <= : Sprawdza, czy wartość po lewej stronie jest mniejsza lub równa (np. 3 <= 5).

Operatory logiczne

Operatory logiczne służą do wykonywania operacji logicznych na wartościach true i false.

  • and : Zwraca true tylko wtedy, gdy obie wartości są true (np. true and false zwraca false).
  • or : Zwraca true, jeśli przynajmniej jedna wartość jest true (np. true or false zwraca true).
  • not : Odwraca wartość logiczną (np. not true zwraca false).
  • && : Alternatywa dla and (np. true && false).
  • || : Alternatywa dla or (np. true || false).

Operatory specjalne

Ruby oferuje także kilka innych operatorów, które mają bardziej specyficzne zastosowania:

  • Operator zakresu (.. i ...): Służy do tworzenia zakresów wartości. 1..5 obejmuje liczby od 1 do 5, a 1...5 obejmuje liczby od 1 do 4.
  • Operator złączenia (+): Służy do łączenia ciągów znaków (np. "Hello " + "World" zwraca "Hello World").
  • Operator porównania (<=>): Zwraca -1, 0, lub 1, w zależności od porównania dwóch wartości (np. 5 <=> 3 zwraca 1).

    • Zwraca -1 – jeśli lewa strona jest mniejsza niż prawa strona.
    Zwraca 0 – jeśli lewa strona jest równa prawej stronie.
    Zwraca 1 – jeśli lewa strona jest większa niż prawa strona.
