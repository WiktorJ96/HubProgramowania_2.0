Składnia, Zmienne i Typy Danych

Ruby to język programowania o dynamicznie typowanej składni, zaprojektowany z myślą o prostocie i czytelności kodu. Poniżej omówione zostaną podstawowe elementy Ruby, takie jak składnia, zmienne, komentarze, typy danych oraz literały.

Składnia

Składnia Ruby jest dość elastyczna i pozwala na pominięcie niektórych znaków, które są wymagane w innych językach (np. średniki). Przykład prostego programu w Ruby:


        name = "John"  # Deklaracja zmiennej i przypisanie wartości
        puts name      # Wyświetlenie wartości zmiennej w konsoli

Komentarze

Komentarze w Ruby to notatki w kodzie, które nie są wykonywane przez program. Można używać komentarzy jednolinijkowych lub wielolinijkowych:


# To jest komentarz jednolinijkowy

=begin
To jest komentarz
wielolinijkowy
=end

Typy Danych

Ruby posiada różne typy danych, w tym liczby, teksty, wartości logiczne i inne struktury. Główne typy danych w Ruby to:

Typy proste

  • Number: Przechowuje liczby całkowite i zmiennoprzecinkowe (np. age = 25).
  • String: Przechowuje tekst (np. name = "Alice").
  • Boolean: Przechowuje wartość logiczną true lub false (np. is_student = true).
  • Nil: Oznacza brak wartości (np. empty = nil).

Typy referencyjne

  • Hash: Kolekcja par klucz-wartość (np. person = {name: "John", age: 30}).
  • Array: Lista elementów (np. colors = ["red", "green", "blue"]).

Zmienne

Zmienna w Ruby to obiekt, do którego można przypisać wartość. Ruby rozróżnia różne typy zmiennych:

  • Zwykłe zmienne: Zmienna zaczyna się od małej litery lub znaku _ (np. name = "Alice").
  • Zmienna globalna: Zmienna zaczyna się od znaku $ (np. $global_variable).
  • Zmienna instancji: Zmienna zaczyna się od znaku @ (np. @instance_variable).
  • Zmienna klasowa: Zmienna zaczyna się od @@ (np. @@class_variable).
  • Stałe: Zmienna, której wartość nie powinna być zmieniana, zaczyna się od wielkiej litery (np. PI = 3.14).

Literały

Literały to wartości wpisane bezpośrednio w kodzie. Oto kilka przykładów literałów w Ruby:

  • Literał liczbowy: x = 10
  • Literał łańcuchowy: text = "Hello, World!"
  • Literał tablicowy: fruits = ["apple", "banana", "cherry"]
  • Literał obiektowy: car = {model: "Toyota", year: 2020}
