Podstawy SQL

SQL (Structured Query Language) to standardowy język służący do komunikacji z relacyjnymi bazami danych. Umożliwia definiowanie struktury bazy, manipulację danymi oraz ich wyszukiwanie za pomocą zapytań.

Podstawowa składnia SQL

Każde zapytanie SQL może składać się z kilku klauzul, z których najważniejsze to:

  • SELECT – wybiera kolumny, które chcemy zobaczyć,
  • FROM – określa tabelę (lub tabele), z której pobieramy dane,
  • WHERE – filtruje wiersze spełniające określone warunki.

SELECT i FROM


// Wybranie wszystkich kolumn z tabeli users
SELECT *
FROM users;

// Wybranie konkretnych kolumn: imię i wiek
SELECT first_name, age
FROM users;

SELECT wskazuje, które kolumny (pola) chcemy pobrać, a FROM określa, z jakiej tabeli.

Klauzula WHERE


// Pobranie użytkowników, których wiek jest większy niż 18
SELECT first_name, age
FROM users
WHERE age > 18;

// Pobranie pracowników z działu 'HR'
SELECT *
FROM employees
WHERE department = 'HR';

WHERE pozwala ustawić warunki, dzięki którym zwrócone zostaną tylko te wiersze, które je spełniają.

Zasady korzystania z bazy danych

  • Zawsze rób kopie zapasowe (backup) przed wprowadzeniem zmian na bazie produkcyjnej.
  • Stosuj klucze główne (PRIMARY KEY) i obce (FOREIGN KEY) dla integralności danych.
  • Normalizuj (porządkuj) dane, by unikać powtórzeń i zapewnić spójność.
  • Używaj transakcji (BEGIN TRANSACTION, COMMIT, ROLLBACK) przy operacjach wielokrokowych.
  • Optymalizuj zapytania poprzez indeksy na kolumnach często filtrowanych lub sortowanych.
  • Zamykaj połączenia do bazy (np. connection.close()), by nie obciążać serwera.
