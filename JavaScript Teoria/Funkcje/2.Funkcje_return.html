Funkcje i return w JavaScript

Funkcje w JavaScript mogą zwracać wartości za pomocą słowa kluczowego return. Gdy funkcja napotka instrukcję return, natychmiast kończy swoje działanie i zwraca określoną wartość do miejsca, z którego funkcja została wywołana.

Podstawowe użycie return

Funkcje mogą zwracać różne typy danych. Poniżej przedstawiamy przykład, w którym funkcja zwraca wartość liczbową:

Przykład zwracania liczby


function add(a, b) {
    return a + b;
}

const result = add(5, 10);
console.log(result); // 15

Funkcja add przyjmuje dwa argumenty a i b, a następnie zwraca ich sumę. Wartość 15 jest wynikiem działania funkcji i zostaje przypisana do zmiennej result.

Zwracanie wartości złożonych

Za pomocą return możemy również zwracać bardziej złożone wartości, takie jak obiekty, tablice czy inne funkcje.

Przykład zwracania obiektu


function createUser(name, age) {
    return {
        name: name,
        age: age
    };
}

const user = createUser('Anna', 25);
console.log(user); // { name: 'Anna', age: 25 }

Funkcja createUser zwraca obiekt zawierający informacje o użytkowniku. W tym przypadku, obiekt user zawiera nazwę i wiek użytkownika.

Przerywanie działania funkcji

Instrukcja return nie tylko zwraca wartość, ale także natychmiast przerywa działanie funkcji. Instrukcje po return nie zostaną wykonane.


function checkNumber(num) {
    if (num > 10) {
        return 'Liczba jest większa niż 10';
    }
    // Ta część kodu zostanie pominięta, jeśli num > 10
    return 'Liczba jest 10 lub mniejsza';
}

console.log(checkNumber(15)); // 'Liczba jest większa niż 10'
console.log(checkNumber(5));  // 'Liczba jest 10 lub mniejsza'

W tym przykładzie, jeśli liczba num jest większa niż 10, funkcja natychmiast zwróci odpowiedni komunikat i zakończy działanie.

Zwracanie funkcji

Funkcja w JavaScript może zwracać inną funkcję, co pozwala na tworzenie bardziej złożonych struktur, takich jak fabryki funkcji czy domknięcia (closures).


function multiplyBy(factor) {
    return function(number) {
        return number * factor;
    };
}

const multiplyBy3 = multiplyBy(3);
console.log(multiplyBy3(10)); // 30

W tym przykładzie funkcja multiplyBy zwraca inną funkcję, która wykonuje mnożenie liczby przez określony czynnik (factor). Jest to przykład funkcji wyższego rzędu (ang. higher-order function).

Podsumowanie

  • Instrukcja return zwraca wartość i natychmiast przerywa działanie funkcji.
  • Funkcje mogą zwracać dowolne typy danych, w tym liczby, ciągi znaków, obiekty, tablice, a nawet inne funkcje.
  • Instrukcja return jest kluczowa w konstrukcji logiki funkcji, umożliwiając przerywanie wykonywania kodu po spełnieniu określonych warunków.
