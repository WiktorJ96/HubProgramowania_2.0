Reflect.has(): Sprawdza, czy obiekt ma określoną właściwość, podobnie jak operator in.
Reflect.set(): Ustawia wartość właściwości obiektu, tutaj dodaje nową właściwość 'brand'.
Reflect.ownKeys(): Zwraca wszystkie klucze (zarówno właściwości zwykłe, jak i symbole) obiektu.
Reflect.defineProperty(): Definiuje lub modyfikuje właściwości obiektu. W tym przypadku nie ma zmian, ponieważ nie podano szczegółów.
Reflect.construct(): Tworzy nowy obiekt, działając jak operator new, ale w bardziej dynamiczny sposób.
